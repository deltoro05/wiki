Linuxでのパッケージ管理

yum または apt-get。

基本操作

apt-get の場合

# パッケージを検索
apt-cache search キーワード

# インストール
sudo apt-get install パッケージ名

# アンインストール
sudo apt-get remove パッケージ名

yum の場合

# パッケージを検索
yum search キーワード

# インストール
sudo yum install パッケージ名

# アンインストール
sudo yum remove パッケージ名

sudo yum -y install パッケージ名 で質問には全部Yesと答えることにして自動で実行する。

パッケージでインストールされるファイルの一覧を見るには

apt-get の場合

dpkg -L パッケージ名

yum の場合

rpm -ql パッケージ名

インストール済みのパッケージ一覧を見るには

apt-get の場合

dpkg -l
