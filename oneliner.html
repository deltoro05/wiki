シェルスクリプトでシンボリックリンクからリンク先を取得するには 2013/04/22

ls -l シンボリックリンク名 | perl -anle '$_ =~ /-> (.+)$/; print $1'

永久ループを書くには 2013/04/24

while : ; do date; sleep 1; done

URLエンコード / URLデコード 2013/05/30

# URLエンコード
echo あ | ruby -r cgi -ne 'puts CGI.escape $_.chomp' | tr "\n" " " | sed 's/ /%0D%0A/g'

最後の tr "\n" " " | sed s/ /%0D%0A/g は改行を %0D%0A に変換するためのもの。

# URLデコード
echo %E3%81%82 | tr % = | nkf -WmQ

-W はUTF-8を意味し、 -mQ はURLデコードする。

Unicodeエスケープをデコードするには 2013/04/16

Unicodeエスケープは、\u5c71 のようなもの。

cat ファイル名 | ruby -ne 'print $_.gsub(/\\u([0-9a-f]{4})/){[$1.hex].pack("U")}'

たくさんのファイルをまとめて less で閲覧するには 2013/05/31

find などのファイル一覧の出力に対して、 less で閲覧するには

find ... | awk '{print "echo " $1 "; cat -n " $1}' | sh | less

cat -n は行番号をつけて出力する。

改行コードを変換するには 2013/06/04

cat ファイル名 | perl -pe 's/\r\n/\n/'

ファイル名に日付が入っているファイルがたくさんあって、日付ごとにファイル数を調べるには 2013/06/22

2013-06-22-hogehoge みたいな名前のファイルがたくさんあって、日付ごとにそのファイル数を集計するには、以下のようにする。

for i in 0 1 2 3 4 5 6 7; do date --date "$i days" "+%Y-%m-%d"; done | awk '{print "echo -n " $1 " \"\"; ls " $1 "-* 2>/dev/null | wc -l";}' | sh

出力例

2013-06-22 2
2013-06-23 0
2013-06-24 937
2013-06-25 860
2013-06-26 673
2013-06-27 281
2013-06-28 977
2013-06-29 0
