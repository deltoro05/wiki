エスケープ処理

Java 2013/05/20

以下ののエスケープが使える。

\uxxxx
Unicodeで文字を表現(Unicodeエスケープ)
\\数字(1ケタ〜3ケタ)
8進数で文字を表現 (\u0000 〜 \u00FF までしか表現できない)
\b
0x08
\t
0x09 タブ
\n
0x0a LF
\f
0x0c
\r
0x0d CR
\"
"
\'
'
\\
\

Unicodeエスケープは、文字列リテラルの機能ではなく、Javaソースコードをコンパイラがトークンに分割する前の処理として機能する。

PHP 2013/06/16

' で囲めば、エスケープが必要になるのは ', \ のみになる。'\n' などと書いても \n そのものになって改行にはならない。 ' で囲まれている場合、 \ の次に ', \ 以外が来ても特別な意味にはならない。

echo 'aaa\nttt'; // => aaa\nttt // \n は改行にはならない
echo 'aaa\'ttt'; // => aaa'ttt
echo 'aaa\ttt'; // => aaa\ttt // \t はタブにはならずに \t のまま
echo 'aaa\\ttt'; // => aaa\ttt // \\ は \ になる

" で囲んだ場合は、以下のエスケープが使える。

\\数字(1ケタ〜3ケタ)
8進数で文字を表現
\\x数字(1ケタ〜2ケタ)
16進数で文字を表現
\t
0x09 タブ
\n
0x0a LF
\f
0x0c
\r
0x0d CR
\e
0x1B ESC
\$
$
\"
"
\\
\

sh 2013/05/20

いくつかの記号はシェルで特別な意味を持つので \ でエスケープが必要。

" で囲めば、エスケープが必要になるのは ", $, \, ` のみになる。

' で囲めば、エスケープ処理が不要になるが、 ' を含めることができなくなる。
