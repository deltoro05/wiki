ヒアドキュメント

PHP 2013/06/16

普通のヒアドキュメントとNowdocというものがある。ヒアドキュメントでは、" で囲んだ文字列リテラルと同じように、$ で始まる部分が変数名としてその値に展開がされるのに対して、Nowdocは、' で囲んだ文字列リテラルと同じように、そういった展開がされない。

ヒアドキュメント | PHPマニュアル
http://www.php.net/manual/ja/language.types.string.php#language.types.string.syntax.heredoc

シングルクオートのヒアドキュメント | PHPマニュアル
http://www.php.net/manual/ja/language.types.string.php#language.types.string.syntax.nowdoc

str = <<EOS

EOS

Python 2013/07/23

ダブルクオーテーション3つで囲むと改行を含む文字列を書くことができる。

"""\
foo
bar
"""

上記の例で1行目にバックスラッシュがあるのは最初の改行を文字列に含めないようにするため。

sh 2013/05/31

例

cat <<EOD
foo
bar
$HOME
`date`
EOD

ヒアドキュメントの中では \` や $` で始まる変数の展開が使える。それらを使いたくない場合は、以下のようにする。

cat <<\EOD
foo
bar
EOD
