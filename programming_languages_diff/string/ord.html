文字(≠文字列)の文字コードを数字として取得するには 2014/09/28

以下は、文字からUnicodeの符号位置を数字として取得する方法。逆に符号位置の数字から文字に変換するには chr を使う。

C言語やJava/Scalaのように、文字列と文字の型が違っていて文字の型が整数としても扱える言語では、文字型はすなわち符号位置の数字となっているのだが、そうでない言語では何らかの関数などで変換する必要がある。

Scala 2014/04/18

Char型をIntにキャストすればよい。

println('A'.asInstanceOf[Int]) // => 65
println('あ'.asInstanceOf[Int]) // => 12354

Python 2014/05/30

ord という関数に長さ1の文字列を渡す。2文字以上渡すとエラーになる。

Python2での例

print ord("A")
# => 65

# 以下は u を付けないとエラーになる
print ord(u"あ")
# => 12354
print ord(u"𪘚")
# => 173594

Python3での例

print(ord("A"))
# => 65

print(ord("あ"))
# => 12354
print(ord("𪘚"))
# => 173594

Ruby 2014/05/30

例

p "A".ord
# => 65
p "あ".ord
# => 12354
p "𪘚".ord
# => 173594

Perl 2014/05/30

例

use utf8;

print ord("A"); # => 65
print ord("あ"); # => 12354 (0x3042)
print ord("𪘚"); # => 173594 (0x2A61A)
