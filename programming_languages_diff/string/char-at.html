インデックスを指定して文字を取得するには (charAt) 2013/09/30

インデックスを指定してその場所の1文字を取得する方法。

インデックスで範囲を指定して部分文字列を取得する方法はsubstr。

Java / Scala 2013/09/07

// String
char charAt(int index)

インデックスは0から始まる。

PHP 2013/10/03

$str[$index];

インデックスは0から始まる。

文字というデータ型はないので、取得した結果も文字列である。従って以下のようなことになる。

$str = "abc";
echo $str[0]; // => a
echo $str[0][0]; // => a
echo $str[0][0][0]; // => a

ただしこれはPHP 5.4以降の話で、 PHP 5.3までだとなぜか $str[0][0] からはエラーになる。 $str[0] が文字列なのは同じなので、なぜだろう？

Python 2013/10/03

文字列の中の文字はリストやタプルと同じように [] でアクセスできる。

str[index]

文字というデータ型はないので、取得した結果も文字列である。

文字列はイミュータブルなので、文字を書き換えることはできない。

Ruby 2013/10/18

str = "abcあいう"

p str[0] # => "a"
p str[4] # => "い"
p str[4].class # => String

文字というデータ型はないので、取得した結果も文字列である。

Perl 2013/09/10

use utf8;

my $s = "あいうえお";
print substr($s, 2, 1);
# => う

use utf8; がないと、ASCII以外の文字を正しく扱えない。
