charAt

Java / Scala 2013/04/17

char String#charAt(int index)

PHP 2013/04/17

str[index];

文字列の中の指定のインデックスの文字を返すが、返り値の型も文字列であるため、 PHP 5.4以降では以下のようなことになる。

$str = "abc";
echo $str[0]; // => a
echo $str[0][0]; // => a
echo $str[0][0][0]; // => a

PHP 5.3までだとなぜか $str[0][0] からはエラーになる。 $str[0] が文字列なのは同じなので、なぜだろう？
