特定の文字や文字列で始まっているかどうか / 終わっているかどうかを見るには (startsWith / endsWith)

Java / Scala 2013/04/10

Signature:

// java.lang.String
String startsWith(String prefix)
String endsWith(String suffix)

PHP 2013/07/23

startsWithに相当するよく使う方法。

strpos($str, $prefix) === 0

endsWithに相当するよく使う方法。

strrpos($str, $suffix) === strlen($str) - strlen($suffix)

ただ、これらは、$strが長い文字列の時にいちいち$strを全部なめてしまい効率よくなさそう。

Python 2013/11/13

str = "abcabc"

print str.startswith("ab")
# => True

print str.startswith("bc")
# => False

print str.endswith("ab")
# => False

print str.endswith("bc")
# => True

Ruby 2013/04/10

str.start_with?(prefix)

str.end_with?(suffix)

Perl 2013/11/13

文字列の先頭にマッチする正規表現の \A または最後にマッチする正規表現 \Z を使う。

例

print "abc" =~ /\Aab/; # => 1

print "abc" =~ /bc\Z/; # => 1

JavaScript 2013/04/10

startsWithに相当するよく使う方法。

str.indexOf(prefix) == 0

ただ、これだと、strが長い文字列の時にいちいちstrの最後までなめるのが効率よくなさそう。

endsWithに相当するよく使う方法。

str.substr(str.length - prefix.length) == prefix

ただ、これだと、prefixがstrより長い場合にどうなることやら。
