startsWith / endsWith

Java / Scala 2013/04/10

String String#startsWith(String prefix)
String String#endsWith(String suffix)

PHP 2013/07/23

startsWithに相当するよく使う方法。

strpos($str, $prefix) === 0

ただ、これだと、$strが長い文字列の時にいちいち$strの最後までなめるのが効率よくなさそう。

endsWithに相当するよく使う方法。

strrpos($str, $suffix) === strlen($str) - strlen($suffix)

これも$strが長い時に効率が悪そう。

Ruby 2013/04/10

String#start_with?(prefix)
String#end_with?(suffix)

Perl 2013/08/21

文字列の先頭にマッチする正規表現の \A、または最後にマッチする \Z を使う。

例

print "abc" =~ /\Aab/; # => 1

print "abc" =~ /bc\Z/; # => 1

JavaScript 2013/04/10

startsWithに相当するよく使う方法。

str.indexOf(prefix) == 0

ただ、これだと、strが長い文字列の時にいちいちstrの最後までなめるのが効率よくなさそう。

endsWithに相当するよく使う方法。

str.substr(str.length - prefix.length) == prefix

ただ、これだと、prefixがstrより長い場合にどうなることやら。
