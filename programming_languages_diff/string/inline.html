文字列に変数・式を埋め込むには

PHP 2013/09/09

" で囲む文字列では中に変数を埋め込むことができる。

例

"http://{$host}/{$path}"

Ruby 2013/10/31

" で囲んだ文字列は、 #{...} を使って式を評価した結果を文字列に埋め込むことができる。 #{...} の中は単純な変数には限らず、関数呼び出しなど任意の式を書ける。

例

require 'date'

puts "It is #{DateTime.now} now."

Perl 2013/11/01

" で囲む文字列では中に変数を埋め込むことができる。

$ または @ で始まる変数が展開される。% は展開されないみたい。

例

"http://$host/$path"

ハッシュの要素の参照もそのまま書ける。

例

"HOME: $ENV{HOME}"

sh 2013/09/17

" で囲む文字列では中に変数を埋め込むことができる。

$ で始まる部分が変数に展開される。変数名の後ろに変数名になりうる文字が続く場合は $ のあとに { } で変数名を囲む。

例

"$VAR/hoge"
"${VAR}_hoge"

JavaScript 2013/10/08

このような機能はたぶんない。

CoffeeScript 2013/09/10

" で囲む文字列では中に式を埋め込むことができる。

例

"http://#{host}/#{path}"
