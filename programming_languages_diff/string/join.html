Go言語 (golang) 2015/02/22

stringsパッケージのJoin関数を使う。

package main

import "fmt"
import "strings"

func main() {
    sl := []string{"abc", "def", "ghi"}
    str := strings.Join(sl, ",")
    fmt.Println(str)
}

Scala 2014/04/07

List などのコレクション型のメソッドでできる。

Signature:

// TraversableOnce[A]
def mkString: String
def mkString(separator: String): String
def mkString(start: String, separator: String, end: String): String

リストの要素が文字列でなくても要素の toString メソッドが呼び出されてから連結される。

PHP 2014/11/04

例

$str = implode($separator, $arr);

implode 関数 | PHP Manual
http://php.net/manual/ja/function.implode.php

実はimplodeの2つの引数はどちらが先に来てもよい。PHPの有名な「歴史的な理由」。

ちなみに (string)$arr とすると、配列の中身に関係なく "Array" という文字列になる。

Ruby / JRuby 2015/01/04

Array#join メソッドまたは Array#* メソッドでできる。

Array#join は引数を省略すると、$, という変数に入っている文字列で連結する。

arr = ['abc', 'def', 'ghi']

p arr.join(',') # => "abc,def,ghi"
p arr * ","     # => "abc,def,ghi"

p arr.join # => "abcdefghi"
$, = "::"
p arr.join # => "abc::def::ghi"

Perl 2013/07/27

例

my @arr = ('abc', 'def', 'ghi');
print @arr;            # => abcdefghi
print "@arr";          # => abc def ghi
print join(' ', @arr); # => abc def ghi
print join(',', @arr); # => abc,def,ghi
