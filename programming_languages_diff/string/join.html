文字列のリストを特定の文字列で連結するには (join / implode) 2013/10/17

逆に文字列を特定の文字で分割するにはsplitを使う。

Scala 2014/04/07

List などのコレクション型のメソッドでできる。

Signature:

// TraversableOnce[A]
def mkString: String
def mkString(separator: String): String
def mkString(start: String, separator: String, end: String): String

リストの要素が文字列でなくても要素の toString メソッドが呼び出されてから連結される。

PHP 2014/11/04

例

$str = implode($separator, $arr);

implode 関数 | PHP Manual
http://php.net/manual/ja/function.implode.php

実はimplodeの2つの引数はどちらが先に来てもよい。PHPの有名な「歴史的な理由」。

ちなみに (string)$arr とすると、配列の中身に関係なく "Array" という文字列になる。

Python 2014/12/30

Pythonでは配列のメソッドではなく、文字列にjoinメソッドがある。セパレータとなる文字列に対してこのメソッドを呼び出す。

separator.join(lst)

str.join | Python 2.7 documentation
http://docs.python.jp/2/library/stdtypes.html?highlight=join#str.join

str.join | Python 3 documentation
http://docs.python.jp/3/library/stdtypes.html?highlight=join#str.join

Ruby / JRuby 2014/12/31

Array#join メソッドまたは Array#* メソッドでできる。

例

arr = ['abc', 'def', 'ghi']

p arr.join(',') # => "abc,def,ghi"
p arr * ","     # => "abc,def,ghi"

Perl 2013/07/27

例

my @arr = ('abc', 'def', 'ghi');
print @arr;            # => abcdefghi
print "@arr";          # => abc def ghi
print join(' ', @arr); # => abc def ghi
print join(',', @arr); # => abc,def,ghi
