文字列の配列を特定の文字列で連結するには (join / implode)

PHP 2013/04/10

implode($separator, $arr)

http://www.php.net/manual/ja/function.implode.php

Ruby 2013/04/10

Array#join(separator)

Perl 2013/07/27

my @arr = ('abc', 'def', 'ghi');
print @arr;            # => abcdefghi
print "@arr";          # => abc def ghi
print join(' ', @arr); # => abc def ghi
print join(',', @arr); # => abc,def,ghi
