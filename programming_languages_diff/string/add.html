文字列の連結をするには (add) 2013/10/28

+ か . という演算子でできる言語が多い。

Java Scala PHP Python Ruby Perl JavaScript
+ + . + + . +

C言語 2013/08/06

strcpy や strcat などを使う。

文字列リテラルを並べて書くと、プリプロセッサによって連結される。

"abc" "def"

Java / Scala 2013/08/06

例

"abc" + "def"

静的な文字列同士の連結はコンパイル時に連結される。

PHP 2013/07/27

例

'abc' . 'def'

Smarty 2014/03/24

例

{$str|cat:'foo'}

Python 2014/03/10

例

'abc' + 'def'

文字列と数字の間で暗黙の型変換をすることはないので、そのまま連結することはできない。

例

print(12 + "42")
# => エラー

print(str(12) + "42")
# => 1242

print(12 + int("42"))
# => 54

文字列リテラルを並べて書くと連結される。改行を挟んでもよい。

例

"abc"
"def"

この2行の文字列は下のと同じになる。

"abc" 'def'

Ruby 2014/01/13

例

'abc' + 'def'

+ 演算子は非破壊的だが、concatメソッドは破壊的に文字列を連結する。

例

str = 'abc'
str.concat('def')

puts str
# => abcdef

Perl 2013/07/27

例

'abc' . 'def'

JavaScript 2013/08/13

例

"abc" + "def"
