文字列の書き方

C言語 2013/08/06

" で囲んで文字列(char[]型)を表す。 ' で囲むと文字列ではなく文字(char型)になる。

Java / Scala 2013/08/06

" で囲んで文字列(String型)を表す。 ' で囲むと文字列ではなく文字(char型)になる。

PHP 2013/06/16

' または " で囲む。またはヒアドキュメントを使う。

" で囲んだ場合とヒアドキュメントでは $ で始まる部分が変数名として値に展開される。

Ruby 2013/08/11

' または " で囲む。

" で囲んだ文字列は、 #{...} を使って式を評価した結果を文字列に埋め込むことができる。

例

now = DateTime.now
puts "It is #{now} now."

Python 2013/08/06

' または " で囲む。

Perl 2013/07/30

' または " で囲む。
