文字列の長さを取得するには (length)

C言語, C++ 2013/09/08

int strlen(char *str);

最後の \x00 はカウントしない。

Java / Scala 2013/04/10

// java.lang.String
int length()

PHP 2013/04/27

int strlen($string)

Python 2013/08/30

例

l = len(str)

Ruby 2013/09/23

String#length
String#size

2つのメソッドは全く同じ。配列のサイズ、 ハッシュのサイズも同じ名前の2つのメソッドがある。

Perl 2013/09/23

例

use utf8;

length($str)

文字列をソースコード中に直接書いた場合は use utf8; としないと、マルチバイトの文字列でもバイト数を数えてしまう。

CoffeeScript 2013/09/07

例

str.length
