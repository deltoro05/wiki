length 2013/09/07

文字列の長さを取得する方法。

Java / Scala 2013/04/10

int String#length()

PHP 2013/04/27

int strlen($string)

Python 2013/08/30

例

l = len(str)

Ruby 2013/05/04

String#length
String#size

2つのメソッドは全く同じ。配列のサイズも同じ名前の2つのメソッドがある。

Perl 2013/08/15

例

use utf8;

length($str)

use utf8; としないと、マルチバイトの文字列でもバイト数を数えてしまう。
