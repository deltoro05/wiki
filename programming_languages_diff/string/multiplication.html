繰り返し 2013/08/11

同じ文字列を繰り返して連結させる方法。乗算の演算子でできる言語が多い。

Ruby 2013/08/11

例

puts "abc" * 3
# => abcabcabc

Perl 2013/08/11

x という演算子で文字列の繰り返しができる。

例

print "abc" x 3;
# => abcabcabc
