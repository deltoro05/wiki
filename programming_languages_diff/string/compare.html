比較

Java 2013/07/31

obj1.equals(obj2) で値を比較する。

== は同じインスタンスかどうかをチェックするもので同じ文字列かどうかはわからない。

Scala 2013/07/31

== は等値の場合に true を返す。 != はその反対。 Javaの equals を呼び出している。

PHP 2013/08/26

== は等値の場合に true を返す。 != はその反対。または strcmp も使える。

if (strcmp($str1, $str2) === 0) {
  // $str1 と $str2 が等しい場合の処理
}

大文字と小文字を区別せずに比較するには strcasecmp を使う。

if (strcasecmp($str1, $str2) === 0) {
  // $str1 と $str2 が等しい場合の処理
}

Ruby 2013/07/14

== は等値の場合に true を返す。 != はその反対。

<, <=, >=, >, <=> で文字列の辞書順の比較ができる。

Perl 2013/07/31

eq は等値の場合に '1' を返し、そうでない場合は、 '' を返す。 ne はその反対。

== は数値として比較してしまう。
