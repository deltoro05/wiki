indexOf

Java / Scala 2013/04/27

int String#indexOf(int ch)
int String#indexOf(String str)

int String#lastIndexOf(int ch)
int String#lastIndexOf(String str)

見つからない場合は -1 を返す。

PHP 2013/04/27

int strpos($mystring, $str)

例：

strpos('abcabc', 'bc') // => 1

見つからない場合は FALSE を返す。

Ruby 2013/04/10

String#index(str)
String#index(str, index)

String#rindex(str)
String#rindex(str, index)

見つからない場合は nil を返す。2つ目の引数を与えると index はその位置以降を探し、rindex はその位置以前を探す。

'abcabc'.index('abc')     # => 0
'abcabc'.index('abc', 1)  # => 3
'abcabc'.index('abc', 3)  # => 3
'abcabc'.index('abc', 4)  # => nil
'abcabc'.rindex('abc')    # => 3
'abcabc'.rindex('abc', 3) # => 3
'abcabc'.rindex('abc', 2) # => 0

JavaScript 2013/04/10

string.indexOf(str)

見つからなかった場合は -1 を返す。
