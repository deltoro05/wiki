文字コードの数字から文字に変換するには 2014/09/28

Unicodeの符号位置の数字から、該当する文字を取得する、各プログラミング言語での方法。例えば65という整数から'A'という文字を取得したい場合。

Scala 2014/04/18

整数を Char にキャストすればよい。

println(65.asInstanceOf[Char]) // => A
println(12354.asInstanceOf[Char]) // => あ

PHP 2013/11/26

echo chr(65); // => A

残念ながらUnicodeはサポートされていない。

chr 関数 | PHP Manual
http://www.php.net/manual/ja/function.chr.php

Python 2014/05/16

Python2での例

print chr(65)         # => A
print unichr(0x3042)  # => あ
print unichr(0x2A61A) # => 𪘚

Python2では、0xFFを超える文字で chr を使おうとすると、 ValueError: chr() arg not in range(256) というエラーになってしまう。代わりに unichr を使う必要がある。

Python3での例

print(chr(65))         # => A
print(chr(0x3042))     # => あ
print(chr(0x2A61A))    # => 𪘚

Python3ではUnicode文字はすべて chr が使える。逆に unichr を使おうとすると、 NameError: name 'unichr' is not defined というエラーになってしまう。

Ruby 2014/05/16

Fixnumクラスのchrメソッドでできるが、0xFFまでしか使えない。代わりに配列のpackメソッドで代用できる。

p 65.chr              # => "A"
p [0x3042].pack('U')  # => "あ"
p [0x2A61A].pack('U') # => "𪘚"
