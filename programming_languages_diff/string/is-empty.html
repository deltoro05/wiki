文字列が空かどうかを取得するには (isEmpty)

Java / Scala 2014/10/06

Javaでの例

if (str.isEmpty()) {
  ...;
}

Scalaでの例

if (str.isEmpty) {
  ...;
}

Signature:

// java.lang.String
boolean isEmpty()

java.lang.String#isEmpty | Java Platform SE 8 Javadoc
http://docs.oracle.com/javase/8/docs/api/java/lang/String.html#isEmpty–

PHP 2013/09/24

例

if (strlen($str) == 0) {
  ...;
}

Python 2013/10/19

if not str:
    ...

条件式の中に文字列を入れると空文字列だけが False の扱いになるためこの書き方ができる。 -> 論理値への変換

Ruby 2013/09/08

例

if str.empty?
  ...
end

Perl 2013/10/25

例

if ($str eq '') {
    ...;
}

sh 2014/05/16

例

if [ -z "$STR" ]; then
    ...
fi

-> シェルスクリプトの演算子
