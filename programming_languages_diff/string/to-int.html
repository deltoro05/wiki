文字列を整数に変換

PHP 2013/06/12

(int)$str

例

(int)"0" // => 0
(int)"-0" // => 0
(int)"1" // => 1
(int)"1.5" // => 1
(int)"-1.5" // => -1
(int)"010" // => 10
(int)"0x10" // => 0
(int)"10 abc" // => 10
