書式化文字列 (format, printf, sprintf)

C言語の printf のような書式の使える言語が多い。 文字列の中に変数を埋め込む書き方を使うことで、細かい書式までは指定できないが printf の代わりに文字列を組み立てることもできる。

C言語, C++ 2013/09/08

int sprintf(char *str, const char *format, ...);
int snprintf(char *str, size_t size, const char *format, ...);

例

char[] dst = new char[1024];
sprintf(dst, "%s %d", str, i);

// バッファオーバーラン(バッファオーバーフロー)を防ぐために
// 文字数の限界を指定できる関数
snprintf(dst, sizeof(dst), "%s %d", str, i);

Java 2013/08/15

例

String.format("%s %d", str, i)

フォーマットのレファレンス
http://docs.oracle.com/javase/7/docs/api/java/util/Formatter.html#syntax

Scala 2013/08/15

例

"%s %d".format(str, i)

フォーマットのレファレンス
http://docs.oracle.com/javase/7/docs/api/java/util/Formatter.html#syntax

Ruby 2013/09/08

%演算子でC言語のsprintfと同様のことができる。

例

"%s %d" % [str, i]

または

sprintf("%s %d", str, i)

フォーマットのレファレンス
http://doc.ruby-lang.org/ja/1.8.7/doc/print_format.html

Perl 2013/09/10

例

sprintf("%s %d", $str, $i)
