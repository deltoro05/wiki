特定の文字や文字列で終わっているかどうかを見るには (endsWith)

始まりを調べるにはstartsWith。

Java / Scala 2013/04/10

Signature:

// String
String endsWith(String suffix)

PHP 2013/07/23

endsWithに相当するよく使う方法。

strrpos($str, $suffix) === strlen($str) - strlen($suffix)

ただ、これだと、$strが長い文字列の時にいちいち$strの最後までなめるのが効率よくなさそう。

Python 2013/10/07

str.endswith(suffix)

Ruby 2013/04/10

String#end_with?(suffix)

Perl 2013/08/21

文字列の最後にマッチする \Z を使う。

例

print "abc" =~ /bc\Z/; # => 1

JavaScript 2013/04/10

endsWithに相当するよく使う方法。

str.substr(str.length - prefix.length) == prefix

ただ、これだと、prefixがstrより長い場合にどうなることやら。
