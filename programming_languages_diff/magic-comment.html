マジックコメント 2013/08/30

Python 2013/08/30

1行目に以下のように書く。シェバングがあれば、その次の行でもよい。

# -*- coding: UTF-8 -*-

非ASCIIコードがあるのにこのマジックコメントがなかった場合にエラーになる。

Ruby 2013/08/30

ソースコードの文字符号化方式(文字コード)は、1行目に以下のように指定する。シェバングがあれば、その次の行でもよい。

Emacs派:

# -*- coding: utf-8 -*-

vi派:

# vim:fileencoding=utf-8

非ASCIIコードがあるのにこのマジックコメントがなかった場合は、Ruby 1.9からはエラーになる。

ソースコード中の文字列リテラルは、このマジックコメントの文字符号化形式でエンコードされた文字列になる。

# -*- coding: UTF-8 -*-
p "abc".encoding         # => #<Encoding:UTF-8>
p "あいうえお".encoding  # => #<Encoding:UTF-8>
