ソースファイル名と行番号をソースコード中で取得するには

PHP 2013/08/23

__FILE__
__DIR__
__LINE__

__DIR__ はPHP 5.3.0以降。

Python 2013/10/12

__file__

行番号はないみたい。

Ruby 2013/09/10

__FILE__
__LINE__

Perl 2013/09/16

__FILE__
__LINE__

Dir::Self を使えば、__DIR__ も利用可能だが、たぶん dirname(__FILE__) と同じ。

use Dir::Self;
__DIR__

sh 2013/07/05

bashでは、行番号は $LINENO という変数で取得できる。
