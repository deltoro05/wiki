2つの変数の内容を入れ替えるには 2013/10/28

多重代入のできる言語であればだいたい簡単にできるのかな。

-> 多重代入

Python 2013/10/28

多重代入を使えば簡単にできる。

a = 1
b = 2
a, b = b, a
print a
# => 2
print b
# => 1

Ruby 2013/09/30

多重代入を使えば簡単にできる。

a = 1
b = 2
a, b = b, a
# a => 2
# b => 1
