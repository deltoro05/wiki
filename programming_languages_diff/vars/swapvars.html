2つの変数の内容を入れ替えるには 2014/12/30

多重代入のできるプログラミング言語であればだいたい簡単にできるのかな。

-> 多重代入

Java 2016/06/30

入れ替えるために一時変数が必要。

int a = 1;
int b = 2;

int tmp = a;
a = b;
b = tmp;

Python 2013/10/28

多重代入を使えば簡単にできる。

a = 1
b = 2
a, b = b, a
print a
# => 2
print b
# => 1

Ruby 2013/09/30

多重代入を使えば簡単にできる。

a = 1
b = 2
a, b = b, a
# a => 2
# b => 1

Perl 2014/12/30

代入の両辺をリストにすれば簡単にできる。

my $a = 1;
my $b = 2;

($a, $b) = ($b, $a);

print "$a $b\n";
# => 2 1
