例外をスローする構文 (throw)

Java 2013/08/22

例

throw new Exception("message ...");

Scala 2013/08/22

例

throw new Exception("message ...");

PHP 2014/05/17

例

throw new Exception("message ...");

Python 2014/03/05

Python2では例外をスローするには、以下のような書き方がある。

raise "message ..."

raise HogeException()

except節で補足した例外を再スローするには、以下のようにシンプルに書ける。

raise

Ruby 2013/12/19

例外をスローするには、以下のような書き方がある。

raise "exception message ..."

raise ZeroDivisionError, "message ..."

2つ目の構文は以下と同じである。

raise ZeroDivisionError.new("message ...")

raiseはRubyの構文ではなく定義済みのメソッドである。

Perl 2015/02/13

die でエラーを発生させる。

die "hogehoge error";

JavaScript 2013/08/22

例

throw "message ...";

CoffeeScript 2013/08/22

例
