例外をスローする構文 (throw) 2016/06/30

各プログラミング言語での、例外をスローする構文の書き方。

Java 2013/08/22

例

throw new Exception("message ...");

Scala 2013/08/22

例

throw new Exception("message ...");

PHP 2014/05/17

例

throw new Exception("message ...");

Python 2014/03/05

raise "message ..."
# または
raise HogeException()

except節で補足した例外を再スローするには、以下のようにシンプルに書ける。

raise

Ruby / JRuby 2013/12/19

raise "exception message ..."
# または
raise ZeroDivisionError, "message ..."

2つ目の構文は以下と同じである。

raise ZeroDivisionError.new("message ...")

raiseはRubyの構文ではなく定義済みのメソッドである。

Perl 2015/02/13

die "hogehoge error";

JavaScript 2013/08/22

throw "message ...";

CoffeeScript 2013/08/22

throw("message")
