例外をスローする構文 (throw)

Java 2013/08/22

throw ...;

Scala 2013/08/22

throw ...;

Ruby 2013/08/22

例外をスローするには、以下のような書き方がある。

raise "exception message ..."

raise ZeroDivisionError, "message ..."

Python 2013/08/22

例外をスローするには、以下のような書き方がある。

raise "message ..."

raise HogeException()

except節で補足した例外を再スローするには、以下のようにシンプルに書ける。

raise

Perl 2013/09/11

die で例外を発生させる。

die "hogehoge error";

JavaScript 2013/08/22

throw "message ...";

CoffeeScript 2013/08/22

throw("message")
