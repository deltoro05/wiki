チェック例外と非チェック例外 2013/08/20

ほとんどの言語ではチェック例外というのはなく、宣言なしに例外をスローすることができる。例外をスローするメソッドに修飾子を付けないといけないのはJavaくらいか。

Java 2013/08/22

チェック例外をスローする可能性のあるメソッドには、メソッドシグニチャにthrowsが必要。チェック例外は Exception でかつ RuntimeException でないもの。非チェック例外は Error または RuntimeException。

-> Javaの例外クラス階層

Scala 2013/07/18

Javaのチェック例外も含めすべての例外が非チェック例外の扱いになる。
