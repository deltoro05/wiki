三項演算子

C言語, C++ 2013/09/17

例

count == 1 ? "foot" : "feet"

右結合であるので以下のような書き方が可能。

count == 0 ? "zero" : count == 1 ? "one" : "many"

Java 2013/09/17

C言語ライクの三項演算子を使える。

count == 1 ? "foot" : "feet"

C言語と同じく右結合。

Scala 2013/08/21

三項演算子がなく、代わりにif式を使う。 Scalaではifは文ではなく、値を返す式。

if(count == 1) "foot" else "feet";

PHP 2013/09/17

C言語ライクの三項演算子を使える。

count == 1 ? "foot" : "feet"

2項目を省略してエルビス演算子として使うこともできる。

他の言語と違いPHPでは左結合なので、三項演算子を入れ子にして使うのはやめたほうがよさそう。

PHPの三項演算子が左結合なのは、PHP言語仕様のバグだ
http://d.hatena.ne.jp/RepsolFireBlade/20130220/1361332690

Perl 2013/07/31

C言語ライクの三項演算子を使える。

$count == 1 ? 'foot' : 'feet'

JavaScript 2013/09/30

C言語ライクの三項演算子を使える。

count == 1 ? 'foot' : 'feet'

C言語と同じく右結合。
