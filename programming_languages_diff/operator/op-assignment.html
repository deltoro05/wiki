複合代入演算子 2014/08/19

a = a + 1 のような、変数の値で演算した結果をその変数自身に再代入する処理を、普通の代入演算子より簡単に書くための演算子が多くのプログラミング言語にあり、これを複合代入演算子など言ったりする。

Java 2014/08/19

以下の演算子がある。

+=
-=
*=
/=
%=
&=
|=
^=
<<=
>>=
>>>=
