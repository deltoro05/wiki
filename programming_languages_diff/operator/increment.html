インクリメント・デクリメント演算子 2013/08/21

整数の変数を1足したり引いたりする演算子。

C言語, C++ 2013/04/10

ある。

++i;
--i;
i++;
i--;
i += 1;
i -= 1;

Java 2013/04/10

ある。

++i;
--i;
i++;
i--;
i += 1;
i -= 1;

Scala 2013/04/10

ない。

i += 1;
i -= 1;

i += 1 などは i = i + 1 などのシンタックスシュガー。

scalaでは、なぜインクリメントやデクリメントができないのか？
http://d.hatena.ne.jp/xuwei/20110130/1296360331

PHP 2013/04/10

ある。

++$i;
--$i;
$i++;
$i--;
$i += 1;
$i -= 1;

Ruby 2013/08/21

ないが、代入演算子はある。

i += 1
i -= 1

オブジェクトだから、というのはわかるが、 i = i + 1 のシンタックスシュガーだ、ということにはできないのかな?

ある。

++$i;
--$i;
$i++;
$i--;
$i += 1;
$i -= 1;

JavaScript 2013/04/10

ある。
