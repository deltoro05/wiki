インクリメント・デクリメント演算子

C言語, C++ 2013/04/10

ある。

++i;
--i;
i++;
i--;
i += 1;
i -= 1;

Java 2013/04/10

ある。

++i;
--i;
i++;
i--;
i += 1;
i -= 1;

Scala 2013/04/10

ない。

i += 1;
i -= 1;

i += 1 などは i = i + 1 などのシンタックスシュガー。

scalaでは、なぜインクリメントやデクリメントができないのか？

PHP 2013/04/10

ある。

++$i;
--$i;
$i++;
$i--;
$i += 1;
$i -= 1;

Ruby 2013/04/10

ない。

i += 1
i -= 1

オブジェクトだから、というのはわかるが、 i = i + 1 のシンタックスシュガーだ、ということにはできないのかな?
