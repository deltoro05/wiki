連想配列(マップ)のキーと値のペアを集合(セット)または配列(リスト)で取得するには (entries)

Python 2014/03/17

dic.items()

例

dic = {'a': 1, 'b': 2}
print(dic.items())
# Python2では
# [('a', 1), ('b', 2)]
# Python3では
# dict_items([('b', 2), ('a', 1)])

for d in dic.items():
    print(d)
# => ('a', 1)
# => ('b', 2)

Ruby / JRuby 2014/12/22

Hash#to_a メソッドを利用すると、長さ2の配列を要素とした配列を取得できる。

hash = {'a' => 10, 'b' => 20, 'c' => 30}

p hash.to_a
# => [["a", 10], ["b", 20], ["c", 30]]
