特定のキーが含まれているかどうかを判定するには (containsKey)

Java 2013/05/06

Signature:

// java.util.Map<K, V>
boolean containsKey(Object key)

Scala 2014/10/27

Signature:

// scala.collection.Map[A, B]
def contains(key: A): Boolean

Javaとメソッド名が違う。

PHP 2013/04/22

array_key_exists($key, $map)

array_key_exists 関数 | PHP Manual
http://www.php.net/manual/ja/function.array-key-exists.php

isset() は NULL 値を持つ配列キーに対して TRUE を返しません。 一方、 array_key_exists() は TRUE を返します。

Python 2014/09/07

Python3ではin演算子を使う。Python2ではin演算子に加えて、has_keyメソッドも同じように使える。どちらもTrueかFalseを返す。

dic = {'a': 1, 'b': 2}

print('a' in dic) # => True
print('x' in dic) # => False

# Python3 ではエラーになる
print(dic.has_key('a')) # => True
print(dic.has_key('x')) # => False

Ruby 2013/05/06

以下の4つはすべて同じ意味。

Hash#has_key?(key)
Hash#key?(key)
Hash#include?(key)
Hash#member?(key)

Perl 2014/02/26

例

if (exists($hash{$key})) {
    ...;
}

レファレンスの場合の例

if (exists($hash_ref->{$key})) {
    ...;
}

defined を使ってしまうと、キーが存在しない場合と、 undef が値として保存されている場合の区別ができなくなってしまう。

JavaScript 2014/01/15

in という演算子があるので、これを使う。

例

var map = {a: 1, b: 2}

console.log('a' in map);
// => true
console.log('c' in map);
// => false
