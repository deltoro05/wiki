特定のキーが含まれているかどうかを判定するには (containsKey)

Java 2013/05/06

Signature:

// java.util.Map<K, V>
boolean containsKey(Object key)

Scala 2013/08/20

Signature:

// scala.collection.Map[A, B]
def contains(key: A): Boolean

PHP 2013/04/22

array_key_exists($key, $map)

array_key_exists | PHP Manual
http://www.php.net/manual/ja/function.array-key-exists.php

isset() は NULL 値を持つ配列キーに対して TRUE を返しません。 一方、 array_key_exists() は TRUE を返します。

Python 2013/10/07

dic.has_key(key)

TrueかFalseを返す。

Ruby 2013/05/06

以下の4つはすべて同じ意味。

Hash#has_key?(key)
Hash#key?(key)
Hash#include?(key)
Hash#member?(key)

Perl 2013/10/20

例

if (defined($hash{$key})) {
    ...;
}

キーが存在しなくても値を取得するときにはエラーにならずに undef が返される。 use warnings; が宣言されていると、 undef が代入された変数を実際に使おうとするときに Use of uninitialized value という警告が表示される。

JavaScript 2014/01/15

in という演算子があるので、これを使う。

例

var map = {a: 1, b: 2}

console.log('a' in map);
// => true
console.log('c' in map);
// => false
