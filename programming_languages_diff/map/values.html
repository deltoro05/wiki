連想配列(マップ)の各エントリの値を集合(セット)または配列(リスト)で取得するには 2015/03/03

各プログラミング言語での、連想配列(マップ)の各エントリの値を集合(セット)または配列(リスト)で取得する方法。

関連

  • キーを集合または配列で取得するには

Scala 2014/12/21

Signature:

// scala.collection.Map[A, B]
def values: Iterable[B]

例

val map = Map("a" -> 10, "b" -> 20, "c" -> 30);

map.values.foreach { key =>
  println(key);
}
// 出力結果
// 10
// 20
// 30

Python 2013/10/04

ディクショナリの各エントリの値をリストで取得する方法

dic.values()

Ruby / JRuby 2014/12/21

ハッシュの各エントリの値を配列で取得する方法

hash.values

例

hash = {'a' => 10, 'b' => 20, 'c' => 30}

p hash.values
# => [10, 20, 30]

Perl 2013/07/28

ハッシュの各エントリの値を配列で取得する方法

my @values = values(%arr);
