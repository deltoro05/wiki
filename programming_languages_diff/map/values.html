連想配列(マップ/ハッシュ/ディクショナリ)の各エントリの値を集合(セット)または配列(リスト)で取得するには (values) 2015/03/03

連想配列(マップ/ハッシュ/ディクショナリ)の各エントリの値を集合(セット)または配列(リスト)で取得する方法。

関連

  • キーを集合または配列で取得するには

Scala 2016/06/14

Signature:

  • def Map[A, B]#values: Iterable[B]
  • def Map[A, B]#valuesIterator: Iterator[B]

例

val map = Map("a" -> 10, "b" -> 20, "c" -> 30);

map.values.foreach { key =>
  println(key);
}
// 出力結果
// 10
// 20
// 30

PHP 2016/08/13

array_values関数は連想配列の値を配列で返す。

PHPの普通の配列は単に整数がキーになっている連想配列であるので、この関数はキーを0から順番に振り直すだけとも言える。

array_values 関数 | PHP Manual
http://php.net/manual/ja/function.array-values.php

Python 2013/10/04

ディクショナリの各エントリの値をリストで取得する方法

dic.values()

Ruby / JRuby 2014/12/21

ハッシュの各エントリの値を配列で取得する方法

hash.values

例

hash = {'a' => 10, 'b' => 20, 'c' => 30}

p hash.values
# => [10, 20, 30]

Perl 2013/07/28

ハッシュの各エントリの値を配列で取得する方法

my @values = values(%arr);
