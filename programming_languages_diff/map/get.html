キーで参照するには

Java 2013/09/03

map.get("a");

Scala 2013/09/03

map("a")

Option で取得するには

map.get("a")

要素が存在しない場合のデフォルト値を設定して、要素を取得するには

map.getOrElse("a", 0);

シグニチャ

// Map[A, B]
def getOrElse(key: A, default: => B): B

2つ目の引数がデフォルト値となる。要素が存在する場合には、2つ目の引数が評価されないので、計算に時間のかかるデフォルト値を設定してもよい。

PHP 2013/09/03

arr['a']

Ruby 2013/09/04

hash['a']

または

hash.fetch('a')

存在しない要素を取得しようとすると [] は nil が返ってくるが、 fetch は KeyError の例外が発生する。

fetch は第2引数かブロックを与えることができ、その場合は、要素が存在しない場合に、第2引数を返すか、ブロックを評価した結果を返す。

例

m = {'a' => 1}

p m.fetch('a') # => 1
p m.fetch('a', 2) # => 1
p m.fetch('b', 2) # => 2
p m.fetch('b') { 2 } # => 2

Perl 2013/09/03

以下のように {} でキーを指定し、先頭は $ を使う。

my %hash = (a => 1, b => 2);

$hash{a} # => 1
$hash{'a'} # => 1

ハッシュへのレファレンスから要素を取り出すには

my $hash = {a => 1, b => 2};

$hash->{a} # => 1
