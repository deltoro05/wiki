連想配列(マップ)の要素の数を取得するには (size)

Java 2013/05/06

Signature:

// java.util.Map<K, V>
int size()

Scala 2014/10/20

Signature:

// scala.collection.Map[A, B]
def size: Int

PHP 2014/10/20

PHPの連想配列は普通の配列と区別がつかないので、配列と同様に以下の2つの関数を使える。

例

count($map)
sizeof($map)

この2つの関数はまったく同じ意味で、配列の要素の数を返す。

count 関数 | PHP Manual
http://www.php.net/manual/ja/function.count.php

sizeof 関数 | PHP Manual
http://www.php.net/manual/ja/function.sizeof.php

Python 2014/10/20

len(dic)

len という関数は、シーケンス型といわれるリスト、タプル、文字列に対しても使える。

Ruby / JRuby 2014/10/20

例

l = hash.length
# または
l = hash.size

2つのメソッドは全く同じ。配列のサイズ、 文字列の長さも同じ名前の2つのメソッドがある。

Hash#length メソッド, Hash#size メソッド | Ruby 2.1 リファレンスマニュアル
http://docs.ruby-lang.org/ja/2.1.0/class/Hash.html#I_LENGTH

Perl 2013/09/14

キーの配列をスカラコンテキストで参照すると、要素の数を取得できる。

my %hash = (a => 1, b => 2);
print scalar keys %hash;
