連想配列(マップ/ハッシュ/ディクショナリ)の要素数を取得するには (size / count / sizeof / len / length) 2015/03/05

連想配列(マップ/ハッシュ/ディクショナリ)の要素数を取得する方法。

関連

  • 配列(リスト)の要素数を取得するには

Java 2016/06/14

Signature:

  • int Map<K, V>#size()

Scala 2016/06/14

Signature:

  • def Map[A, B]#size: Int

PHP 2015/03/25

PHPの連想配列は普通の配列と区別がつかないので、配列と同様に以下の2つの関数を使える。

count($arr)
sizeof($arr)

この2つの関数はまったく同じ意味で、配列の要素の数を返す。

count 関数 | PHP Manual
http://php.net/manual/ja/function.count.php

sizeof 関数 | PHP Manual
http://php.net/manual/ja/function.sizeof.php

Python 2014/12/26

l = len(dic)

len という関数は、シーケンス型といわれるリスト、タプル、文字列に対しても使える。

Ruby / JRuby 2014/10/20

例

l = hash.length
# または
l = hash.size

2つのメソッドは全く同じ。配列のサイズ、 文字列の長さも同じ名前の2つのメソッドがある。

Hash#length メソッド, Hash#size メソッド | Ruby 2.1 リファレンスマニュアル
http://docs.ruby-lang.org/ja/2.1.0/class/Hash.html#I_LENGTH

Perl 2014/12/26

キーの配列をスカラコンテキストで参照すると、要素の数を取得できる。

my %hash = (a => 1, b => 2);
print scalar keys %hash;
# => 2

JavaScript 2015/07/18

Object.keys という関数でキーの配列にして、length を取得すればよい。

var len = Object.keys(obj).length;
