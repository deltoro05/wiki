連想配列(マップ)のデータ型を相互に変換するには (asScala / asJava) 2015/03/23

各プログラミング言語での、連想配列(マップ)のデータ型を相互に変換する方法。といってもScalaしか書いてないけど。

Scala 2013/10/14

例

import scala.collection.JavaConverters._;

// JavaのMapからScalaのミュータブルなMapに変換
val mutableMap = javaMap.asScala;

// ScalaのミュータブルなMapからイミュータブルなMapに変換
val immutableMap = Map(mutableMap.toSeq:_*);
