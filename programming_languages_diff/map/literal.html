マップ(連想配列)の書き方

Java 2013/04/17

マップは一発では作れない。

import java.util.HashMap;
map = new HashMap();
map.put("a", 1);
map.put("b", 2);

キーで参照するには

map.get("a");

Scala 2013/05/06

map = Map("a" -> 1, "b" -> 2);

これは以下のシンタックスシュガー。

map = scala.collection.immutable.Map.apply(scala.Predef.any2ArrowAssoc("a").->(1),
                                           scala.Predef.any2ArrowAssoc("a").->(2));

キーで参照するには

map("a")

Option で取得するには

map.get("a")

PHP 2013/05/08

arr = array('a' => 1, 'b' => 2);

PHP 5.4からは以下のようにも書ける。

arr = ['a' => 1, 'b' => 2];

キーで参照するには

arr['a']

Ruby 2013/05/06

hash = {'a' => 1, 'b' => 2}

キーで参照するには

hash['a']

Python 2013/04/17

map = {'a': 1, 'b': 2}

Perl 2013/07/27

my %map = (a => 1, b => 2);

ハッシュへのレファレンスは

my $map = {a => 1, b => 2};

で、これは以下と同じ。

my %m = (a => 1, b => 2);
my $map = \%m;

レファレンスの実体を取り出すには、

%$map;

JavaScript 2013/04/15

下の2つは全部同じ連想配列を生成する。

map = {a: 1, b: 2}
map = {'a': 1, 'b': 2}

CoffeeScript 2013/04/15

map = {a: 1, b: 2}
map = a: 1, b: 2
map =
    a: 1
    b: 2
