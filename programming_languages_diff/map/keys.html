連想配列(マップ)のキーを集合(セット)または配列(リスト)で取得するには (keys)

Java 2013/09/21

Signature:

// java.util.Map[K, V]
java.util.Set<K> keySet()

Scala 2014/12/21

Signature:

// scala.collection.Map[A, B]
def keys: Iterable[A]
def keySet: Set[A]

例

val map = Map("a" -> 10, "b" -> 20, "c" -> 30);

println(map.keySet);
// 出力結果
// Set(a, b, c)

map.keys.foreach { key =>
  println(key);
}
// 出力結果
// a
// b
// c

Python 2013/10/04

ディクショナリのキーをリストで取得する方法

dic.keys()

Ruby / JRuby 2014/12/21

ハッシュのキーを配列で取得する方法

hash.keys

例

hash = {'a' => 10, 'b' => 20, 'c' => 30}

p hash.keys
# => ["a", "b", "c"]

Perl 2013/09/06

ハッシュのキーを配列で取得する方法

my @keys = keys(%hash);
