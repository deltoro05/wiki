概要 2013/10/05

キーと値の2つのオブジェクトのペアを複数保持し、キーからオブジェクトを取得できるコンテナオブジェクト。配列の数字を添字とするものに対し、任意のオブジェクトを添字にできるもの。連想配列、マップ、ディクショナリなど、言語によって呼び方は違う。

キーと値の型は違っていてもよいが、キーと値はそれぞれすべて同じ型である必要がある。ただ、型が違うものがまざっていてもよい言語もある。

PHP 2013/10/05

連想配列が組み込み型として存在する。連想配列のキーに整数を使えば、単純な配列として使うことができ、連想配列と通常の配列の区別がない。

Python 2013/10/07

ディクショナリ(辞書)というデータ型がある。

キーにはイミュータブル(不変)なオブジェクトであればなんでもよいので、 タプルもキーに使える。

ディクショナリ自体はミュータブル(可変)である。

Perl 2013/09/21

ハッシュという。偶数個の要素のリストをハッシュの変数(%で始まる名前)に代入するとハッシュになる。

JavaScript 2013/10/05

オブジェクトがそのまま連想配列のように使える。 JavaScriptにおいてオブジェクトと連想配列は同じ。
