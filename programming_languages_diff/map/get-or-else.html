getOrElse 2013/08/21

要素をキーで取得する際に、該当する要素がない場合に特定の値をデフォルト値とさせるには。

Scala 2013/08/20

// Map[A, B]
def getOrElse(key: A, default: => B): B

Ruby 2013/08/20

fetch はハッシュの要素をキーで取得するが、該当する要素がない場合に、第2引数を返すか、ブロックを評価した結果を返す。

例

m = {'a' => 1}

p m.fetch('a') # => 1
p m.fetch('a', 2) # => 1
p m.fetch('b', 2) # => 2
p m.fetch('b') { 2 } # => 2
