連想配列(マップ)が空かどうかを判定するには (isEmpty)

連想配列(マップ)が空かどうかを判定するメソッドは、配列と同じ名前のメソッドになっていることが多いかもしれない。

-> 配列(リスト)が空かどうか

Java 2013/10/14

Signature:

// java.util.Map<K, V>
boolean isEmpty();

Scala 2013/08/20

Signature:

// scala.collection.Map[A, B]
def isEmpty: Boolean

Python 2014/12/13

ディクショナリはif文などの条件式に入れると、空のときに false、そうでないときに true の扱いになる。

-> 論理値への変換

Ruby / JRuby 2014/12/13

hash.empty?

Perl 2014/12/13

ハッシュはif文などの条件式に入れると、空のときに false、そうでないときに true の扱いになる。

-> 論理値への変換

例

if (%map) {
    # 空でない場合の処理
} else {
    # 空の場合の処理
}

if (!%map) {
    # 空の場合の処理
}
