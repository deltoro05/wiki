連想配列(マップ)の要素の値を設定するには (put)

Java 2013/10/10

Signature:

// java.util.Map<K, V>
void put(K key, V value)

Scala 2014/05/09

Signature:

// scala.collection.Map[A, +B]
def +(kv: (A, B)): Map[A, +B]

PHP 2013/10/08

例

arr['a'] = 10;

Python 2014/12/13

ディクショナリの要素をキーで指定して値を設定する方法

dic['a'] = 10

Ruby / JRuby 2014/12/13

例

hash['a'] = 10

Perl 2013/10/08

以下のように {} でキーを指定し、先頭は $ を使う。

$hash{a}   = 10;
# または
$hash{'a'} = 10;
