連想配列(マップ)の要素の値を設定するには (put / +) 2015/03/04

各プログラミング言語での、連想配列(マップ)の要素の値を設定する方法。

Java 2013/10/10

Signature:

void java.util.Map<K, V>#put(K key, V value)

Scala 2014/05/09

Signature:

def scala.collection.Map[A, +B]#+(kv: (A, B)): Map[A, +B]

PHP 2013/10/08

例

arr['a'] = 10;

Python 2015/03/23

ディクショナリの要素は [] でアクセスして代入すれば、値を設定できる。

dic['a'] = 10

Ruby / JRuby 2015/03/23

ハッシュの要素は [] でアクセスして代入すれば、値を設定できる。

hash['a'] = 10

Perl 2014/12/22

{} でキーを指定し、変数名の先頭は $ を使う。

ハッシュでの例

$hash{a}   = 10;
# または
$hash{'a'} = 10;

ハッシュへのレファレンスでの例

$hash->{a}   = 10;
# または
$hash->{'a'} = 10;
