キーで要素を設定するには (put)

Java 2013/10/10

Signature:

// java.util.Map<K, V>
void put(K key, V value)

PHP 2013/10/08

例

arr['a'] = 10;

Python 2013/10/04

例

dic['a'] = 10

Ruby 2013/12/02

例

hash['a'] = 10

Perl 2013/10/08

以下のように {} でキーを指定し、先頭は $ を使う。

$hash{a}   = 10;
# または
$hash{'a'} = 10;
