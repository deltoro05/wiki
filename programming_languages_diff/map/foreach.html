  • プログラミング言語の比較
    • マップ(連想配列)
      • 概要
      • マップの書き方
      • マップの要素の数を取得するには (size)
      • キーで値を参照するには (get)
      • キーで要素を設定するには (put)
      • マップの要素を1つずつ順番に処理するには (foreach)
      • 特定のキーが含まれているかどうかを判定するには (containsKey)
      • マップの種類を相互に変換するには (convert)
      • マップをコピーするには (clone)
      • マップが空かどうかを判定するには (isEmpty)
      • マップのキーをセットまたはリストで取得するには (keys)
      • マップにマップで複数の要素を追加するには (putAll)
      • 特定の要素をキーで指定して削除するには (remove)
      • マップの各エントリの値をセットまたはリストで取得するには (values)

マップの要素を1つずつ順番に処理するには (foreach)

Java 2013/08/01

Java5からは、以下の拡張for文(foreach構文)の利用が可能。

for(String key: map.keySet()){
  String value = map.get(key);
  ...;
}

PHP 2013/04/27

foreach($arr as $key => $value){
  ...
}

Ruby 2013/05/06

ハッシュの全エントリに対してキーのみを引数としてブロックを実行

hash.each_key do |key|
  ...
end

ハッシュの全エントリに対して値のみを引数としてブロックを実行

hash.each_value do |value|
  ...
end

ハッシュの全エントリに対してキーと値を引数としてブロックを実行

hash.each_pair do |key, value|
  ...
end

Perl 2013/10/17

キーの配列を取得して foreach で回す方法

foreach my $key (keys %hash) {
  ...;
}

キーと値のペアを each を使って回す方法

while ((my $key, my $value) = each(%hash)) {
  ...;
}
