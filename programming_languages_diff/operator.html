インクリメント・デクリメント演算子

C言語, C++

ある。

++i;
--i;
i++;
i--;
i += 1;
i -= 1;

Java

ある。

++i;
--i;
i++;
i--;
i += 1;
i -= 1;

Scala

ない。

i += 1;
i -= 1;

i += 1 などは i = i + 1 などのシンタックスシュガー。

ある。

比較演算子

Ruby

ソートに便利な比較の演算子

a <=> b

a の方が小さければ、-1 を返し、大きければ 1 を返し、等しければ 0 を返す。
