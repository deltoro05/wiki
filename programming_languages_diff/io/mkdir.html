ディレクトリを作成するには

Java 2013/08/21

boolean java.io.File#mkdir()
ディレクトリを作成する。作成するパスの File オブジェクトを先に生成してから このメソッドを呼び出す。作成できたら true を返す。
boolean java.io.File#mkdirs()
上の mkdir とほぼ同等だが、mkdir -p に相当する。

Ruby 2013/06/24

Dir::mkdir(dirname)

または、mkdir -p に相当する方法として

require 'fileutls'
FileUtils::mkdir_p(dirname)

Perl 2013/08/17
