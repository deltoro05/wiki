標準出力・標準エラー出力するには

Ruby 2014/01/19

# 標準出力
print "OK\n"
puts "OK"
STDOUT.print "OK\n"
STDOUT.puts "OK"

# 標準エラー出力
STDERR.print "Error\n"
STDERR.puts "Error"

puts だと最後に自動で改行される。

Perl 2014/01/15

# 標準出力
print "OK\n";
print STDOUT "OK\n";

# 標準エラー出力
print STDERR "Error\n";
warn "Error\n";

STDOUT, STDERR はファイルハンドル。ちなみに標準入力のファイルハンドルは STDIN。

print の使い方はファイルに書き込むのと同じ。-> print

warn は標準エラー出力する。出力する文字列に改行がなかった場合は行番号を出力し、その点は die に似ている。
