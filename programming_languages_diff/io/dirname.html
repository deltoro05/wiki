パスからディレクトリ名を取得するには (dirname) 2013/08/17

ディレクトリの部分を取り出す方法。ファイルパスからファイル名の部分を取り出す方法はbasename。

PHP 2013/09/13

dirname($path);

dirname | PHPマニュアル http://www.php.net/manual/ja/function.dirname.php

Python 2013/09/01

import os

print os.path.dirname(path)

Ruby 2013/08/21

File.dirname(path)

Perl 2013/08/24

例

use File::Basename qw/dirname/;

print dirname("/aaa/bbb") . "\n"; # => /aaa
print dirname("/aaa/") . "\n";    # => /
print dirname("/aaa") . "\n";     # => /
print dirname("aaa") . "\n";      # => .

sh 2013/08/24

dirnameコマンドは引数で与えられたパスのディレクトリ名を標準出力に出力する。

例

dirname /aaa/bbb # => /aaa
dirname /aaa/    # => /
dirname /aaa     # => /
dirname aaa      # => .
