パスからディレクトリ名を取得するには 2013/08/17

ディレクトリの部分を取り出す方法。ファイルパスからファイル名の部分を取り出す方法はbasename。

Ruby 2013/08/21

File.dirname(path)

Perl 2013/08/17

例

use File::Basename qw/dirname/;

print dirname("/aaa/bbb") . "\n"; # => /aaa
print dirname("/aaa/") . "\n"; # => /
print dirname("/aaa") . "\n"; # => /
print dirname("aaa") . "\n"; # => ./
