ファイル名を変更するには

Java / Scala 2014/10/09

Java/Scalaでの例

import java.io.File;

if (! new File(oldName).renameTo(new File(newName))) {
  throw new IOException();
}

Ruby 2015/08/22

File.rename('foo.txt', 'bar.txt')

または

require 'fileutils'
FileUtils.mv('foo.txt', 'bar.txt')

Perl 2015/08/21

renameという関数でリネームできる。

rename($oldName, $newName);
