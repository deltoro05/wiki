ファイルサイズを取得するには

-> ファイルサイズが0かどうかを取得

Python 2013/10/14

import os.path

print os.path.getsize(path)

pathがシンボリックリンクの場合は、リンク先のファイルサイズを取得する。

Ruby 2013/05/28

File::stat('./hoge.txt').size

Perl 2013/08/19

-s という演算子を利用する。

print -s './hoge.txt'; # => 整数が出力される
