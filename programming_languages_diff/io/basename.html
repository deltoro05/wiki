パスからファイル名を取得するには 2013/08/17

ファイルパスからファイル名の部分を取り出す方法。ディレクトリの部分を取り出す方法はdirname。

PHP 2013/09/13

basename($path);

basename | PHPマニュアル http://www.php.net/manual/ja/function.basename.php

Ruby 2013/08/29

File.basename(path)

Perl 2013/08/17

例

use File::Basename qw/basename/;

print basename("/aaa/bbb") . "\n"; # => bbb
print basename("/aaa/") . "\n"; # => aaa
print basename("/aaa") . "\n"; # => aaa
print basename("aaa") . "\n"; # => aaa
