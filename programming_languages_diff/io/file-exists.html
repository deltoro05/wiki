ファイルまたはディレクトリが存在するかどうかを取得するには

PHP 2013/07/24

file_exists('/home/foo/testfile')

シンボリックリンクの場合は、シンボリックリンクの指す先が存在するかどうかをチェックする。

file_exists | PHPマニュアル
http://www.php.net/manual/ja/function.file-exists.php

Ruby 2013/10/09

File.exists?('/home/foo/testfile')

シンボリックリンクの場合は、リンク先が存在するかどうかをチェックする。リンク先が存在しなければリンクそのものが存在していても false を返す。

Perl 2013/11/22

if( -e '/home/foo/testfile' ){
    ...;
}

シンボリックリンクの場合は、リンク先が存在するかどうかをチェックする。リンク先が存在しなければリンクそのものが存在していても -e '/home/foo/testfile' は undef になる。存在すれば -e '/home/foo/testfile' は 1 になる。

sh 2013/07/27

if [ -e /home/foo/testfile ]; then
    ...
fi
