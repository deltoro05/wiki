PHP 2014/10/16

opendir関数でディレクトリハンドルを開いて、readdir関数で1つずつ読み取っていく。

$lst = array();
$dhandle = opendir($dir);
if ($dhandle) {
  while (false !== ($fname = readdir($dhandle))) {
    $lst[] = $fname;
  }
  closedir($dhandle);
}

Perl 2013/11/22

例

my @files = glob('/home/foo/*');
@files = map { if ( /\/([^\/]+)\Z/ ){ $1; } else { $_; } } @files;

1行目だけだと、フルパスでファイル一覧が取得されるので、２行目の正規表現でファイル名の部分を取り出している。
