ファイルの中身を1行ずつテキスト処理するには

Java 2013/10/08

import java.io.BufferedReader;
import java.io.FileInputStream;
import java.io.InputStreamReader;

BufferedReader reader = new BufferedReader(
    new InputStreamReader(new FileInputStream(tsvFilePath), "UTF-8"));
while (true) {
    String line = reader.readLine();
    if (line == null)
        break;
    // line には最後の改行を含まない
    ...;
}
fclose(reader);

BufferedReader#readLine() は "\n", "\r", "\r\n" のいずれかを改行としてみなす。

PHP 2013/10/08

$fh = fopen('./hoge.txt', 'r');
if($fh){
  while (!feof($fh)) {
    $line = fgets($fh);
    // $line には最後の改行を含む
    ...;
  }
  fclose($fh);
}

fgets は '\r'(0x0D) を改行としてみなさない。

Ruby 2013/08/22

IO.foreach('./hoge.txt') do |line|
  ...
end

または

open('./hoge.txt') do |fh|
  fh.each do |line|
    // line には最後の改行を含む
    ...
  end
end

または

open('./hoge.txt') do |fh|
  while line = fh.gets
    // line には最後の改行を含む
    ...
  end
end

Perl 2013/09/20

open(IN, '<', './hoge.txt') or die "Cannot open";
while (my $line = <IN>) {
    # $line には最後の改行を含む
    ...;
}
close IN;

openせずに <IN> の代わりに <STDIN> と書けば、標準入力を1行ずつ処理できる。
