ファイルの中身を1行ずつテキスト処理するには

PHP 2013/07/18

$fh = fopen('./hoge.txt', 'r');
if($fh){
  while (!feof($fh)) {
    $line = fgets($fh);
    // $line には最後の改行を含む
    ...;
  }
  fclose($fh);
}

Ruby 2013/08/22

IO.foreach('./hoge.txt') do |line|

end

または

open('./hoge.txt') do |fh|
  fh.each do |line|
    // line には最後の改行を含む
    ...
  end
end

または

open('./hoge.txt') do |fh|
  while line = fh.gets
    // line には最後の改行を含む
    ...
  end
end

Perl 2013/09/20

open(IN, '<', './hoge.txt') or die "Cannot open";
while (my $line = <IN>) {
    # $line には最後の改行を含む
    ...;
}
close IN;

openせずに <IN> の代わりに <STDIN> と書けば、標準入力を1行ずつ処理できる。
