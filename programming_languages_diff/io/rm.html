ファイル・ディレクトリを削除するには (rm / unlink / rmdir)

PHP 2015/07/27

unlink('./foo.txt');

空のディレクトリを削除するには rmdir 関数を使う。

rmdir('./bar');

unlink 関数 | PHP Manual
http://php.net/manual/ja/function.unlink.php

rmdir 関数 | PHP Manual
http://php.net/manual/ja/function.rmdir.php

中身が入っている可能性のあるディレクトリをまるごと削除する関数はなさそう。以下のページ参照。

PHP - ディレクトリを再帰的に削除する方法のメモ | Qiita

Ruby 2013/05/04

File.unlink('./foo.txt')

Perl 2014/07/21

unlink('./foo.txt');

複数まとめて削除することもできる。

unlink('./foo.txt', './foo.txt');
unlink(@files);

空のディレクトリを削除するには rmdir 関数を使う。

rmdir('./bar);

中身が入っている可能性のあるディレクトリをまるごと削除するには File::Path の rmtree 関数を使う。

rmtree('./bar);
