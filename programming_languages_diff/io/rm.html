ファイルを削除するには (rm / unlink)

PHP 2013/12/14

unlink('./hoge.txt');

unlink | PHPマニュアル http://www.php.net/manual/ja/function.unlink.php

Ruby 2013/05/04

File.unlink('./hoge.txt')

Perl 2014/07/21

unlink('./hoge.txt');

複数まとめて削除することもできる。

unlink('./hoge.txt', './foo.txt');
unlink(@files);

空のディレクトリを削除するには rmdir 関数を使う。

rmdir('./bar);

中身が入っている可能性のあるディレクトリをまるごと削除するには File::Path の rmtree 関数を使う。

rmtree('./bar);
