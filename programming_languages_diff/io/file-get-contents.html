ファイルの中身を1つの文字列として取得するには

PHP 2013/08/01

str = file_get_contents('./hoge.txt');

失敗した場合は FALSE を返される。

http://www.php.net/manual/ja/function.file-get-contents.php

Ruby 2013/08/21

str = File.read('./hoge.txt')

または

str = File.open('./hoge.txt').read

Perl 2013/09/18

open(IN, '<', './hoge.txt') or die "Cannot open";
my @lines = <IN>;
# @lines には行単位の配列で、各行には最後の改行を含む
close IN;

手抜きでcatコマンドを使う方法

my $str = `cat ./hoge.txt`;

sh 2013/09/16

str=`cat ./hoge.txt`
