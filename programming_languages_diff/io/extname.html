パスやファイル名から拡張子を取得するには (extname) 2013/09/16

ファイルパスからファイル名の部分を取り出す方法はbasename。ディレクトリの部分を取り出す方法はdirname。

PHP 2013/10/15

var_export(pathinfo("/abc/def.txt", PATHINFO_EXTENSION));
// => 'txt'

pathinfo | PHPマニュアル
http://www.php.net/manual/ja/function.pathinfo.php

Ruby 2013/08/21

File.extname(path)

拡張子がない場合は空文字列を返す。拡張子がある場合は '.' から始まる拡張子部分の文字列を返す。
