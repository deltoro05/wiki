while文

C言語, C++ 2013/07/14

while(...){
  ...
}

本体の文が1つだけの場合は {} を省略可。

while(...)
  ...

永久ループは

while(true){
  ...
}

または

for(;;){
  ...
}

Go言語 2013/06/21

for ... {
  ...
}

本体の文が1つだけの場合でも {} を省略不可。

永久ループは

for {
  ...
}

Java 2013/07/14

while(...){
  ...;
}

本体が式1つだけの場合は {} を省略可

while(...)
  ...;

永久ループは

while(true){
  ...;
}

Scala 2013/07/14

while(...){
  ...;
}

本体が式1つだけの場合は {} を省略可

while(...)
  ...;

Scalaのif式などと違い、whileは常に Unit を返す。

break や continue は存在しない。再帰関数を使うなどする。

永久ループは

while(true){
  ...;
}

Ruby 2013/06/21

while ... do
  ...
end

do は省略可能。

永久ループは

while true do
  ...
end

Python 2013/06/30

while ...:
    ...
    ...

Perl 2013/08/01

while (...) {
    ...;
}

条件が否定になった以下の形式もある。

until (...) {
    ...;
}

doループも使える。

do {
    ...;
} while (...);

do {
    ...;
} until (...);

修飾子の書き方も可能。

... while ...;
... until ...;

CoffeeScript 2013/06/24

while ...
  ...

または1行にまとめて

... while ...

ループの中では break や continue が使える。
