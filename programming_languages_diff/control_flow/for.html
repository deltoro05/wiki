for文(繰り返し) 2013/10/14

特定の条件をみたしている間同じ処理を繰り返す構文。 while文でもループできる。

配列やリストの要素に対して繰り返し処理をさせる方法は foreachも参照。

C言語, C++ 2013/06/21

for(int i = 0; i < 10; i++){
    ...;
}

以下の2つは同じ意味になる。

for(式1; 式2; 式3){
    ...;
}

または

式1;
while(式2){
    ...;
    式3;
}

Java 2013/07/14

C言語ライクのfor文を使える。

for(int i = 0; i < 10; i++){
    ...;
}

Scala 2013/09/21

for(i <- 0 until 10){
    ...;
}

またはforeachを使って

(0 until 10).foreach { i =>
    ...;
}

PHP 2013/07/14

C言語ライクのfor文を使える。

for($i = 0; $i < 10; $i++){
    ...;
}

Python 2014/02/03

0から9までを繰り返す例

for i in range(10):
    ...

range は0から始まる指定の長さのシーケンスを返す。

Ruby 2013/12/08

0 から n-1 までを繰り返すには

n.times do |i|
  ...
end

または

for i in 0..(n-1) do
  ...
end

または

for i in (0).upto(n-1) do
  ...
end

逆順にするには

for i in (n-1).downto(0) do
  ...
end

引数を伴った break でfor文全体の返り値を指定することができる。 break で指定しなかった場合は、for文は、in で指定した値の each メソッドの返り値を返す。

Perl 2013/08/01

C言語ライクのfor文を使える。

for(my $i = 0; $i < 10; $i++){
    ...;
}

$i のスコープはfor文の中のみである。

JavaScript 2013/07/27

C言語ライクのfor文を使える。

for(var i = 0; i < 10; i++){
    ...;
}

var i と書いているが、スコープはfor文を含む関数内であり、for文に限定できるものではない。
