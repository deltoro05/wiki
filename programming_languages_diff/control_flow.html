if文

if ... then
  ...
elsif ... then
  ...
else
  ...
end

bash 2013/04/10

if [ ... ]; then
    ...
elif [ ... ]; then
    ...
else
    ...
fi

CoffeeScript 2013/04/10

if ... then ... else ...

if ...
    ...
else if ...
    ...
else
    ...

elseだけを1行にまとめることができる。

if ...
    ...
else if ...
    ...
else ...

3項演算子としても使える。

a = if ... then ... else ...

Rubyの用に修飾子としても使える。

alert "hoge" if condition

switch文

C言語, C++ 2013/05/15

switch(x){
case 1:
  ...
  break;
case 2:
  ...
  break;
default:
  ...
  break;
}

break を書かないとswitch文から脱出せずに次のラベルを引き続き実行してしまう。(fall through)

Java 2013/05/15

switch(x){
case 1:
  ...
  break;
case 2:
  ...
  break;
default:
  ...
  break;
}

break を書かないとswitch文から脱出せずに次のラベルを引き続き実行してしまう。(fall through)

x には、byte, short, char, int, Byte, Short, Character, Integer, enum, String が使える。

String が使えるのは、Java SE 7 から。ただ、String の場合、 null だと NullPointerException が発生してしまい、caseで null を指定したり、 defaultに飛んでくれたりはしない。 Integer の場合も、NullPointerException が発生しうるのは同様。

Byte, Short, Character, Integer が使えるのは、たぶん autoboxing が導入された Java SE 5 から。

Scala 2013/05/15

x match {
  case 1 =>
    ...
  case 2 =>
    ...
  case 3 =>
    ...
}

C言語やJavaのように　fall through することはない。
