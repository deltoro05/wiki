if文

Ruby

if ... then
  ...
elsif ... then
  ...
else
  ...
end

bash

if [ ... ]; then
    ...
elif [ ... ]; then
    ...
else
    ...
fi

CoffeeScript

if ... then ... else ...

if ...
    ...
else if ...
    ...
else
    ...

elseだけを1行にまとめることができる。

if ...
    ...
else if ...
    ...
else ...

3項演算子としても使える。

a = if ... then ... else ...

Rubyの用に修飾子としても使える。

alert "hoge" if condition

for文(繰り返し)

-> プログラミング言語の比較 > リスト(配列) > foreach
-> プログラミング言語の比較 > マップ(連想配列) > foreach

C言語, C++

for(int i = 0; i < 10; i++){
    ...
}

以下の2つは同じ意味になる。

for(式1; 式2; 式3){
    ...
}

式1;
while(式2){
    ...
    式3;
}

Ruby

0 から n-1 までを繰り返すには

for i in 0..(n-1) do
  ...
end
