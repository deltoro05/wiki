制御構文

  • if文
  • for文(繰り返し)

if文

bash

if [ ... ]; then
    ...
elif [ ... ]; then
    ...
else
    ...
fi

CoffeeScript

if ... then ... else ...

if ...
    ...
else if ...
    ...
else
    ...

elseだけを1行にまとめることができる。

if ...
    ...
else if ...
    ...
else ...

3項演算子としても使える。

a = if ... then ... else ...

Rubyの用に修飾子としても使える。

alert "hoge" if condition

for文(繰り返し)

Java

配列をループ。

for(int i = 0; i < arr.length; i++){
  ...
}

Java5からは、拡張for文の利用が可能。

for(String str: str_arr){
  ...
}

PHP

配列をループ。

for(int $i = 0; $i < $arr.length; $i++){
  ...
}

foreach($arr as $value){
  ...
}

連想配列をループ。

foreach($arr as $key => $value){
  ...
}

Ruby

for value in arr do
  ...
end

value の部分には複数の値が入る場合に複数の変数が使える。例：

for i, str in [[1, 'abc'], [2, 'def'], [3, 'ghi']]
  ...
end

配列の each メソッドも使える。 break でのループの中断は、上記の for 文以外にこの each メソッドでも使える。例：

[1, 2, 3].each do |i|
  p i
  break if i == 2
end
# => 1 2

CoffeeScript

for value in arr
  ...

for key, value of map
  ...
