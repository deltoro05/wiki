小数の切り捨てまたは切り上げをするには (floor, ceil)

Python 2014/02/19

import math

# 切り捨て
math.floor(x)

# 切り上げ
math.ceil(x)

切り捨てても切り上げても、値としては整数になるが型はfloatのままである。

Perl 2014/05/10

例

use POSIX qw(floor ceil);

my $a = 12.3;

# 切り捨て
print floor($a) . "\n"; # => 12

# 切り上げ
print ceil($a) . "\n";  # => 13

my $b = -12.3;

# 負の切り捨ては絶対値の大きい方に丸められる
print floor($b) . "\n"; # => -13

# 負の切り上げは0に近い方に丸められる
print ceil($b) . "\n";  # => -12
