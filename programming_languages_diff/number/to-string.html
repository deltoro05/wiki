数値を文字列に変換するには 2014/08/30

多くの言語では動的/静的の違いはあっても数値と文字列とで型が区別されるので変換が必要になることがある。以下はプログラミング言語ごとの変換の仕方のメモ。

逆に文字列から数値に変換する方法はこちら。

Java 2014/08/25

プリミティブ型あれば、対応するラッパークラスのsitaticメソッドの Integer.toString などを使う。すでにラッパークラスのオブジェクトになっていれば、そのインスタンスメソッドの #toString を使う。

Signature:

// java.lang.Byte のメソッド
static String toString(byte i)

// java.lang.Short のメソッド
static String toString(short i)

// java.lang.Integer のメソッド
static String toString(int i)

// java.lang.Long のメソッド
static String toString(long i)

// java.lang.Float のメソッド
static String toString(float i)

// java.lang.Double のメソッド
static String toString(float i)

// java.lang.Byte のメソッド
String toString()

// java.lang.Short のメソッド
String toString()

// java.lang.Integer のメソッド
String toString()

// java.lang.Long のメソッド
String toString()

// java.lang.Float のメソッド
String toString()

// java.lang.Double のメソッド
String toString()

+演算子でオペランドのどちらかがString型であれば、他方の整数または浮動小数点数のプリミティブ型は自動でString型に変換してくれるし、プリミティブ型以外であればもともとtoStringでString型に変換してくれるので、文字列と連結する際には上記メソッドを使うことはあまりない。

PHP 2014/03/29

// 整数または浮動小数点数を文字列に変換
(string)$num

Python 2014/10/02

str関数を使う。reprでもたぶん同じ。

str(num)

Ruby 2014/02/15

to_sメソッドを使う。

num.to_s

16進数にしたい場合は to_s の引数に16を渡す。

例

p 24.to_s
# => "24"
p 27.to_s(16)
# => "1b"

10進数、または2進数、8進数、16進数であれば、書式化文字列で %演算子を使ってもできる。

例

p "%d" % 25
# => "24"
p "%x" % 27
# => "1b"

JavaScript 2014/08/27

String(num)

例

String(24)
// => "24"
