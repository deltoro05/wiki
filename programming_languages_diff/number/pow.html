べき乗演算子 2014/07/02

各プログラミング言語におけるべき乗( \(x^y\) )を計算する方法の比較。

Java / Scala 2014/07/02

Math.powを利用する。引数も返り値も浮動小数点数(double)である。

Math.pow(5, 3);
// => 125.0

Math.pow(2.72, 3.14);
// => 23.149752843459943

PHP 2014/02/15

pow関数を利用する。

var_export(pow(5, 3));
// => 125

pow関数 | PHP Manual
http://jp1.php.net/manual/ja/function.pow.php

Python 2014/09/19

** 演算子を利用する。またはmathモジュールにあるpow関数でも同じ。

Python2での実行例

import math

print 5 ** 3    # => 125
print pow(5, 3) # => 125

print 2.72 ** 3.14    # => 23.1497528435
print pow(2.72, 3.14) # => 23.1497528435

自分の環境のPython3だとなぜか浮動小数点数のほうが、23.14975284345994 と桁数が多く表示されたが、同じ方法でできる。

ちなみに ^ は排他的論理和(XOR)のビット演算子になる。

Ruby 2014/07/07

** 演算子を利用する。

例：

p 5 ** 3       # => 125
p 2.72 ** 3.14 # => 23.14975284345994

Perl 2014/08/02

** 演算子を利用する。

または pow 関数を使う。ただしこれには use POSIX; が必要。

例

print(10.0 ** 1.5);

または

use POSIX;
print(pow(10.0, 1.5)); # => 31.6227766016838

Haskell 2014/05/19

中置記法の関数 ^ を利用する。

R言語 2014/03/29

** と ^ のどちらでもよい。

JavaScript 2014/07/02

Math.pow 関数を利用する。

Math.pow(5, 3); // => 125

Math.pow(2.72, 3.14) // => 23.14975284345994
