除算演算子

C言語, C++ 2013/08/14

演算子 / を使う。

整数同士の除算は、小数点以下を切り捨てられて整数になる。どちらかが浮動小数点数なら、結果も浮動小数点数になる。

整数同士では演算子 % を使って剰余を取得することができる。
