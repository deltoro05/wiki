書き方

Java 2013/08/25

0で始まる8進数表記、0xで始まる16進数表記を使える。

Java7からは、整数は3ケタごとの区切りなどとして途中にアンダースコアを入れることができる。入れる場所はどこでもよく、コンパイラは単に途中のアンダースコアを無視する。

同じくJava7からの機能では、2進数表記もできる。
