シェルに渡すコマンドの引数のエスケープ処理

PHP 2013/09/17

escapeshellarg で外部コマンドに渡す引数のエスケープ処理をする。

escapeshellarg | PHPマニュアル http://www.php.net/manual/ja/function.escapeshellarg.php

日本語文字列の場合、なぜか setlocale(LC_CTYPE, "en_US.UTF-8"); を呼び出さないと日本語部分が削除されてしまう。

例

setlocale(LC_CTYPE, "en_US.UTF-8");
$cmd = '/home/hoge/bin/foo.sh ' . escapeshellarg($arg);
passthru($cmd);

ja_JP.UTF-8 でもいいみたい。
