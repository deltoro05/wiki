マップ(連想配列)の書き方

Java

マップは一発では作れない。

import java.util.HashMap;
map = new HashMap();
map.put("a", 1);
map.put("b", 2);

キーで参照するには

map.get("a");

Scala

map = Map('a' -> 1, 'b' -> 2);

これは以下のシンタックスシュガー。

map = scala.collection.immutable.Map.apply(scala.Predef.any2ArrowAssoc('a').->(1),
                                           scala.Predef.any2ArrowAssoc('a').->(2));

PHP

map = array('a' => 1, 'b' => 2);

PHP 5.4からは以下のようにも書ける。

map = ['a' => 1, 'b' => 2];

Ruby

map = {'a' => 1, 'b' => 2}

Python

map = {'a': 1, 'b': 2}

JavaScript

下の2つは全部同じ連想配列を生成する。

map = {a: 1, b: 2}
map = {'a': 1, 'b': 2}

CoffeeScript

map = {a: 1, b: 2}
map = a: 1, b: 2
map =
    a: 1
    b: 2

clone

JavaScript

shallow copy

function clone(src){
    var dst = {}
    for(var k in src){
        dst[k] = src[k];
    }
    return dst;
}
map_copy = map.clone();
