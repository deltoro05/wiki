論理値への変換

Java

boolean型以外が暗黙のうちに boolean に変換されることはない。つまり、false のみが false、true のみが true になる。

Scala

Boolean型以外が暗黙のうちに Boolean に変換されることはない。つまり、false のみが false、true のみが true になる。

PHP

false, null, 0, 0.0, '', '0', array() が false になり、それ以外は true になる。文字列 '0' は false だが、文字列 'false' は true になる。

http://php.net/manual/ja/types.comparisons.php

Ruby

false, nil が false になり、それ以外は true になる。0 も '' も true。

JavaScript

false, 0, '', null, undefined, NaN が false になり、それ以外は true になる。
