概要 2013/10/05

同じ型のオブジェクトの集合を表すコンテナ型。順序の概念がなく、また同じオブジェクトを複数格納することはできない。型が違うオブジェクトが混ざっていてもよい言語もある。

Java 2013/10/05

java.util.Set インターフェースがある。

Scala 2013/10/05

java.util.Set インターフェースの他に scala.collection.Set トレイトがある。 scala.collection.Set は scala.collection.Iterable, scala.collection.Traversable, scala.collection.TraversableOnce を継承している。

Python 2013/10/07

セット(集合)というデータ型があり、イミュータブル(変更不可)とミュータブル(変更可能)の 2種類がある。集合型ともいう。
