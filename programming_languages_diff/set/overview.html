集合(セット)のデータ型の各プログラミング言語別の概要 2014/10/29

集合を表すコンテナ型。複数のオブジェクトを順序の概念なしに格納するが、等値のオブジェクトを複数格納することはできない。

型が静的な言語では、格納されている要素の型も静的に決定されている必要がある。

Java 2013/10/05

java.util.Set インターフェースがある。

Scala 2014/10/30

java.util.Set インターフェースの他に scala.collection.Set トレイトがある。 scala.collection.Set は scala.collection.Iterable, scala.collection.Traversable, scala.collection.TraversableOnce を継承している。

順序が関係なく同じオブジェクトを複数格納する必要がない場合はリストだとかよりもセットを使うべき。ただそんなときも、自分はscala.collection.Setが共変でないのが邪魔で Seqの類を使ってしまう場合がある。

scala.collection.Set は scala.Predef にも Set として宣言してあるので、 importしなくても Set として型を参照できる。

Setの生成の例

val st: Set[Int] = Set(3, 4, 5);

Python 2014/09/17

セット(集合)というデータ型があり、イミュータブル(変更不可)とミュータブル(変更可能)の 2種類がある。集合型ともいう。
