remove: 集合(セット)から特定の要素を削除するには

Java 2013/11/26

Signature:

// java.util.Set<E>
boolean remove(Object o)

セットに含まれていた場合は true を返す。もともと含まれていなかった場合は false を返す。

Scala 2014/10/28

Signature:

// scala.collection.Set[A]
def - (elem: A): Set[A]

例

// 集合stから要素elemを削除した新しい集合を取得
st - elem

もともと含まれない要素を削除しようとしてもなにも変化せずに同じ集合が得られる。

Python 2013/10/08

st.remove(elem)
