和集合を求めるには (++ / |) 2015/03/20

各プログラミング言語での、和集合を求める方法。

Scala 2015/03/21

Signature:

def Set[A]#++(elems: TraversableOnce[A]): Set[A]

例

val a = Set(1, 2, 3);
val b = Set(3, 4);

println(a ++ b);
// => Set(1, 2, 3, 4)

Ruby / JRuby 2015/03/21

Setクラスのインスタンスに演算子 & を使うと和集合を求めることができる。

require 'set'

a = Set.new [1, 2, 3]

b = Set.new [3, 4]

p a | b
# => #<Set: {1, 2, 3, 4}>

演算子 | は配列にも適用することができる。

a = [1, 2, 2, 3]

b = [3, 4]

p a | b
# => [1, 2, 3, 4]
