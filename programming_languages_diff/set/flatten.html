セットのセットをセットに平坦化するには (flatten)

Scala 2013/10/05

Signature:

// Set[A]
def flatten[B]: Set[B]

処理結果のセットを返し、自身は変更しない。A は Seq[B], Set[B] などである必要がある。

Seq#flatten と使い方が同じ。
