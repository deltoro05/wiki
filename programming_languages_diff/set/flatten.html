flatten: 集合(セット)の集合(セット)を平坦化するには

Scala 2014/10/29

Signature:

// scala.collection.Set[A]
def flatten[B]: Set[B]

処理結果のセットを返し、自身は変更しない。A は Seq[B], Set[B] などである必要がある。

Seq#flatten と使い方が同じ。

例

val st = Set(List(3, 4, 5), List(4, 6, 8));
println(st.flatten);
// => Set(5, 6, 3, 8, 4)

// 以下と同じになる
println(st.flatMap(e => e));
// => Set(5, 6, 3, 8, 4)

Ruby 2014/10/29

配列に対してはflattenというメソッドがこの動作になる。
