Scala 2013/10/05

Signature:

// scala.collection.Set[E]
def contains(elem: E): Boolean

Python 2013/10/06

elem in set

elem が set に含まれれば true を返す。

この否定は

elem not in set

in 演算子はリストに対しても使える。
