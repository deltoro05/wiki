セットに特定の要素が含まれているかどうかを判定するには (contains)

参考: リストに特定の要素が含まれているかどうかを判定するには

Java 2013/08/06

Signature:

// java.util.Set<E>
boolean contains(Object elem)

Scala 2013/10/05

Signature:

// scala.collection.Set[E]
def contains(elem: E): Boolean

Python 2013/10/06

elem in set

elem が set に含まれれば true を返す。

この否定は

elem not in set

in 演算子はリストに対しても使える。
