集合(セット)に特定の値が含まれているかどうかを判定するには (contains / in / include? / member?) 2015/03/21

各プログラミング言語での、集合に特定の値が含まれているかどうかを判定する方法。

関連

  • 集合に別の集合の要素がすべて含まれているかどうかを判定するには
  • 配列に特定の値が含まれているかどうかを判定するには

Java 2013/08/06

Signature:

boolean java.util.Set<E>#contains(Object elem)

Scala 2015/03/21

Signature:

def scala.collection.Set[E]#contains(elem: E): Boolean

例

val st = Set("abc", "def", "ghi");

println(st.contains("abc")) // => true
println(st.contains("ABC")) // => false

Python 2013/10/06

elem in set

elem が set に含まれれば true を返す。

この否定は

elem not in set

in 演算子はリストに対しても使える。

Ruby / JRuby 2015/03/18

Set#include? または Set#member? メソッドを使う。

require 'set'

st = Set.new ["abc", "def", "ghi"]

p st.include? "abc" # => true
p st.include? "ABC" # => false

この2つのメソッドは配列に対しても使える。
