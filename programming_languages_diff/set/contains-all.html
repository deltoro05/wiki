セットに別のセットの要素がすべて含まれているかどうかを判定するには (containsAll)

Java 2013/11/06

Signature:

// java.util.Set<E>
boolean containsAll(Collection<?> elems);

例

// set が elems のすべてを含んでいるかどうか
set.containsAll(elems);

Python 2013/10/05

# set が elems のすべてを含んでいるかどうか
# elems のすべてが set に含まれているかどうか
# elems も set も集合型
elems.issubset(set)
# または
elems <= set
set.issuperset(elems)
# または
set >= elems

issubset, issuperset はセット(集合型)以外にリストなども受け入れる。
