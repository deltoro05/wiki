積集合を求めるには (&)

Scala 2015/03/21

Signature:

  • def Set[E]#&(that: Set[E]): Set[E]

例

val a = Set(1, 2, 3);
val b = Set(3, 4);

println(a & b);
// => Set(3)

Ruby / JRuby 2015/03/21

Setクラスのインスタンスに演算子 & を使うと積集合を求めることができる。

require 'set'

a = Set.new [1, 2, 3]

b = Set.new [3, 4]

p a & b
# => #<Set: {3}>

演算子 & は配列にも適用することができる。

a = [1, 2, 2, 3]

b = [3, 4]

p a & b
# => [3]
