論理値への変換 2013/08/24

if文の条件式など、論理値を必要とされる場所に任意の型を入れたときにどう評価されるか。

まとめ 2013/09/21

  C言語 PHP Ruby Perl JavaScript
0 false false true false false
0.0 false false true false false
空文字列 true false true false false
空の配列 true false true false true

C言語 2013/09/21

C言語には true や false という値はない。if文などの条件式には、 NULL, 0, 0.0, ‘\x00’ が入ると false の扱いになり、それ以外は true` の扱いになる。

空文字列は、NULLでないので、true になる。

Java / Scala 2013/09/21

boolean型(ScalaはBoolean型)以外が暗黙のうちに論理値に変換されることはない。つまり、false のみが false、true のみが true になる。 if文などの条件式にはboolean以外を置くことはできない。

PHP 2013/11/12

FALSE, null, 0, 0.0, '', '0', array() が FALSE になり、それ以外は TRUE になる。文字列 '0' は FALSE だが、文字列 'FALSE' や '0.0' は TRUE になる。

http://www.php.net/manual/ja/types.comparisons.php

Python 2013/10/07

False, None, [], (), {}, ””, 0, 0L, 0.0, 0j が False になる。その他に nonzero() や len() メソッドがあるオブジェクトで False や 0 を返すものも False になる。それ以外は True` になる。

Ruby 2013/10/20

false, nil が false になり、それ以外はすべて true になる。0 も '' も [] も true。

Perl 2013/09/21

Perlには true や false という値はない。if文の条件式などには、 0, '0', '', undef, () が入ると false の扱いになり、それ以外は true の扱いになる。

JavaScript 2013/09/21

false, 0, '', null, undefined, NaN が false になり、それ以外は true になる。[] も true になる。
