論理値への変換 2013/08/24

if文の条件式など、論理値を必要とされる場所に任意の型を入れたときにどう評価されるか。

まとめ 2013/09/21

  C言語 PHP Ruby Perl JavaScript
0 false false true false false
0.0 false false true false false
空文字列 true false true false false
空の配列 true false true false true

C言語 / C++ 2015/02/14

NULL, 0, 0.0, '\x00' が入ると false の扱いになり、それ以外は true の扱いになる。

空文字列は、NULLでないので、true になる。

C++ではキーワードtrue/falseもある。

Java / Scala 2015/02/14

if文やwhile文などの条件式にはboolean型(ScalaはBoolean型)とjava.lang.Boolean型以外を置くことはできない。置こうとしてもコンパイルエラーになる。
