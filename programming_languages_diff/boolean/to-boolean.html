論理値への変換

Java 2013/04/10

boolean型以外が暗黙のうちに boolean に変換されることはない。つまり、false のみが false、true のみが true になる。

Scala 2013/04/10

Boolean型以外が暗黙のうちに Boolean に変換されることはない。つまり、false のみが false、true のみが true になる。

PHP 2013/05/14

false, null, 0, 0.0, '', '0', array() が false になり、それ以外は true になる。文字列 '0' は false だが、文字列 'false' や '0.0' は true になる。

http://php.net/manual/ja/types.comparisons.php

Ruby 2013/04/10

false, nil が false になり、それ以外は true になる。0 も '' も true。

Perl 2013/07/30

Perlには false や true という値はない。if文などの条件式には、 0, '0', '', undef が入ると false の扱いになり、それ以外は true の扱いになる。

JavaScript 2013/04/10

false, 0, '', null, undefined, NaN が false になり、それ以外は true になる。
