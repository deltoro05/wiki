スレッドを一定時間停止する

Java 2013/09/01

3秒間停止する例

try {
    Thread.sleep(3000);
} catch (InterruptedException e) {
    ...;
}

Ruby 2013/08/21

3秒間停止する例

sleep 3

秒数に浮動小数点数も使える。

Perl 2013/08/14

3秒間停止する例

sleep(3);

引数を省略すると永久にスリープし、SIGALRMを受け取ると再開するようになる。
