ファイル操作

  • ディレクトリの一覧を取得

ディレクトリの一覧を取得

Ruby

Dir::entries('/home/foo')

ファイルが存在するかどうかを取得

Ruby

File.exists?('/home/foo/testfile')

ファイルかどうかを調べる

ディレクトリでなくてファイルかどうかを調べるには

Ruby

File.ftype('/home/foo/testfile') == 'file'

ディレクトリかどうかを調べる

Ruby

File.ftype('/home/foo/testfile') == 'directory'

シンボリックリンクかどうかを調べる

Ruby

File.ftype('/home/foo/testfile') == 'link'

シンボリックリンクのリンク先を取得する

Ruby

path がシンボリックリンクの場合に、path をリンク先に書き換える。

if File.exists?(path) && File.ftype(path) == 'link' then
  link = File.readlink(path)
  path = File.expand_path(link, File.dirname(path))
end
