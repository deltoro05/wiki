ファイル操作

  • ディレクトリの一覧を取得

ファイルの中身を1つの文字列として取得するには

Ruby

str = File.open('./hoge.txt').read

ファイルの中身を1行ずつテキスト処理するには

Ruby

IO.foreach('./hoge.txt') do |line|
  ...
end

ファイルを削除するには

Ruby

File.unlink('./hoge.txt')

ディレクトリの一覧を取得

Ruby

Dir::entries('/home/foo')

ファイルが存在するかどうかを取得

Ruby

File.exists?('/home/foo/testfile')

ファイルかどうかを調べる

ディレクトリでなくてファイルかどうかを調べるには

Ruby

File.ftype('/home/foo/testfile') == 'file'

ディレクトリかどうかを調べる

Ruby

File.ftype('/home/foo/testfile') == 'directory'

シンボリックリンクかどうかを調べる

Ruby

File.ftype('/home/foo/testfile') == 'link'

シンボリックリンクのリンク先を取得する

Ruby

path がシンボリックリンクの場合に、path をリンク先に書き換える。

if File.exists?(path) && File.ftype(path) == 'link' then
  link = File.readlink(path)
  path = File.expand_path(link, File.dirname(path))
end

ディレクトリの中をを再帰的にたどってすべて削除するには

PHP 2011/10/13

function rmdir_all($dir) {
  if (!file_exists($dir)) {
    return;
  }
  $dhandle = opendir($dir);
  if ($dhandle) {
    while (false !== ($fname = readdir($dhandle))) {
       if (is_dir( "{$dir}/{$fname}" )) {
          if (($fname != '.') && ($fname != '..')) {
             rmdir_all("$dir/$fname");
          }
       } else {
          unlink("$dir/$fname");
          echo "delete $dir/$fname<br />";
       }
    }
    closedir($dhandle);
  }
  rmdir($dir);
  echo "delete $dir<br />";
}
