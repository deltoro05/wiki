ファイル操作

ファイルに追記するには

PHP 2013/06/11

$handle = fopen($file_path, 'a');
fwrite($handle, $data);
fclose($handle);

Ruby 2013/07/16

File.open(file_path, 'a') do |handle|
  handle.puts($data)
end

ファイルを削除するには

Ruby 2013/05/04

File.unlink('./hoge.txt')

Perl 2013/08/13

unlink('./hoge.txt');

複数まとめて削除することもできる。

unlink('./hoge.txt', './foo.txt');
unlink(@files);

ディレクトリの中のファイル一覧を取得

Ruby 2013/04/10

Dir::entries('/home/foo')

Perl 2013/08/14

例

my @files = glob('/home/foo/*');
@files = map { if ( /\/([^\/]+)\Z/ ){ $1; } else { $_; } } @files;

1行目だけだと、フルパスでファイル一覧が取得される。

ファイルサイズを取得

Ruby 2013/05/28

File::stat('./hoge.txt').size

ファイルまたはディレクトリが存在するかどうかを取得

PHP 2013/07/24

file_exists('/home/foo/testfile')

シンボリックリンクの場合は、シンボリックリンクの指す先が存在するかどうかをチェックする。

http://www.php.net/manual/ja/function.file-exists.php

Ruby 2013/04/29

File.exists?('/home/foo/testfile')

Perl 2013/07/27

if( -e '/home/foo/testfile' ){
    ...;
}

sh 2013/07/27

if [ -e /home/foo/testfile ]; then
    ...
fi

ファイルかどうかを調べる

ディレクトリでなくてファイルかどうかを調べるには

Ruby 2013/04/30

File.ftype('/home/foo/testfile') == 'file'

Perl 2013/07/27

if( -f '/home/foo/testfile' ){
    ...;
}

sh 2013/07/27

if [ -f /home/foo/testfile ]; then
    ...
fi

ディレクトリかどうかを調べる

Ruby 2013/04/30

File.ftype('/home/foo/testfile') == 'directory'

Perl 2013/07/27

if( -d '/home/foo/testfile' ){
    ...;
}

sh 2013/07/27

if [ -d /home/foo/testfile ]; then

シンボリックリンクかどうかを調べる

Ruby 2013/04/30

File.ftype('/home/foo/testfile') == 'link'

Perl 2013/07/27

if( -l '/home/foo/testfile' ){
    ...;
}

sh 2013/07/27

if [ -L /home/foo/testfile ]; then
    ...
fi

シンボリックリンクのリンク先を取得する

Ruby 2013/04/30

path がシンボリックリンクの場合に、path をリンク先に書き換える。

if File.exists?(path) && File.ftype(path) == 'link' then
  link = File.readlink(path)
  path = File.expand_path(link, File.dirname(path))
end

ディレクトリの中をを再帰的にたどってすべて削除するには

PHP 2011/10/13

function rmdir_all($dir) {
  if (!file_exists($dir)) {
    return;
  }
  $dhandle = opendir($dir);
  if ($dhandle) {
    while (false !== ($fname = readdir($dhandle))) {
       if (is_dir( "{$dir}/{$fname}" )) {
          if (($fname != '.') && ($fname != '..')) {
             rmdir_all("$dir/$fname");
          }
       } else {
          unlink("$dir/$fname");
          echo "delete $dir/$fname<br />";
       }
    }
    closedir($dhandle);
  }
  rmdir($dir);
  echo "delete $dir<br />";
}
