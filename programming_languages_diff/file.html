ファイルの中身を1つの文字列として取得するには

PHP 2013/05/28

str = file_get_contents('./hoge.txt');

http://php.net/manual/ja/function.file-get-contents.php

Ruby 2013/05/04

str = File.open('./hoge.txt').read

ファイルの中身を1行ずつテキスト処理するには

PHP 2013/07/18

$fh = fopen('./hoge.txt', 'r');
if($fh){
  while (!feof($fh)) {
    $line = fgets($fh);
    // $line には最後の改行を含む
    ...;
  }
  fclose($fh);
}

Ruby 2013/07/17

IO.foreach('./hoge.txt') do |line|
  ...
end

または

open(script_name) do |fh|
  while line = fh.gets
    // line には最後の改行を含む
    ...
  end
end

ファイルに追記するには

PHP 2013/06/11

$handle = fopen($file_path, 'a');
fwrite($handle, $data);
fclose($handle);

Ruby 2013/07/16

File.open(file_path, 'a') do |handle|
  handle.puts($data)
end

ファイルを削除するには

Ruby 2013/05/04

File.unlink('./hoge.txt')

ディレクトリの中のファイル一覧を取得

Ruby 2013/04/10

Dir::entries('/home/foo')

ファイルサイズを取得

Ruby 2013/05/28

File::stat('./hoge.txt').size

ディレクトリを作成

Ruby 2013/06/24

Dir::mkdir(dirname)

または、mkdir -p に相当する方法として

require 'fileutls'
FileUtils::mkdir_p(dirname)

シンボリックリンクのリンク先を取得する

Ruby 2013/04/30

path がシンボリックリンクの場合に、path をリンク先に書き換える。

if File.exists?(path) && File.ftype(path) == 'link' then
  link = File.readlink(path)
  path = File.expand_path(link, File.dirname(path))
end

ディレクトリの中をを再帰的にたどってすべて削除するには

PHP 2011/10/13

function rmdir_all($dir) {
  if (!file_exists($dir)) {
    return;
  }
  $dhandle = opendir($dir);
  if ($dhandle) {
    while (false !== ($fname = readdir($dhandle))) {
       if (is_dir( "{$dir}/{$fname}" )) {
          if (($fname != '.') && ($fname != '..')) {
             rmdir_all("$dir/$fname");
          }
       } else {
          unlink("$dir/$fname");
          echo "delete $dir/$fname<br />";
       }
    }
    closedir($dhandle);
  }
  rmdir($dir);
  echo "delete $dir<br />";
}
