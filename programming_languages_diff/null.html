null

C言語 2013/10/14

NULL というのがある。0と同じで、ポインタに0を代入するのと NULL を代入するのはたぶん同じ。

Java 2013/12/15

null というのがある。任意の参照型になりえる。intなどのプリミティブ型にnullを使うことはできない。

PHP 2013/11/12

NULL というのがある。TRUE / FALSEと同じく大文字小文字が区別されない。

Python 2013/10/14

None というのがある。関数にreturn文がないときに、Noneが返されるものとみなされる。

Ruby 2013/12/18

nil というのがある。Rubyではnilもオブジェクトであるので、メソッドを呼ぶ出すことができる。

print nil.nil?
# => true

Perl 2013/09/21

変数の値が未定義の状態というのがありえる。変数を宣言だけして、一度も代入しないと undef になる。

定義済みの変数を未定義の状態に戻すには、以下のようにする。

undef $a;

または以下のようにもできる。

$a = undef;

defined を使うと、定義済みかどうかを判定することができる。

if (defined $a) {
    # 定義済みの場合
} else {
    # 未定義の場合
}
