文字列の書き方

Java 2013/05/20

" で囲んで文字列を表す。 ' で囲むと文字列ではなく文字になる。

PHP 2013/06/16

" で囲んだ場合とヒアドキュメントでは $ で始まる部分が変数名として値に展開される。

ヒアドキュメント

PHP 2013/06/16

普通のヒアドキュメントとNowdocというものがある。ヒアドキュメントでは、" で囲んだ文字列リテラルと同じように、$ で始まる部分が変数名としてその値に展開がされるのに対して、Nowdocは、' で囲んだ文字列リテラルと同じように、そういった展開がされない。

上記の例で1行目にバックスラッシュがあるのは最初の改行を文字列に含めないようにするため。

Unicodeエスケープは、文字列リテラルの機能ではなく、Javaソースコードをコンパイラがトークンに分割する前の処理として機能する。

PHP 2013/06/16

' で囲めば、エスケープが必要になるのは ', \ のみになる。'\n' などと書いても \n そのものになって改行にはならない。 ' で囲まれている場合、 \ の次に ', \ 以外が来ても特別な意味にはならない。

echo 'aaa\nttt'; // => aaa\nttt // \n は改行にはならない
echo 'aaa\'ttt'; // => aaa'ttt
echo 'aaa\ttt'; // => aaa\ttt // \t はタブにはならずに \t のまま
echo 'aaa\\ttt'; // => aaa\ttt // \\ は \ になる

" で囲んだ場合は、以下のエスケープが使える。

\\数字(1ケタ〜3ケタ)
8進数で文字を表現

16進数で文字を表現
\t
0x09 タブ
\n
0x0a LF
\f
0x0c
\r
0x0d CR
\e
0x1B ESC
\$
$
\"
"
\\
\

sh 2013/05/20

いくつかの記号はシェルで特別な意味を持つので \ でエスケープが必要。

" で囲めば、エスケープが必要になるのは ", $, \, ` のみになる。

' で囲めば、エスケープ処理が不要になるが、 ' を含めることができなくなる。

substr

Java / Scala 2013/04/10

String String#substring(int from)
String String#substring(int from, int to)

PHP 2013/07/23

string substr(string $string, int $start [, int $length])

http://www.php.net/manual/ja/function.substr.php

start が負の場合、返される文字列は、 stringの後ろから数えて start番目から始まる文字列となります。
string の長さが start 文字以下の場合は FALSE が返されます。
length が指定され、かつ負である場合、 string の終端から多くの文字が省略されます (start が負の場合は、開始位置を算出したあとで)。 もし start が切り出し位置を超える場合、 false が返されます。
length が指定され、かつ 0 である場合は、 空文字列なら NULL、そうでなければ FALSE を返します。
length を省略した場合は、 start の位置から文字列の最後までの部分文字列を返します。

$lengthが$stringに比べ長すぎる場合は、エラーにならずに$stringの最後までを返す。

Ruby 2013/04/12

String#slice(index)
String#slice(from, length)

lengthを省略したら、from以降の文字列ではなく、その場所の1文字を返す。

配列に対しても同じように slice が使える。

JavaScript 2013/04/10

string.substring(from [, to])
string.substr(from [, len])

trim

Java / Scala 2013/04/10

String String#trim()

PHP 2013/06/11

string trim(string $str [, string $charlist])

http://php.net/manual/ja/function.trim.php

2番目のパラメータを指定しない場合、 trim()は以下の文字を削除します。
“ “ (ASCII 32 (0x20)), 通常の空白。
“\t” (ASCII 9 (0x09)), タブ。
“\n” (ASCII 10 (0x0A)), リターン。
“\r” (ASCII 13 (0x0D)), 改行。
“\0” (ASCII 0 (0x00)), NULバイト
“\x0B” (ASCII 11 (0x0B)), 垂直タブ 

# 改行コードのみを削除
trim($str, "\r\n")

Ruby 2013/04/10

String#strip
String#strip!

http://doc.ruby-lang.org/ja/2.0.0/method/String/i/strip.html

空白文字の定義は " \t\r\n\f\v" です。 また、文字列右側からは "\0" も取り除きますが、 左側の "\0" は取り除きません。

最後の改行のみを削除するのは chomp。

String#chomp(str)
String#chomp!(str)

http://doc.ruby-lang.org/ja/2.0.0/method/String/i/chomp.html

startsWith / endsWith

Java / Scala 2013/04/10

String String#startsWith(String prefix)
String String#endsWith(String suffix)

PHP 2013/07/23

startsWithに相当するよく使う方法。

strpos($str, $prefix) === 0

ただ、これだと、$strが長い文字列の時にいちいち$strの最後までなめるのが効率よくなさそう。

endsWithに相当するよく使う方法。

strrpos($str, $suffix) === strlen($str) - strlen($suffix)

これも$strが長い時に効率が悪そう。

Ruby 2013/04/10

String#start_with?(prefix)
String#end_with?(suffix)

JavaScript 2013/04/10

startsWithに相当するよく使う方法。

str.indexOf(prefix) == 0

ただ、これだと、strが長い文字列の時にいちいちstrの最後までなめるのが効率よくなさそう。

endsWithに相当するよく使う方法。

str.substr(str.length - prefix.length) == prefix

ただ、これだと、prefixがstrより長い場合にどうなることやら。

Ruby 2013/04/10

String#index(str)
String#index(str, index)

String#rindex(str)
String#rindex(str, index)

見つからない場合は nil を返す。2つ目の引数を与えると index はその位置以降を探し、rindex はその位置以前を探す。

'abcabc'.index('abc')     # => 0
'abcabc'.index('abc', 1)  # => 3
'abcabc'.index('abc', 3)  # => 3
'abcabc'.index('abc', 4)  # => nil
'abcabc'.rindex('abc')    # => 3
'abcabc'.rindex('abc', 3) # => 3
'abcabc'.rindex('abc', 2) # => 0
