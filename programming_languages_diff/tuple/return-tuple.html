複数の返り値(多値返却) 2013/10/28

関数は普通は返り値を1つだけ返すか、1つも返さない(C言語のvoidなど)ものが多い。ただ、言語によっては複数の値を関数から返す方法がある。

Scala 2013/09/20

タプルを返せばよい。

PHP 2013/04/22

配列を使って無理やりやろうと思えばできなくもない。

function f(){
  return array(1, "abc");
}
list($num, $str) = f();

Ruby 2013/09/06

return にカンマ区切りで複数の値を書けば、配列を返すことになり、受け取り側は配列の多重代入を使う。

def f
  return 1, "abc"
end
num, str = f

Perl 2013/09/21

リストを返して、受け取り側は多重代入を使う。

sub f {
    return (1, "abc");
}
my ($num, $str) = f;
