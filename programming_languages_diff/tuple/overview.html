概要 2013/10/05

型が違っていてもよい2つ以上のデータを格納できるコンテナ型。格納するデータの数は固定。各フィールドに名前が付いていないC言語の構造体みたいなもの。

静的な型付けの言語では要素ごとの型も静的に決まる。

Scala 2013/10/28

コンマ区切りの2つ以上の値を()で囲めば、タプルになる。

関数にタプルを1つだけ渡して呼び出すには、 ()はもともと関数呼び出しの引数を囲む記号なので、二重にカッコで囲まないといけない。

関数の返り値としてタプルを使えば、 複数の結果を返すことができる。

タプルは実際には普通のオブジェクトで、 2要素であれば scala.Tuple2、3要素であれば scala.Tuple3 というクラスのインスタンスである。

Python 2013/10/05

Pythonのタプルは、リストがイミュータブルになっただけのようである。
