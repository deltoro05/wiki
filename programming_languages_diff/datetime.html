日付と時刻

今日の日付を取得するには／現在時刻を取得するには 2013/06/19

PHP 2013/06/11

# 現在日時を YYYY/MM/DD hh:mm:ss の書式の文字列で取得する
date('Y/m/d H:i:s');

Ruby 2013/06/19

require 'date'
# DateTime を使う場合でも require は date だけでよい

# Dateオブジェクトを取得する
Date.today

# DateTimeオブジェクトを取得する
DateTime.now

日付オブジェクト・日時オブジェクトの各フィールドを取得するには

Ruby 2013/06/19

date.year
date.month
date.day    # その月の中での日付(1-31)
date.wday   # 曜日(0-6)。日曜日が0

datetime.year
datetime.month
datetime.day    # その月の中での日付(1-31)
datetime.wday   # 曜日(0-6)。日曜日が0
datetime.hour
datetime.min
datetime.sec

日付オブジェクト・日時オブジェクトを比較するには

Ruby 2013/08/08

require 'date'

a = DateTime.now
sleep 3
b = DateTime.now
p (a < b) # => true
p (b < a) # => false

日付単位の計算

Ruby 2013/05/06

date + 3       # 3日後の日付
date1 - date2  # 日付の差

文字列を日付に変換

PHP 2013/06/16

var_export(strptime('2013/06/16', '%Y/%m/%d'));
// => array (
//   'tm_sec' => 0,
//   'tm_min' => 0,
//   'tm_hour' => 0,
//   'tm_mday' => 16,
//   'tm_mon' => 5,
//   'tm_year' => 113,
//   'tm_wday' => 0,
//   'tm_yday' => 166,
//   'unparsed' => '',
// )

http://www.php.net/manual/ja/function.strptime.php

p [date.year, date.month, date.day, date.wday]
