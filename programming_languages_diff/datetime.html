日付と時刻

今日の日付を取得するには

Ruby

rquire 'date'
Date.today

日付オブジェクトの各フィールドを取得するには

Ruby

date.year
date.month
date.day    # その月の中での日付(1-31)
date.wday   # 曜日(0-6)。日曜日が0

日付単位の計算

Ruby

date + 3       # 3日後の日付
date1 - date2  # 日付の差

文字列を日付に変換

Ruby

require 'date'
date = Date.strptime('2013/03/17', '%Y/%m/%d')
p [date.year, date.month, date.day, date.wday]

date.wday で曜日を整数で取得でき、日曜日が0。
