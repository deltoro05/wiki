クラス階層

C++ 2016/08/12

すべてのクラスに共通するスーパークラスは存在しない。

Java 2013/09/21

すべてのクラスのスーパークラスとして Object がある。

Scala 2013/09/21

すべてのクラスのスーパークラスとして Any がある。

PHP 2016/08/12

すべてのクラスに共通するスーパークラスは存在しない。

get_parent_class でスーパークラスを調べると false になる。

class A {
}

class B extends A {
}

var_export(get_parent_class(new B())); echo "\n"; // => 'A'
var_export(get_parent_class(new A())); echo "\n"; // => false

Ruby 2014/02/22

すべてのクラスのスーパークラスとして BasicObject がある。
