インスタンス生成

C++ 2013/09/09

new 演算子を使う。

HogeClass hoge = new HogeClass();

引数はコンストラクタに渡されるので、コンストラクタの引数型に合わせて、 new演算子に引数を渡す必要がある。

Java 2013/09/09

new 演算子を使う。

HogeClass hoge = new HogeClass();

引数はコンストラクタに渡されるので、コンストラクタの引数型に合わせて、 new演算子に引数を渡す必要がある。

Scala 2013/10/21

Javaと同様に new 演算子を使う。

case classに関してはクラスオブジェクトにインスタンスを生成するapplyメソッドが自動で定義されていているので、newを省略できる。 apply というメソッドがあるオブジェクトは、それ自体が関数であるかのように後ろにカッコで引数を置くことで、applyメソッドを呼び出すことができるScalaの仕組みである。

val foo = new FooClass(...);
val bar = BarCaseClass(...);

Ruby 2014/11/29

new というクラスメソッドを使う。

hoge = HogeClass.new

initializeメソッドに引数があれば、newメソッドに引数を渡す。

hoge = HogeClass.new(arg1, arg2)

JRubyでJavaのクラスのインスタンスを生成する場合にもnewメソッドを使う。 Javaのクラスに new というクラスメソッドがあるかのように振る舞う。
