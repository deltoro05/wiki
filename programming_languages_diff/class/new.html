インスタンス生成

C++ 2013/09/09

new 演算子を使う。

HogeClass hoge = new HogeClass();

引数はコンストラクタに渡されるので、コンストラクタの引数型に合わせて、 new演算子に引数を渡す必要がある。

Java 2013/09/09

new 演算子を使う。

HogeClass hoge = new HogeClass();

引数はコンストラクタに渡されるので、コンストラクタの引数型に合わせて、 new演算子に引数を渡す必要がある。

Ruby 2013/08/27

new というクラスメソッドを使う。

hoge = HogeClass.new

initializeメソッドに引数があれば、newメソッドに引数を渡す。

hoge = HogeClass.new(arg1, arg2)
