インスタンス生成

C++ 2013/09/09

new 演算子を使う。

HogeClass hoge = new HogeClass();

引数はコンストラクタに渡されるので、コンストラクタの引数型に合わせて、 new演算子に引数を渡す必要がある。

Java 2013/09/09

new 演算子を使う。

HogeClass hoge = new HogeClass();

引数はコンストラクタに渡されるので、コンストラクタの引数型に合わせて、 new演算子に引数を渡す必要がある。

Scala 2013/09/23

Javaと同様に new 演算子を使う。

case classに関してはクラスオブジェクトにインスタンスを生成するapplyメソッドが自動で定義されていているので、newを省略できる。

val foo = new FooClass(...);
val bar = BarCaseClass(...);

Ruby 2013/08/27

new というクラスメソッドを使う。

hoge = HogeClass.new

initializeメソッドに引数があれば、newメソッドに引数を渡す。

hoge = HogeClass.new(arg1, arg2)
