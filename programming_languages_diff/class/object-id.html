オブジェクトのIDを取得するには

Java 2013/12/11

Signature:

// java.lang.System
static int identityHashCode(Object x)

すべてのオブジェクトにある hashCode メソッドは、別々のオブジェクトでもそのオブジェクトが表す値が等しければ同じ値を返すが、この identityHashCode は別々のオブジェクトであれば異なる値を返すのが原則。

hashCode のデフォルト実装は identityHashCode と同じになる。

Ruby 2013/12/11

すべてのオブジェクトにある object_id メソッドを使うと、オブジェクトのIDが取得できる。
