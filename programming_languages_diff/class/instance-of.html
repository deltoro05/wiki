Perl 2013/11/23

レファレンスの種類は ref という関数(?)で判定できる。レファレンスでない場合は '' を返し、レファレンスの場合はその指す先のオブジェクトの種類を文字列で返す。

my $num = 1;
my $str = "abc";
my @arr = (1, 2, 3);
my %hash = (a => 1, b => 2);
sub func {
    ...;
}

print (ref $num eq ''); # => 1
print (ref $str eq ''); # => 1
print (ref @arr eq ''); # => 1
print (ref %hash eq ''); # => 1
print (ref &func eq ''); # => 1

my $ref_arr = [1, 2, 3];
my $ref_hash = {a => 1, b => 2};
my $ref_func = sub { print "Hello\n"; };

print (ref $ref_arr); # => ARRAY
print (ref $ref_hash); # => HASH
print (ref $ref_func); # => CODE
