クラスの定義

C言語 2013/09/10

C言語にはクラスはないが、データ構造を作れる構造体というものがある。

構造体の例

struct FooStruct {
    /* フィールド宣言 */
    int field1;
    int field2;
}

C++ 2013/08/28

例

class FooClass : public SuperFooClass {

private:
    // フィールド宣言
    int field1;
    int field2;

public:
    // コンストラクタ宣言
    FooClass();

    // デストラクタ宣言
    ~FooClass();

    // メンバ関数(メソッド)宣言
    void method1();

}

// メソッド定義
void FooClass::method1(){
    ...;
}

アクセス修飾子(public や private)は、メンバごとに付けるのではなく、上記のようにグループ分けして書く。

またクラス宣言の中にはメンバ関数(メソッド)はシグニチャの宣言のみを書いて、メンバ関数の定義はクラス宣言の外側に書く。普通はクラス宣言はヘッダファイルに書き、メンバ関数定義はソースファイルに書く。

Ruby 2013/08/27

例

class FooClass < SuperFooClass

  # インスタンス初期化メソッド
  def initialize
    @field1 = ... # インスタンス変数
  end

  # メソッド定義
  def method1
    ...
  end

  # フィールドへの読み取り専用アクセッサ
  attr_reader :field1, :field2

  # フィールドへの読み書きアクセッサ
  attr_accessor :field3, :field4

end
