メソッド定義

Java 2013/07/10

public String f(String arg1, int arg2){
  ...;
  return ...;
}

返り値がない場合

public void f(String arg1, int arg2){
  ...;
}
  • メソッド呼び出しは、引数のカッコを省略不可

Scala 2013/09/12

def f(arg1: String, arg2: Int): String = {
  ...;
}

返り値が Unit の場合(返り値がない場合)は、以下でも可。

def f(arg1: String, arg2: Int){
  ...;
}
  • メソッド呼び出しは、引数がないか、引数が1つの場合は、カッコを省略できる

PHP 2013/06/09

function f($arg1, $arg2){
  ...;
  return ...;
}
  • 関数定義と呼び出しでは引数のカッコを省略不可

Ruby 2013/04/10

def f(arg1, arg2)
  ...
  return ...
end
  • メソッド呼び出しは引数のカッコを省略可

Perl 2013/08/12

サブルーチンの定義。

sub f {
  my ($arg1, $arg2) = @_;
  ...;
  return ...;
}

サブルーチンの呼び出し。

f($a1, $a2);
  • サブルーチン定義の中では引数は、@_, $_[...] などで参照する
  • サブルーチン呼び出しは、引数がない場合に限り、カッコを省略できる

sh 2013/08/24

関数の定義。

f()
{
    ...
}

関数の呼び出し。

f a1 a2
  • 関数定義の中では引数は $@, $1, $2,,, などを使う
  • 関数呼び出しは通常の外部コマンドの実行と同じくカッコを付けずに引数を並べる

return 0 のように書くと関数の処理から戻り値を指定して脱出できる。

JavaScript 2013/06/09

function f(arg1, arg2){
    ...;
    return ...;
}

または

var f = function(arg1, arg2){
    ...;
    return ...;
}
  • 関数定義と呼び出しでは引数のカッコを省略不可

CoffeeScript 2013/04/28

(arg1, arg2) ->
  ...
  ...

引数がない場合

->
  ...
  ...
  • 関数呼び出しは、引数がある場合はカッコを省略可。引数がない場合はカッコを省略できない。
