メソッドのオーバーロード 2013/09/06

同じ名前で引数や返り値の型が異なるメソッドを複数定義することをオーバーロードという。

C++ 2013/09/20

引数の型や数が異なれば関数のオーバーロードが可能。

返り値の方だけが異なる関数のオーバーロードはできない。

Java 2013/09/20

引数の型や数が異なれば関数のオーバーロードが可能。

返り値の型だけが異なる関数のオーバーロードはできない。

Java5以降でジェネリックス(総称型)が導入されたが、引数の型がジェネリックス(総称型)の場合で、その型パラメータのみが異なるメソッドは定義できない。オーバーロードがJVMのレベルでのサポートだが、ジェネリックスの機能はJVMではサポートされておらず、コンパイル時に型パラメータが削除されてしまうため。
