関数が外側のスコープの変数にアクセスできるかどうか

Ruby 2013/06/09

def builder
  i = 0
  Proc.new do
    puts i
    i = i + 1
  end
end

f1 = builder
f2 = builder
f1.call # 0
f1.call # 1
f2.call # 0
f1.call # 2

Procオブジェクトは、その外側の変数を読み書きできる。

JavaScript 2013/06/09

var builder = function(){
    var i = 0;
    return function(){
        console.log(i);
        i = i + 1;
    }
}
var f1 = builder();
var f2 = builder();
f1(); // 0
f1(); // 1
f2(); // 0
f1(); // 2
