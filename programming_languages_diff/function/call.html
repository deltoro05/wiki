関数呼び出しまたはメソッド呼び出し

Java 2013/10/07

obj.method1(arg1, arg2);

引数のカッコは省略不可。

クラスの中で同じインスタンスのメソッドを呼び出す場合はレシーバーを省略できる。

method1(arg1, arg2);

Scala 2013/11/15

// 関数
func1(arg1, arg2);

// メソッド
obj.method1(arg1, arg2);

メソッド呼び出しの場合、引数が1つの場合はカッコを省略可。

クラスの中で同じインスタンスのメソッドを呼び出す場合はレシーバーを省略できる。

method1(arg1, arg2);

PHP 2013/11/15

// 関数
func1($arg1, $arg2);

// メソッド
$obj->method1($arg1, $arg2);

引数のカッコは省略不可。

クラスの中で同じインスタンスのメソッドを呼び出す場合であってもレシーバーを省略できない。

$this->method1($arg1, $arg2);
