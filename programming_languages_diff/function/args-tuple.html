タプルで複数の引数をまとめて渡すには

Python 2013/10/10

タプルを関数に渡すときに * を付けると、タプルの各要素を関数の引数として展開されて渡される。 可変長引数と逆のイメージ。

def f(a, b):
    ...

args = (10, "abc")

f(*args)

Ruby / JRuby 2015/03/18

Pythonと同じく、配列をメソッドに渡すときに * を付けると、配列の各要素をメソッドの引数として展開されて渡される。 可変長引数と逆のイメージ。

def f(a, b)
  print "a:#{a} b:#{b}\n"
end

args = [1, 2]
f(*args); # => a:1 b:2
