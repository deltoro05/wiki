関数・メソッド定義 / インデント / セミコロン

PHP

function f($arg1, $arg2){
  ...;
  return ...;
}
  • ブロックはC言語ライクに {} で表現
  • 行末のセミコロンは省略不可
  • 関数呼び出しはカッコを省略不可

Ruby

def f(arg1, arg2)
  ...
  return ...
end
  • ブロックはブロック開始のキーワード(def, do などの開始キーワードと end)で表現
  • 行末のセミコロンは省略可
  • メソッド呼び出しはカッコを省略可

Scala

def f(arg1: String, arg2: Int): String = {
  ...
}
  • ブロックはC言語ライクに {} で表現
  • 行末のセミコロンは省略可
  • メソッド呼び出しは引数のカッコ省略不可だが、引数がない場合の呼び出しはカッコを省略できる

CoffeeScript

(arg1, arg2) ->
  ...
  ...

引数がない場合

->
  ...
  ...
  • ブロックをインデントで表現
  • 行末のセミコロンは省略可
  • 関数呼び出しはカッコを省略可

例外処理

Ruby

  ...
  raise ...
  ...

  ...

  ...

  ...
end

rescue で特定のクラスの例外を補足したい場合は、以下のように書く。

rescue ZeroDivisionError => ex

rescue 節は複数書くことができる。

例外をスローするには、以下のような書き方がある。

raise "exception message ..."
raise ZeroDivisionError, "message ..."
