コマンドライン引数の取得

PHP 2013/05/17

$argc
引数の数。PHPスクリプトのファイル名が最初の引数になるので、これが0になることはない。
http://www.php.net/manual/ja/reserved.variables.argc.php
$argv
引数が配列で入っている。$argv[0] はPHPスクリプトのファイル名になる。
http://www.php.net/manual/ja/reserved.variables.argv.php

Ruby 2013/05/17

ARGV という定数に引数が配列で入っている。スクリプトファイル名は含まれておらず、先頭から本当の引数が入っている。

Perl 2013/07/31

@ARGV, $ARGV[..] で引数を取得できる。

ソースファイル名と行番号をソースコード中で取得するには

PHP 2013/08/23

__FILE__
__DIR__
__LINE__

__DIR__ はPHP 5.3.0以降。

Ruby 2013/05/26

__FILE__

sh 2013/07/05

bashでは、行番号は $LINENO という変数で取得できる。

そのソースファイルが他からの参照ではなく、直接実行された場合にのみ何らかの処理をするには

Ruby 2013/05/26

if __FILE__ == $PROGRAM_NAME
    # 何らかの処理
end

Python 2013/07/30

if __name__ =='__main__':
    # 何らかの処理

プロセスを終了するには

Ruby 2013/08/19

正常終了の場合

exit(0);

異常終了の場合(0以外を終了コードにする)

exit(1);

Perl 2013/07/27

正常終了の場合

exit(0);

異常終了の場合(0以外を終了コードにする)

exit(1);
