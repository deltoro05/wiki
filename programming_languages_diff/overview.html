コメント

C言語, C++

// 単一行コメント
/*
  複数行コメント
  複数行コメント
*/

複数行コメントはネスト不可。

C言語では、単一行コメントは正式にはC99以降の規格。

Perl

# 単一行コメント
=comment
  複数行コメント
  複数行コメント
=cut

PHP

// 単一行コメント
# 単一行コメント
/*
  複数行コメント
  複数行コメント
*/

複数行コメントはネスト不可。

http://php.net/manual/ja/language.basic-syntax.comments.php

コメントアウトを一時的に簡単に解除するようにしておくには 2013/04/24

以下のようにすると、1行目を // /* にすることで、コメントアウトの一時的な解除を簡単にできる。

/*
  コメントアウトされたコード
  コメントアウトされたコード
// */

以下のようにすると、1行目を // /* にすることで、コメントアウトのコードと生きているコードを簡単に入れ替えできる。

/*
  コメントアウトされたコード
  コメントアウトされたコード
/*/
  生きているコード
  生きているコード
// */

Ruby

# 単一行コメント
=begin
  複数行コメント
  複数行コメント
=end

Python

# 単一行コメント

複数行のコメントはない。そのかわり、文字列を置いてもプログラムの意味が変わらない箇所で、複数行の文字列リテラルを置くことで、複数行コメントの代わりにすることができる。

'''
複数行コメント
複数行コメント
'''

JavaScript

*/

CoffeeScript

###
  複数行コメント
  複数行コメント
###

インデント / セミコロン

C言語, C++

  • ブロックは {} で表現
  • 行末のセミコロンは省略不可

PHP

  • ブロックはC言語ライクに {} で表現
  • 行末のセミコロンは省略不可

Ruby

  • ブロックはブロック開始のキーワード(def, do などの開始キーワードと end)で表現
  • 行末のセミコロンは省略可

Scala

  • ブロックはC言語ライクに {} で表現
  • 行末のセミコロンは省略可

CoffeeScript

  • ブロックをインデントで表現
  • 行末のセミコロンは省略可

関数・メソッド定義

PHP

function f($arg1, $arg2){
  ...;
  return ...;
}
  • 関数呼び出しはカッコを省略不可

Ruby

def f(arg1, arg2)
  ...
  return ...
end
  • メソッド呼び出しはカッコを省略可

Scala

def f(arg1: String, arg2: Int): String = {
  ...
}
  • メソッド呼び出しは、引数がない場合に限り、カッコを省略できる

CoffeeScript

(arg1, arg2) ->
  ...
  ...

引数がない場合

->
  ...
  ...
  • 関数呼び出しは、引数がある場合はカッコを省略可。引数がない場合はカッコを省略できない。

例外処理

Ruby

begin
  ...
  raise ...
  ...
rescue
  ...
else
  ...
ensure
  ...
end

rescue で特定のクラスの例外を補足したい場合は、以下のように書く。

rescue ZeroDivisionError => ex

rescue 節は複数書くことができる。

例外をスローするには、以下のような書き方がある。

raise "exception message ..."
raise ZeroDivisionError, "message ..."

Python

try:
  ...
  raise "message ..."
  raise HogeException()
  ...
except:
  ...
finally:
  ...

except で例外オブジェクトを扱う場合は、 except ex: のように書く。

特定の例外クラスだけ except で扱う場合は、 except HogeException: または except HogeException, ex: のように書く。

except の中で同じ例外を再スローするには raise とだけ書く。

JavaScript

try {
  ...
  throw "message ...";
  ...
} catch(ex){
  ...
} finally {
  ...
}

CoffeeScript

try
  ...
  throw("message")
  ...
catch ex
  ...
finally
  ...

複数の返り値

PHP

配列を使って無理やりやろうと思えばできなくもない。

function f(){
  return array(1, "abc");
}
list($num, $str) = f();

Ruby

return にカンマ区切りで複数の値を書けば、配列を返すことができ、受け取り側は配列の多重代入を使う。

def f
  return 1, "abc"
end
num, str = f
