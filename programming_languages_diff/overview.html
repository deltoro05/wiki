コメント

C言語, C++ 2013/04/22

// 単一行コメント
/*
  複数行コメント
  複数行コメント
*/

複数行コメントはネスト不可。

C言語では、単一行コメントは正式にはC99以降の規格。

コメントアウトを一時的に簡単に解除するようにしておくには 2013/04/24

以下のようにすると、1行目を // /* にすることで、コメントアウトの一時的な解除を簡単にできる。

/*
  コメントアウトされたコード
  コメントアウトされたコード
// */

以下のようにすると、1行目を // /* にすることで、コメントアウトのコードと生きているコードを簡単に入れ替えできる。

/*
  コメントアウトされたコード
  コメントアウトされたコード
/*/
  生きているコード
  生きているコード
// */

インデント / セミコロン

C言語, C++ 2013/04/22

  • ブロックは {} で表現
  • 行末のセミコロンは省略不可

Go言語 2013/06/19

  • ブロックは {} で表現
  • 行末のセミコロンは省略可

Scala 2013/04/22

  • ブロックはC言語ライクに {} で表現
  • 行末のセミコロンは省略可

PHP 2013/04/22

  • ブロックはC言語ライクに {} で表現
  • 行末のセミコロンは省略不可

Ruby 2013/04/22

  • ブロックはブロック開始のキーワード(def, do などの開始キーワードと end)で表現
  • 行末のセミコロンは省略可

Python 2013/06/30

  • ブロックをインデントで表現
  • 行末のセミコロンは省略可(省略推奨)

CoffeeScript 2013/04/22

  • ブロックをインデントで表現
  • 行末のセミコロンは省略可

コマンドライン引数の取得

PHP 2013/05/17

$argc
引数の数。PHPスクリプトのファイル名が最初の引数になるので、これが0になることはない。
http://www.php.net/manual/ja/reserved.variables.argc.php
$argv
引数が配列で入っている。$argv[0] はPHPスクリプトのファイル名になる。
http://php.net/manual/ja/reserved.variables.argv.php

Ruby 2013/05/17

ARGV という定数に引数が配列で入っている。スクリプトファイル名は含まれておらず、先頭から本当の引数が入っている。

ソースファイル名と行番号をソースコード中で取得するには

Ruby 2013/05/26

__FILE__

sh 2013/07/05

bashでは、行番号は $LINENO という変数で取得できる。

そのソースファイルが他からの参照ではなく、直接実行された場合にのみ何らかの処理をするには

Ruby 2013/05/26

if __FILE__ == $PROGRAM_NAME
    # 何らかの処理
end
