コメントアウトを一時的に簡単に解除するようにしておくには 2013/04/24

以下のようにすると、1行目を // /* にすることで、コメントアウトの一時的な解除を簡単にできる。

/*
  コメントアウトされたコード
  コメントアウトされたコード
// */

以下のようにすると、1行目を // /* にすることで、コメントアウトのコードと生きているコードを簡単に入れ替えできる。

/*
  コメントアウトされたコード
  コメントアウトされたコード
/*/
  生きているコード
  生きているコード
// */

変数

Perl 2013/07/30

$ で始まる変数は、配列やハッシュ(連想配列 or マップ)以外のアトミックな値を保存する変数。
@ で始まる変数は、配列を保存する変数。要素にアクセスするには $...[..] とする。
% で始まる変数は、ハッシュを保存する変数。要素にアクセスするには $...{..} とする。

$, @, % の3つはシジル(sigil)という。

変数宣言をせずにいきなり値を代入するとグローバル変数になる。
my で宣言すると、ローカル変数になる。

ローカル変数では、{} のブロックの中で宣言すると、ブロックの中のスコープを持つ。

ENV['HOME']

Perl 2013/07/27

$ENV{'HOME'}

コマンドライン引数の取得

PHP 2013/05/17

$argc
引数の数。PHPスクリプトのファイル名が最初の引数になるので、これが0になることはない。
http://www.php.net/manual/ja/reserved.variables.argc.php
$argv
引数が配列で入っている。$argv[0] はPHPスクリプトのファイル名になる。
http://php.net/manual/ja/reserved.variables.argv.php

Ruby 2013/05/17

ARGV という定数に引数が配列で入っている。スクリプトファイル名は含まれておらず、先頭から本当の引数が入っている。

Perl 2013/07/31

@ARGV, $ARGV[..] で引数を取得できる。

ソースファイル名と行番号をソースコード中で取得するには

__FILE__

sh 2013/07/05

そのソースファイルが他からの参照ではなく、直接実行された場合にのみ何らかの処理をするには

    # 何らかの処理
