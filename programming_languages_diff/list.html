配列の書き方

Java

arr = {1, 2, 3};
arr = new int[]{1, 2, 3};

連想配列は一発では作れない。

import java.util.HashMap;
map = new HashMap();
map.put("a", 1);
map.put("b", 2);

Scala

arr = Array(1, 2, 3);
map = Map('a' -> 1, 'b' -> 2);

これは以下のシンタックスシュガー。

arr = scala.Array.apply(1, 2, 3);
map = scala.collection.immutable.Map.apply(scala.Predef.any2ArrowAssoc('a').->(1),
                                           scala.Predef.any2ArrowAssoc('a').->(2));

PHP

arr = array(1, 2, 3);
map = array('a' => 1, 'b' => 2);

Ruby

arr = [1, 2, 3]
map = {'a' => 1, 'b' => 2}

JavaScript

下の3つは全部同じ配列を生成する。

arr = [1, 2, 3];
arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;

下の2つは全部同じ連想配列を生成する。

map = {a: 1, b: 2}
map = {'a': 1, 'b': 2}

subList

Java
