リスト(配列)の書き方

Java

配列

arr = {1, 2, 3};
arr = new int[]{1, 2, 3};

インデックスで参照するには

arr[index];

Scala

arr = Array(1, 2, 3);
list = List(1, 2, 3);

これは以下と同じ意味。

arr = scala.Array.apply(1, 2, 3);
list = scala.collection.immutable.List.apply(1, 2, 3);

インデックスで参照するには

arr(index);
list(index);

これは以下と同じ意味。

arr.apply(index);
list.apply(index);

PHP

arr = array(1, 2, 3);

PHP 5.4からは以下のようにも書ける。

arr = [1, 2, 3];

インデックスで参照するには

arr[index];

Ruby

arr = [1, 2, 3]

インデックスで参照するには

arr[index];

Python

arr = [1, 2, 3]

インデックスで参照するには

arr.index(idx)

JavaScript

下の3つは全部同じ配列を生成する。

arr = [1, 2, 3];
arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;

size

Java

int List#size()

Scala

# Seq[+A]
def length: Int
def size: Int

2つのメソッドは全く同じ。

Ruby

Array#length
Array#size

2つのメソッドは全く同じ。

JavaScript

arr.length

カッコはいらない。

厳密にはサイズではなくインデックスの最大値+1(でも負にはならない)を表すので、以下のようになる。

arr = [];
console.log(arr.length); // => 0
arr[100] = 1;
console.log(arr.length); // => 101

arr = [];
console.log(arr.length); // => 0
arr[-2] = 1;
console.log(arr.length); // => 0

foreach

Java

配列をループ。

for(int i = 0; i < arr.length; i++){
  ...
}

Java5からは、拡張for文の利用が可能。

for(String str: str_arr){
  ...
}

Scala

# Seq[+A]
def foreach(f: A => Unit): Unit

PHP

配列をループ。

for(int $i = 0; $i < $arr.length; $i++){
  ...
}

foreach($arr as $value){
  ...
}

連想配列をループ。

foreach($arr as $key => $value){
  ...
}

Ruby

for value in arr do
  ...
end

value の部分には複数の値が入る場合に複数の変数が使える。例：

for i, str in [[1, 'abc'], [2, 'def'], [3, 'ghi']]
  ...
end

配列の each メソッドも使える。 break でのループの中断は、上記の for 文以外にこの each メソッドでも使える。例：

[1, 2, 3].each do |i|
  p i
  break if i == 2
end
# => 1 2

CoffeeScript

for elem in arr
  ....
