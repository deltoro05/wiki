リスト(配列)の書き方

Java

配列

arr = {1, 2, 3};
arr = new int[]{1, 2, 3};

インデックスで参照するには

arr[index];

Scala

arr = Array(1, 2, 3);
list = List(1, 2, 3);

これは以下と同じ意味。

arr = scala.Array.apply(1, 2, 3);
list = scala.collection.immutable.List.apply(1, 2, 3);

インデックスで参照するには

arr(index);
list(index);

これは以下と同じ意味。

arr.apply(index);
list.apply(index);

PHP

arr = array(1, 2, 3);

PHP 5.4からは以下のようにも書ける。

arr = [1, 2, 3];

インデックスで参照するには

arr[index]

配列 | PHP Manual
http://jp2.php.net/manual/ja/language.types.array.php

配列関係の関数 | PHP Manual
http://jp2.php.net/manual/ja/book.array.php

Ruby

arr = [1, 2, 3]
arr = %w(abc def ghi) # => ['abc', 'def', 'ghi'] と同じ

%w の記法は、カッコは比較的なんでもよくて、以下はすべて同じ意味になる。

['abc', 'def', 'ghi']
%w(abc def ghi)
%w[abc def ghi]
%w!abc def ghi!

インデックスで参照するには

arr[index]

Python

arr = [1, 2, 3]

インデックスで参照するには

arr.index(idx)

JavaScript

下の3つは全部同じ配列を生成する。

arr = [1, 2, 3];
arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;

JavaScript の配列と連想配列の違い

CoffeeScript

arr = [1, 2, 3]
arr = [1, 2, 3, ]
arr = [
    1
    2
    3
]

size

Java

int List<E>#size()

Scala

def Seq[A]#length: Int
def Seq[A]#size: Int

2つのメソッドは全く同じ。

PHP

count($arr)
sizeof($arr)

Ruby

Array#length
Array#size

2つのメソッドは全く同じ。文字列の長さも同じ名前の2つのメソッドがある。

JavaScript

arr.length

カッコはいらない。

厳密にはサイズではなくインデックスの最大値+1(でも負にはならない)を表すので、以下のようになる。

arr = [];
console.log(arr.length); // => 0
arr[100] = 1;
console.log(arr.length); // => 101

arr = [];
console.log(arr.length); // => 0
arr[-2] = 1;
console.log(arr.length); // => 0

foreach

Java

配列をループ。

for(int i = 0; i < arr.length; i++){
  ...
}

Java5からは、拡張for文の利用が可能。

for(String str: str_arr){
  ...
}

Scala

def Seq[A]#foreach(f: A => Unit): Unit

PHP

for(int $i = 0; $i < $arr.length; $i++){
  ...
}

foreach($arr as $value){
  ...
}

Ruby

for value in arr do
  ...
end

value の部分には複数の値が入る場合に複数の変数が使える。例：

for i, str in [[1, 'abc'], [2, 'def'], [3, 'ghi']]
  ...
end

配列の each メソッドも使える。 break でのループの中断は、上記の for 文以外にこの each メソッドでも使える。例：

[1, 2, 3].each do |i|
  p i
  break if i == 2
end
# => 1 2

CoffeeScript

for elem in arr
  ....

foldLeft

Scala

def Seq[A]#foldLeft[B](initial: B)(f: (B, A) => B): B

Ruby

arr = [1, 2, 3]
arr.inject(:+)               # => 6
arr.inject {|i, x| i + x}    # => 6
arr.inject(0) {|i, x| i + x} # => 6

inject に初期値を与えないと、1つ目の要素が初期値であるかのような動作をする。配列の要素が1つもなくて初期値も与えないと、nil が返される。

grouped

Scala

def Seq[A]#grouped(size: Int): Iterator[Seq[A]]

Ruby

以下のような関数で同様のことができる。

def grouped(src, size)
  dst = []
  index = 0
  len = dst.size
  while index < len do
    group_data = (0 .. (size - 1)).map do |i|
      src[index + i]
    end
    dst.push(group_data)
    index = index + size
  end
  return dst
end
