概要

Java 2013/09/23

java.util.Dateクラス、java.util.Calendarクラス、 Java8からは java.time.LocalDateTimeクラス, java.time.ZonedDateTimeクラスがある。

Ruby 2013/09/01

Timeクラス、Dateクラス、DateTimeクラスがある。Dateクラスまたは DateTimeクラスを使うには require 'date' する必要がある。

日付オブジェクト・日時オブジェクトの各フィールドを取得するには 2013/09/01

# Timeクラス
time.year
time.month
time.day    # その月の中での日付(1-31)
time.wday   # 曜日(0-6)。日曜日が0
time.hour
time.min
time.sec

# DateTimeクラス
datetime.year
datetime.month
datetime.day    # その月の中での日付(1-31)
datetime.wday   # 曜日(0-6)。日曜日が0
datetime.hour
datetime.min
datetime.sec

# Dateクラス
date.year
date.month
date.day    # その月の中での日付(1-31)
date.wday   # 曜日(0-6)。日曜日が0
