時刻を計算するには

Ruby 2013/09/01

# timeがTime型だとして
time + 3       # 3秒後の時刻
time1 - time2  # 時刻の差を秒単位の整数で計算

Date型またはDateTime型だと日単位での計算になる。-> 日付単位の計算

sh 2013/12/21

現在時刻を基準にした計算であれば簡単にできる。 dateコマンドで簡単にできる。

例

# 100秒前
date --date "-100 seconds"
