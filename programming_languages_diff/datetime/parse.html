文字列を日付に変換するには

PHP 2014/05/07

var_export(strptime('2013/06/16', '%Y/%m/%d'));
// => array (
//   'tm_sec' => 0,
//   'tm_min' => 0,
//   'tm_hour' => 0,
//   'tm_mday' => 16,
//   'tm_mon' => 5,     // 1を足すと月になる
//   'tm_year' => 113,  // 1900を足すと西暦になる
//   'tm_wday' => 0,
//   'tm_yday' => 166,
//   'unparsed' => '',
// )

パースできない場合は strptime は FALSE を返す。

strptime 関数 | PHP Manual
http://www.php.net/manual/ja/function.strptime.php

Ruby 2013/06/16

require 'date'
date = Date.strptime('2013/03/17', '%Y/%m/%d')
p [date.year, date.month, date.day, date.wday]

date.wday で曜日を整数で取得でき、日曜日が0になる。
