同じ要素数の複数のリストを要素ごとにタプルにまとめて1つのリストを作るには (zip)

Scala 2013/09/17

// Iterable[A]
def zip(that: Iterable[B]): Iterable[(A, B)]

// Seq[A]
def zip(that: Iterable[B]): Seq[(A, B)]

Ruby 2013/05/06

Array#zip(a1, a2, ...)

例

['a', 'b'].zip([1, 2])         # => [["a", 1], ["b", 2]]
['a', 'b'].zip([1, 2], [3, 4]) # => [["a", 1, 3], ["b", 2, 4]]
