範囲外のインデックスにアクセスしたときにどうなるか

Ruby 2013/08/08

nil が返ってくる。インデックスをマイナスにすると後ろから数えてくれるので必ずしも nil にはならない。

arr = [1, 2, 3]
p arr[2] # => 3
p arr[3] # => nil
p arr[-1] # => 3
p arr[-3] # => 1
p arr[-4] # => nil

Python 2013/08/08

例外がスローされる。

Perl 2013/09/14

undef になり、use warnings; と宣言していると、 Use of uninitialized value という警告が表示される。

インデックスをマイナスにすると後ろから数えてくれる。

my @a = (1, 2, 3);
print $a[-1]; # => 3
print $a[-3]; # => 1
print $a[-4]; # => Use of uninitialized value

JavaScript 2013/08/08

undefined が返ってくる。
