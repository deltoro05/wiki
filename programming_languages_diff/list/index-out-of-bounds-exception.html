範囲外のインデックスにアクセスしたときにどうなるか？

Ruby 2013/08/08

nil が返ってくる。インデックスをマイナスにすると後ろから数えてくれるので必ずしも nil にはならない。

arr = [1, 2, 3]
p arr[2] # => 3
p arr[3] # => nil
p arr[-1] # => 3
p arr[-3] # => 1
p arr[-4] # => nil

Python 2013/08/08

例外がスローされる。

JavaScript 2013/08/08

undefined が返ってくる。
