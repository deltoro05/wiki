リストから特定の要素(特定の条件を満たす要素)を探してそのインデックスを取得するには (indexOf) 2014/07/10

インデックスが不要で、存在するかどうかの論理値だけがほしい場合は contains 参照。

Java 2014/05/17

Signature:

// java.util.List
int indexOf(Object obj)
int lastIndexOf(Object obj)

要素が見つからなかった場合は -1 を返す。

複数ある場合は最初の要素のインデックスを返す。

要素の比較は equals メソッドで行われる。

Scala 2014/05/17

Signature:

// scala.collection.Seq
def indexOf(obj: A): Int
def indexOf(obj: A, from: Int): Int
def indexWhere(p: A => Boolean): Int
def indexWhere(p: A => Boolean, from: Int): Int

要素が見つからなかった場合は -1 を返す。(Option[Int]じゃないところがちょっと気持ち悪い)

複数ある場合は最初の要素のインデックスを返す。

indexOfメソッドでの要素の比較はequalsメソッドで行われる。 indexWhereメソッドではp関数がtrueを返す要素を取得する。

PHP 2013/06/11

array_search($obj, $arr)

見つからなかった場合は false を返す。0 を返す場合と区別するために以下のように === または !== での比較が必要。

if(array_search($obj, $arr) === false){
    ...
}

array_search 関数 | PHP Manual
http://www.php.net/manual/ja/function.array-search.php

Python 2013/10/03

例

lst.index(obj);

要素が見つからなかった場合はエラーを吐く。

複数ある場合は最初の要素のインデックスを返す。

Ruby / JRuby 2015/01/01

Array#index または Array#find_index を使う。2つはまったく同じ。

見つからなかった場合は nil を返す。
