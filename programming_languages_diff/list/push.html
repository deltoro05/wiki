push 2013/07/31

リストの最後に要素を追加する。

先頭への追加はunshift。最後の要素の削除はpop。

PHP 2013/04/22

$arr[] = $obj;

array_push でも同じ。

array_push | PHP Manual
http://jp2.php.net/manual/ja/function.array-push.php

Python 2013/08/21

arr.append(obj)

破壊的メソッド

Ruby 2013/08/21

Array#push(obj)

破壊的メソッド

複数の要素を最後に追加したい、つまり配列を連結させるには、

Array#concat(other_array)

concat は、破壊的メソッド。非破壊的メソッドは Array#+ 。

Perl 2013/08/21

push(@arr, $obj);

破壊的メソッド

JavaScript 2013/08/21

破壊的メソッド

破壊的メソッド

複数の引数を渡すこともできる。

var arr1 = [1, 2, 3];
arr1.push(4, 5, 6);
console.log(arr1); // => [1, 2, 3, 4, 5, 6]
