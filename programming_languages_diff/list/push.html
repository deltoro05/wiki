リストの最後に要素1つまたは複数追加するには (push) 2013/09/07

先頭への追加はunshift。
最後の要素の削除はpop。

Scala 2013/07/30

// Seq[A]
def ++(arr2: A): Seq[A]

追加した結果のリストを返し、自身は変更しない。

PHP 2013/09/07

$arr[] = $obj;

配列自身を変更する破壊的な方法。

array_push でも同じ。

array_push | PHP Manual
http://www.php.net/manual/ja/function.array-push.php

複数の要素を配列で指定してまとめて追加するには

array_merge($arr1, $arr2);

array_merge | PHP Manual
http://www.php.net/manual/ja/function.array-merge.php

Python 2013/10/04

lst.append(obj)

これはリスト自身を変更する破壊的メソッドである。

複数の要素を配列で指定してまとめて追加するには

lst.extend(lst2)

これも破壊的メソッド。

タプルはイミュータブルなので、これらに相当するメソッドはない。

Ruby 2013/08/28

Array#push(obj)

例

arr.push(obj1)

これは配列自身を変更する破壊的メソッドで、配列自身を返す。以下のように、複数の要素を追加することもできる。

arr.push(obj1, obj2, obj3)

Array#<< を使うこともできる。このメソッドは引数1つしか使えないが、配列自身を返すので、連続して書くことができる。

arr << obj1 << obj2 << obj3

配列を連結させるには、

Array#concat(other_array)

これは破壊的メソッドである。これに対応する非破壊的メソッドは Array#+ 。

例

arr1 + arr2

Perl 2013/09/07

push(@arr, $obj);

破壊的メソッド

2つ目の引数に配列を渡すことができ、その場合はその配列の要素をすべて追加する。

push(@arr1, @arr2);

JavaScript 2013/09/07

破壊的メソッド

Array#push(obj)

破壊的メソッド

複数の引数を渡すこともできる。

var arr1 = [1, 2, 3];
arr1.push(4, 5, 6);
console.log(arr1); // => [1, 2, 3, 4, 5, 6]

非破壊的に新しい配列を作成するには以下のようにする。

例

var arr1 = [1, 2, 3];
var arr2 = [1, 2, 3];
var new_arr = arr1.concat(arr2)

arr1 自身は変更されない。
