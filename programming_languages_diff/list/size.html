int List<E>#size()

// Seq[A]
def length: Int
def size: Int

2つのメソッドは全く同じ。

PHP 2013/04/27

count($arr)
sizeof($arr)

Ruby 2013/04/16

Array#length
Array#size

2つのメソッドは全く同じ。文字列の長さも同じ名前の2つのメソッドがある。

Perl 2013/07/31

スカラコンテキストで @... を使うと要素の数を取得できる。

my @arr = ('abc', 'def', 'ghi');
print @arr;        # => abcdefghi
print scalar @arr; # => 3
print @arr + 0;    # => 3

$#... で最後の要素のインデックスを取得できる。

print $#arr;    # => 2

JavaScript 2013/04/19

arr.length

カッコはいらない。

厳密にはサイズではなくインデックスの最大値+1(でも負にはならない)を表すので、以下のようになる。

arr = [];
console.log(arr.length); // => 0
arr[100] = 1;
console.log(arr.length); // => 101

arr = [];
console.log(arr.length); // => 0
arr[-2] = 1;
console.log(arr.length); // => 0
