リストの要素の数を取得するには (size)

Java 2014/05/17

配列の場合の例

int size = arr.length;

配列の length はメソッドではないので括弧がいらない。

java.util.List での例

int size = lst.size();

Signature:

// java.util.List<E>
int size()

メソッドであるので括弧が必要。

Scala 2014/10/20

Javaの配列(Array)であってもScalaのコレクション型であっても size というメソッドが使える。 Javaの配列ではJavaと同じlengthもメソッドとして使える。

Signature:

// scala.Array[A]
def length: Int
def size: Int

// scala.collection.Seq[A]
def size: Int

上記はメソッドだが括弧はいらない。

PHP 2014/05/17

例

count($arr)
sizeof($arr)

この2つの関数はまったく同じ意味で、配列の要素の数を返す。

Python 2014/10/20

以下はPythonのシーケンスといわれるリスト、タプル、文字列で共通操作。

len(lst)

ディクショナリに対しても同じ関数が使える。

Ruby / JRuby 2014/12/31

Array#length または Array#size を使う。

l = arr.length
# または
l = arr.size

2つのメソッドは全く同じ。ハッシュのサイズ、

文字列の長さの取得も同じ名前の2つのメソッドでできる。

Perl 2013/07/31

スカラコンテキストで @... を使うと要素の数を取得できる。

my @arr = ('abc', 'def', 'ghi');
print @arr;        # => abcdefghi
print scalar @arr; # => 3
print @arr + 0;    # => 3

$#... で最後の要素のインデックスを取得できる。

print $#arr;    # => 2

JavaScript 2013/04/19

arr.length

カッコはいらない。

厳密にはサイズではなくインデックスの最大値+1(でも負にはならない)を表すので、以下のようになる。

arr = [];
console.log(arr.length); // => 0
arr[100] = 1;
console.log(arr.length); // => 101

arr = [];
console.log(arr.length); // => 0
arr[-2] = 1;
console.log(arr.length); // => 0
