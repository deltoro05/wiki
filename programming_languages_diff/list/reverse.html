配列(リスト)を逆順に並び替えるには (reverse)

Java

java.util.List 2015/04/25

import java.util.Collections;

List<String> lst = ...;
Collections.reverse(lst); # lst自体を変更

Signature:

void java.util.Collections.reverse(List<?> list)

リスト自体を逆順に書き換える破壊的なメソッド。Collectionsのstaticメソッドである。

java.util.Collections.reverse | Java Platform SE 8 Javadoc
http://docs.oracle.com/javase/8/docs/api/java/util/Collections.html#reverse-java.util.List-

Scala 2013/07/30

Signature:

// scala.collection.Seq[A]
def reverse: Seq[A]

反転した結果のリストを返し、自身は変更しない。

PHP 2015/01/11

array_reverse 関数で逆順の配列を生成できる。この関数は引数の配列自身を変更しない。

$arr = array(10, 20, 30, 40, 50);

var_export(array_reverse($arr));
// 出力結果
// array (
//   0 => 50,
//   1 => 40,
//   2 => 30,
//   3 => 20,
//   4 => 10,
// )

var_export($arr);
// 出力結果
// array (
//   0 => 10,
//   1 => 20,
//   2 => 30,
//   3 => 40,
//   4 => 50,
// )

array_reverse 関数 | PHP Manual
http://php.net/manual/ja/function.array-reverse.php

Python 2013/10/06

# 逆順のリストを返し、引数のリスト自体は変更しない
reversed(lst)

# リスト自体を逆順にしたものに変更する
lst.reverse()

Ruby / JRuby 2015/01/04

Array#reverse メソッドは逆順の配列を生成して返す。 Array#reverse! メソッドは配列自身を逆順に変更して自身を返す。

p [10, 20, 30, 40, 50].reverse
# => [50, 40, 30, 20, 10]

Array#reverse, Array#reverse! | Ruby 2.1 リファレンスマニュアル
http://docs.ruby-lang.org/ja/2.1.0/method/Array/i/reverse.html

Perl 2013/09/11

reverse(@arr)

並び替えられた新しい配列を返し、引数の配列自体は変更しない。

sh (シェルスクリプト) 2013/12/20

行単位で逆順にするには tacコマンド を使えばよい。

JavaScript 2013/04/20
