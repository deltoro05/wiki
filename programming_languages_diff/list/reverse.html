リストを逆順に並び替えるには (reverse)

Java 2013/10/01

// java.util.Collections
static void reverse(List<?> list)

リスト自体を逆順に書き換える。

Scala 2013/07/30

// Seq[A]
def reverse: Seq[A]

反転した結果のリストを返し、自身は変更しない。

PHP 2013/04/27

array array_reverse($arr)

Python 2013/10/06

# 逆順のリストを返し、引数のリスト自体は変更しない
reversed(lst)

# リスト自体を逆順にしたものに変更する
lst.reverse()
