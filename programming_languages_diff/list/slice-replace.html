配列(リスト)の一部を別の配列で置き換えるには (array_splice / splice) 2015/03/31

各プログラミング言語での、配列(リスト)の一部を別の配列で置き換える方法。

関連

  • 配列の一部を配列として取り出すには

PHP 2013/11/08

array_splice関数を使う。

$arr = array(10, 20, 30, 40, 50);
var_export($arr); echo "\n";
// =>
// array (
//   0 => 10,
//   1 => 20,
//   2 => 30,
//   3 => 40,
//   4 => 50,
// )

array_splice($arr, 3, 1, array(100, 200, 300));
var_export($arr); echo "\n";
// =>
// array (
//   0 => 10,
//   1 => 20,
//   2 => 30,
//   3 => 100,
//   4 => 200,
//   5 => 300,
//   6 => 50,
// )

array_splice | PHP Manual
http://php.net/manual/ja/function.array-splice.php

Python 2014/12/31

スライスの書式を代入演算子の左辺に置くと、その部分リストを新しいリストに置き換えられる。

lst = [10, 20, 30, 40, 50]

lst[3:4] = [100, 200, 300]
print(lst)
# => [10, 20, 30, 100, 200, 300, 50]

lst[2:] = [1, 2, 3]
print(lst)
# => [10, 20, 1, 2, 3]

Ruby / JRuby 2014/12/31

Array#[] を使ったスライスを代入演算子の左辺に置くと、その部分リストを新しいリストに置き換えられる。

arr = [10, 20, 30, 40, 50]

arr[3..4] = [100, 200, 300]
p arr
# => [10, 20, 30, 100, 200, 300]

Pythonと違って、[]の中の2つ目の数字のインデックス位置を含むので、見た目同じでも結果が異なる。また、Pythonの lst[2:] みたいに後ろを省略する書き方ができない。

[] の中を範囲指定ではなくfromとlengthを指定しうる方法でもできる。

arr = [10, 20, 30, 40, 50]

arr[3, 1] = [100, 200, 300]
p arr
# => [10, 20, 30, 100, 200, 300, 50]
