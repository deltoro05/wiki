特定の要素をキーで指定して削除するには (remove)

Java 2013/09/21

Signature:

// java.util.List[E]
E remove(int index);

Python 2013/10/06

リストの指定のインデックスの要素を取得し、削除して詰める。

lst.pop(index)
# または
del(lst[index])

popは引数を指定しないと最後の要素を取得して削除する。(->pop)
