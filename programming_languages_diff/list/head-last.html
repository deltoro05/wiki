Scala 2014/10/23

Signature:

// scala.collection.Seq[A]
def head: A
def headOption: Option[A]
def last: A
def lastOption: Option[A]

空のSeqに対して、head や last を呼び出すと、 java.util.NoSuchElementException の例外がスローされてしまう。

Ruby / JRuby 2015/01/04

Array#first, Array#last というメソッドを使う。配列が空の場合は nil になる。

arr = [10, 20, 30, 40, 50]

# 先頭の要素
p arr.first
p arr[0]
# => 10

# 最後の要素
p arr.last
p arr[-1]
# => 50

firstやlastは数字を引数で与えると、先頭または最後からその数だけ要素を抜き出して、配列で返してくれる。

arr = [10, 20, 30, 40, 50]

p arr.first
# => 10
p arr.first(1)
# => [10]
p arr.first(3)
# => [10, 20, 30]

p arr.last
# => 50
p arr.last(1)
# => [50]
p arr.last(3)
# => [30, 40, 50]

Perl 2014/01/11

# 配列の先頭の要素を取得
$arr[0]

# 配列の最後の要素を取得
$arr[$#arr]

$#arr で最後の要素のインデックスを取得できる。
