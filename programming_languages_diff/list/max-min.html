配列(リスト)の中の最大値・最小値を求めるには (max, min)

Scala 2015/01/19

Signature:

// scala.collection.Seq[A]
def max: A
def min: A

例

val lst = List(20, 30, 40, 10, 20, 30);

println(lst.max);
// => 40
println(lst.min);
// => 10

要素が文字列の場合は辞書順になる。Scalaでは文字列と数字は型が違っていて勝手に変換することはないので、文字列を数字として比較することはない。

val lst = List("8", "10", "20");

println(lst.max);
// => 8
println(lst.min);
// => 10

リストが空の場合は java.lang.UnsupportedOperationException: empty.max というような例外がスローされる。

Python 2013/10/03

max(lst)
min(lst)

リストが空の場合はエラーを吐く。

NumPy 2013/10/15

NumPyの配列の中で最大値を求めるには numpy.argmax を使う。 numpy.argmax は最大値となる要素のインデックスを返し、最大値が複数ある場合はその中で最初のインデックスを返す。

import numpy as np

x = np.array([10, 20, 30, 10, 20, 30, 40])
print np.argmax(x)
# => 2

x = np.array([10, 20, 30, 10, 20, 30, 40])
print np.argmax(x)
# => 6

同様にNumPyの配列の中で最小値を求めるには numpy.argmin を使う。

Perl 2015/01/24

List::Util というモジュールにあるmax関数またはmin関数を使う。

use List::Util qw/max min/;

my @arr = (20, 30, 40, 10, 20, 30);

print max(@arr);
# => 40

print min(@arr);
# => 10

JavaScript 2014/12/08

Math.max, Math.min を使う。

例

var arr = [3, 5, 7, 3, 5];
var max = Math.max.apply(null, arr);
var min = Math.min.apply(null, arr);

console.log(max);
// => 7
console.log(min);
// => 3
