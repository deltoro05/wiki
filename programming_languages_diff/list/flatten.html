flatten

Scala 2013/07/30

// Seq[A]
def flatten[B]: Seq[B]

処理結果のリストを返し、自身は変更しない。

Ruby 2013/05/06

Array#flatten
Array#flatten(level)

level を指定すると、その数のレベルだけ平坦にする。

例

arr = [1, [2, [3, 4]]]
arr.flatten(1) #=> [1, 2, [3, 4]]
arr.flatten(2) # => [1, 2, 3, 4]
arr.flatten    # => [1, 2, 3, 4]
