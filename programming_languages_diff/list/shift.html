shift 2013/04/22

リストの先頭の要素を取得し、削除する。

PHP 2013/04/22

array_shift($arr);

array_shift | PHP Manual
http://www.php.net/manual/ja/function.array-shift.php

Ruby 2013/08/04

Array#shift

要素が1つもない場合は、 nil を返す。

JavaScriptと同じメソッド名。

Python 2013/07/31

arr.pop(0)

引数を指定しないと最後の要素を取得して削除する。(->pop)

以下の例は、リストの指定のインデックスの要素を取得し、削除して詰める。

arr.pop(index)

Perl 2013/07/31

shift(@arr);

引数を指定しないと、mainモジュールの中では @ARGV を対象とし、 subの中では @_ を対象とする。

JavaScript 2013/04/19

Array#shift

Rubyと同じメソッド名。

CoffeeScript 2013/04/22

Array#shift

これはJavaScriptと同じ。
