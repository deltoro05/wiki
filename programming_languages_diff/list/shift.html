リストの先頭から要素を1つ取得してリストから削除するには (shift) 2013/09/11

最後の要素の削除はpop。
先頭への追加はunshift。

PHP 2013/04/22

array_shift($arr);

array_shift | PHP Manual
http://www.php.net/manual/ja/function.array-shift.php

Ruby 2013/08/04

Array#shift

要素が1つもない場合は、 nil を返す。

JavaScriptと同じメソッド名。

Python 2013/09/21

arr.pop(0)

引数を指定しないと最後の要素を取得して削除する。(->pop)

以下の例は、リストの指定のインデックスの要素を取得し、削除して詰める。(->remove)

arr.pop(index)

Perl 2013/09/26

shift(@arr);

引数を指定しないと、mainモジュールの中では @ARGV を対象とし、 subの中では @_ を対象とする。

リストの先頭の要素を返し、配列自身は先頭が削除される。
