配列(リスト)をコピーするには (clone) 2013/10/22

イミュータブルなオブジェクトのコピーは意味がないので、ここではミュータブルなリストのコピーについて。

PHP 2015/01/21

配列をコピーするには単に代入すればよい。

$src = array(1, 2, 3);
$dst = $src;
$src[0] = 10;

var_export($src);
// 出力結果
// array (
//   0 => 10,
//   1 => 2,
//   2 => 3,
// )

var_export($dst);
// 出力結果
// array (
//   0 => 1,
//   1 => 2,
//   2 => 3,
// )

要素が配列だとそれもコピーされるので、deep copyということか。

$src = array(1, array(2, 3));
$dst = $src;
$src[1][0] = 20;

var_export($src);
// 出力結果
// array (
//   0 => 1,
//   1 =>
//   array (
//     0 => 20,
//     1 => 3,
//   ),
// )

var_export($dst);
// 出力結果
// array (
//   0 => 1,
//   1 =>
//   array (
//     0 => 2,
//     1 => 3,
//   ),
// )
)

PHPでは普通の配列と連想配列は同じものなので、連想配列のコピーも同じ。

Python 2015/04/26

NumPyでない普通の配列であれば以下の書き方を使う。

dst = src[:]

[:] はスライス表記であり、この書き方で shallow copy が可能。

例

arr1 = [1, 2, 3]
arr2 = arr1[:]

print(id(arr1))
# => 139997756485216

print(id(arr2))
# => 139997756311240

arr2[1] = 20

print(arr2)
# => [1, 20, 3]

# コピー先で書き換えてもコピー元には影響しない
print(arr1)
# => [1, 2, 3]

なお、この書き方はNumPyではコピーにはならずビューを提供するのみになる。

スライス表記を左辺に置いてもコピーにはなるが、左辺の変数が予めなんでもいいのでとにかく何かしらの配列である必要がある。この方法では、コピー先の変数に新しいオブジェクトが代入されるのではなく、既存の配列がコピー元と同じになるように書き換えられる。

dst[:] = src

例

arr1 = [1, 2, 3]
arr2 = [100, 200, 300, 400]

arr2[:] = arr1

print(id(arr1))
# => 139892140826208

print(id(arr2))
# => 139892140652232

arr2[1] = 20

print(arr2)
# => [1, 20, 3]

# コピー先で書き換えてもコピー元には影響しない
print(arr1)
# => [1, 2, 3]

# この方法でコピーするとコピー先のIDは変わらない。
print(id(arr2))
# => 139892140652232

この左辺にスライス表記を使う方法はNumPyでもコピーになるが、コピー前の配列は同じ大きさである必要がある。

または別の方法としては、copyモジュールがある。

import copy

copy.copy(lst)     # shallow copy
copy.deepcopy(lst) # deep copy

Ruby / JRuby 2014/12/31

Object#clone または Object#dup を使う。

shallow copy である。

Array#clone, Array#dup | Ruby 2.1 リファレンスマニュアル
http://docs.ruby-lang.org/ja/2.1.0/method/Array/i/clone.html

Perl 2015/03/03

shallow copy の例

my @src = ...;
my @dst = @src

代入するだけでコピーされる。

変数が配列へのレファレンスの場合の例

my $src = ...;
my $dst = [@$src];

デリファレンスしてから配列を生成し直している。

JavaScript 2013/04/20

shallow copy

arr_copy = arr.concat();

または

Array.prototype.clone = function(){
    return Array.apply(null, this)
}
arr_copy = arr.clone();
