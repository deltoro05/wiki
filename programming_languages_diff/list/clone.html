リストをコピーするには (clone) 2013/10/22

イミュータブルなオブジェクトのコピーは意味がないので、ここではミュータブルなリストのコピーについて。

PHP 2013/11/18

配列をコピーするには単に代入すればよい。

$src = array(1, 2, 3);
$dst = $src;
$src[0] = 10;

var_export($src);
// 出力:
// array (
//   0 => 10,
//   1 => 2,
//   2 => 3,
// )

var_export($dst);
// 出力:
// array (
//   0 => 1,
//   1 => 2,
//   2 => 3,
// )

要素が配列だとそれもコピーされるので、deep copyということか。

$src = array(1, array(2, 3));
$dst = $src;
$src[1][0] = 20;

var_export($src);
// 出力:
// array (
//   0 => 1,
//   1 =>
//   array (
//     0 => 20,
//     1 => 3,
//   ),
// )

var_export($dst);
// 出力:
// array (
//   0 => 1,
//   1 =>
//   array (
//     0 => 2,
//     1 => 3,
//   ),
// )
)

当然連想配列でも同じ。

Python 2014/03/12

copied = lst[:]

[:] はスライス表記であり、この書き方で shallow copy が可能。

または

import copy

copy.copy(lst)     # shallow copy
copy.deepcopy(lst) # deep copy
