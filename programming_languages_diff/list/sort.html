Ruby 2013/05/10

Array#sort

# ソートする
arr.sort

# 破壊的メソッド
arr.sort!

# ソート条件をブロックで指定
arr.sort  {|a, b| a.num <=> b.num}  # 非破壊的
arr.sort! {|a, b| a.num <=> b.num} # 破壊的

Perl 2013/09/11

sort(@arr)

ソートされた新しい配列を返し、引数の配列自体は変更しない。
