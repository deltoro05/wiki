概要 2013/09/25

同じ型のオブジェクトを並べたもので、配列、リストなど、各言語によって呼び方は違う。

sed などのコマンドも、このサイトでは文字列のリストを処理するものと捉えて、解説することにする。

C言語 2013/09/16

配列の仕組みがある。

インデックスでアクセスする場合は先頭の要素が0になる。

Java 2013/09/21

言語の組み込み型である配列と、クラスライブラリでサポートされている java.util.List インターフェースがある。

配列も java.util.List もインデックスでアクセスする場合は先頭の要素が0になる。

Scala 2013/09/21

Javaの組み込み型である配列と、Javaのクラスライブラリでサポートされている java.util.List インターフェースのほかに、Scalaのクラスライブラリにある scala.collection.Seq トレイトなどがある。

Javaの配列は、Scalaでは scala.Array というクラスのインスタンスということになっていて、いくつかのメソッドが存在する。

Javaの配列も Java/Scalaのクラスライブラリでも、インデックスでアクセスする場合は先頭の要素が0になる。

PHP 2013/09/16

連想配列が組み込み型として存在し、連想配列のキーに整数を使えば、単純な配列として使うことができる。

Python 2013/09/21

リストおよびタプルというデータ型がある。

インデックスでアクセスする場合は先頭の要素が0になる。

Pythonのタプルはイミュータブル(要素の書き換えが不能な)なリストという感じで、 Scalaなどのタプルとはだいぶイメージが違う気がする。

Pythonのタプルはイミュータブルなので、 ディクショナリのキーにできる。

Perl 2013/09/23

配列の仕組みがある。

Perlは、リストという用語は配列とは厳密には異なる意味を持ち、 (..., ..., ...) のようなソースコード上の表現をリストといい、それを配列の変数に代入すると配列になり、ハッシュの変数に代入するとハッシュ)になる。

配列そのものは変数名が @ で始まり、要素を参照するには $ で始める。

インデックスは0から始まる。

JavaScript / CoffeeScript 2013/09/21

オブジェクトのキーに整数を使って要素を定義し、 prototype に Array.prototype が指定されたものが配列のように振舞う。

下の3つは全部同じ配列を生成する。

var arr = [1, 2, 3];
var arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
var arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;
