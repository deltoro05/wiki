リスト(配列)の書き方

Java 2013/09/05

配列のほかに、java.util.List インターフェースを実装するクラスも使える。

配列 2013/09/04

int[] arr = {1, 2, 3};

または

int[] arr = new int[]{1, 2, 3};

java.util.List 2013/09/04

import java.util.List;
import java.util.ArrayList;

List<String> list = new ArrayList<String>();
list.add("a");
list.add("b");
list.add("c");

配列と java.util.List の相互変換 2013/09/04

java.util.List のインスタンスを配列に変換するには

ArrayList<String> list = ...;
String[] arr = new String[list.size()];
list.toArray(arr);

逆に配列を java.util.List に変換するには

import java.util.Arrays;

String[] arr = ...;
List<String> list = Arrays.asList(arr);

Scala 2013/09/05

Javaの配列、java.util.List の他に、scala.collection.immutable.List などもある。 scala.collection.immutable.List やその他のリストや配列のようなScalaのクラスは scala.collection.Seq を実装している。

arr = Array(1, 2, 3); // Javaの配列
list = List(1, 2, 3); // scala.collection.immutable.List

これは以下と同じ意味。

arr = scala.Array.apply(1, 2, 3);
list = scala.collection.immutable.List.apply(1, 2, 3);

PHP 2013/09/05

$arr = array(1, 2, 3);

最後に余計なカンマがあっても問題ない。

$arr = array(
             1,
             2,
             3,
            );

PHP 5.4からは以下のようにも書ける。

$arr = [1, 2, 3];

Ruby 2013/09/05

arr = [1, 2, 3]
arr = %w(abc def ghi) # => ['abc', 'def', 'ghi'] と同じ

%w の記法は、カッコは比較的なんでもよくて、以下はすべて同じ意味になる。

['abc', 'def', 'ghi']
%w(abc def ghi)
%w[abc def ghi]
%w!abc def ghi!

Perl 2013/09/05

配列そのものは変数名が @ で始まり、要素を参照するには $ で始める。

my @arr = ('abc', 'def', 'ghi');

('abc', 'def', 'ghi') の表記をリストという。リストを @ で始まる変数に代入すると配列になる。

最後に余計なカンマがあっても問題ない。

my @arr = (
           'abc',
           'def',
           'ghi',
          );

配列をネストすることはできず、ネストして書いたつもりでも平坦になってしまう。

my @arr = (1, 2, (3, 4));
print $arr[2]; # => 3
print $arr[3]; # => 4
my @arr2 = (10, 11, @arr);
print "@arr2"; # => 10 11 1 2 3 4

JavaScript 2013/08/21

下の3つは全部同じ配列を生成する。

var arr = [1, 2, 3];
var arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
var arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;
