Java 2013/04/17

配列

arr = {1, 2, 3};
arr = new int[]{1, 2, 3};

インデックスで参照するには

arr[index];

Scala 2013/04/17

arr = Array(1, 2, 3);
list = List(1, 2, 3);

これは以下と同じ意味。

arr = scala.Array.apply(1, 2, 3);
list = scala.collection.immutable.List.apply(1, 2, 3);

インデックスで参照するには

arr(index);
list(index);

これは以下と同じ意味。

arr.apply(index);
list.apply(index);

Ruby 2013/05/08

arr = [1, 2, 3]
arr = %w(abc def ghi) # => ['abc', 'def', 'ghi'] と同じ

%w の記法は、カッコは比較的なんでもよくて、以下はすべて同じ意味になる。

['abc', 'def', 'ghi']
%w(abc def ghi)
%w[abc def ghi]
%w!abc def ghi!

インデックスで参照するには

arr[index]

Python 2013/04/17

arr = [1, 2, 3]

インデックスで参照するには

arr.index(idx)

Perl 2013/07/29

配列そのものは変数名が @ で始まり、要素を参照するには $ で始める。

my @arr = ('abc', 'def', 'ghi');

インデックスで参照するには

$arr[0];

JavaScript 2013/04/10

下の3つは全部同じ配列を生成する。

arr = [1, 2, 3];
arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;
