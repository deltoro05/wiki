リスト(配列)の書き方

Java

配列 2013/04/17

int[] arr = {1, 2, 3};

int[] arr = new int[]{1, 2, 3};

インデックスで参照するには

arr[index]

java.util.List 2013/08/12

import java.util.List;
import java.util.ArrayList;

List<String> lst = new ArrayList<String>();
lst.add("a");
lst.add("b");
lst.add("c");

インデックスで参照するには

lst.get(index)

java.util.List のインスタンスを配列に変換するには

ArrayList<String> lst = ...;
String[] arr = new String[lst.size()];
lst.toArray(arr);

逆に配列を java.util.List に変換するには

import java.util.Arrays;

String[] arr = ...;
List<String> lst = Arrays.asList(arr);

Scala 2013/04/17

arr = Array(1, 2, 3);
list = List(1, 2, 3);

これは以下と同じ意味。

arr = scala.Array.apply(1, 2, 3);
list = scala.collection.immutable.List.apply(1, 2, 3);

インデックスで参照するには

arr(index);
list(index);

これは以下と同じ意味。

arr.apply(index);
list.apply(index);

PHP 2013/07/31

arr = array(1, 2, 3);

最後に余計なカンマがあっても問題ない。

arr = array(
            1,
            2,
            3,
           );

PHP 5.4からは以下のようにも書ける。

arr = [1, 2, 3];

インデックスで参照するには

arr[index]

Ruby 2013/05/08

arr = [1, 2, 3]
arr = %w(abc def ghi) # => ['abc', 'def', 'ghi'] と同じ

%w の記法は、カッコは比較的なんでもよくて、以下はすべて同じ意味になる。

['abc', 'def', 'ghi']
%w(abc def ghi)
%w[abc def ghi]
%w!abc def ghi!

インデックスで参照するには

arr[index]

Python 2013/04/17

arr = [1, 2, 3]

インデックスで参照するには

arr.index(idx)

Perl 2013/07/31

配列そのものは変数名が @ で始まり、要素を参照するには $ で始める。

my @arr = ('abc', 'def', 'ghi');

('abc', 'def', 'ghi') の表記をリストという。リストを @ で始まる変数に代入すると配列になる。

最後に余計なカンマがあっても問題ない。

my @arr = (
           'abc',
           'def',
           'ghi',
          );

インデックスで参照するには、要素がスカラのためシジルとして $ を使う。

$arr[0];

インデックスが最後よりも後ろになってしまうと、undef になり、 use warnings; を使っていると警告が表示される。

インデックスに負を使うと後ろから数えられる。

$arr[-1]; # 最後の要素
$arr[-2]; # 最後から2番目の要素

負のインデックスが先頭よりも前になってしまうと、undef になり、 use warnings; を使っていると警告が表示される。

配列をネストすることはできず、ネストして書いても平坦になってしまう。

my @arr = (1, 2, (3, 4));
print $arr[2]; # => 3
print $arr[3]; # => 4
my @arr2 = (10, 11, @arr);
print "@arr2"; # => 10 11 1 2 3 4

JavaScript 2013/08/21

下の3つは全部同じ配列を生成する。

var arr = [1, 2, 3];
var arr = {0: 1, 1: 2, 2: 3}; arr.__proto__ = Array.prototype;
var arr = {'0': 1, '1': 2, '2': 3}; arr.__proto__ = Array.prototype;
