リストの最初のn個の要素を削除するには (drop)

Scala 2013/07/30

// scala.collection.Seq[A]
def drop(n: Int): Seq[A]

結果を新しいリストで返し、自身は変更しない。

Python 2014/03/02

スライス表記でできる。

例

lst = [10, 20, 30, 40]

print lst[2:]
# => [30, 40]

Ruby 2013/04/28

arr.slice(n, arr.size - n)

sh 2014/03/02

tailコマンドを使うと、複数行テキストに対して、最初のn行を削除する処理をできる。

-n オプションの後に + と数字をつけると、その行番号から後ろだけを出力してくれる。従って先頭の3行を削除するには4行目から出力すればよいので、 -n+4 と指定する。

# 最初の3行を除く後続の行を出力
tail -n+4
