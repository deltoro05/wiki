配列(リスト)の最初のn個の要素以外の残りを配列で取得するには (drop) 2015/01/22

関連

  • 最初のn個の要素のみを配列で取得するには

Scala 2015/01/15

// scala.collection.Seq[A]
def drop(n: Int): Seq[A]

結果を新しいリストで返し、自身は変更しない。

例

val lst = List(10, 20, 30, 40, 50);

val result = lst.drop(2);

println(result);
// => List(30, 40, 50)

Python 2015/01/15

スライス表記でできる。

lst = [10, 20, 30, 40, 50]

print(lst[2:])
# => [30, 40, 50]

Ruby / JRuby 2015/01/05

Enumerable#drop メソッドを使う。これは非破壊的なメソッド。 ArrayはEnumerableをインクルードしているので、配列でもEnumerableのメソッドを使える。

arr.drop(n)

またはスライスを使う。

arr.slice(n, arr.size - n)
# または
arr[n, arr.size - n]
# または
arr.slice(n .. arr.size)
# または
arr[n .. arr.size]

上の2つの書き方は、2つ目の引数が配列の後ろの境界を超えてもエラーを出さないようなので、以下でも同じ。

arr.slice(n, arr.size)
# または
arr[n, arr.size]

sh 2015/01/15

tailコマンドを使うと、複数行テキストに対して、最初のn行を削除する処理をできる。

-n オプションの後に + と数字をつけると、その行番号から後ろだけを出力してくれる。従って先頭の2行を削除するには3行目から出力すればよいので、 -n+3 と指定する。

# 最初の2行を除く後続の行を出力
tail -n+3 foo.txt
