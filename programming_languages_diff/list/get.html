インデックスで要素を参照するには (get)

Java

配列 2013/09/04

arr[index]

java.util.List 2013/09/04

list.get(index)

Scala 2013/09/04

Array でも List でも同じ。

arr(index);
list(index);

これは以下と同じ意味。

arr.apply(index);
list.apply(index);

PHP 2013/09/05

$arr[$index]

Ruby 2013/09/05

arr[index]

Python 2013/09/05

arr.index(idx)

Perl 2013/09/10

配列そのもののシジルは @ だが、インデックスで参照するには、要素がスカラのためシジルとして $ を使う。

$arr[0];

インデックスが最後よりも後ろになってしまうと、undef になり、 use warnings; を使っていると警告が表示される。

インデックスに負を使うと後ろから数えられる。

$arr[-1]; # 最後の要素
$arr[-2]; # 最後から2番目の要素

負のインデックスが先頭よりも前になってしまうと、undef になり、 use warnings; を使っていると警告が表示される。

配列へのレファレンスから要素を取り出すには

my $arr = ['a', 'b', 'c'];

$arr->[0] # => 'a'
