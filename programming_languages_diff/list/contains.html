リストに特定の要素または特定の条件にマッチする要素が含まれているかどうかを判定するには (contains)

参考: セットに特定の要素が含まれているかどうかを判定するには

Java 2013/07/30

Signature:

// java.util.List<E>
boolean contains(Object elem)

Scala 2014/02/28

Signature:

// scala.collection.Seq[A]
def contains(elem: Any): Boolean

// scala.collection.Seq[A]
def find(p: A => Boolean): Option[A]

find は p の条件にマッチする最初の要素を取得する。

PHP 2013/11/14

in_array($elem, $arr)

TRUEかFALSEを返す。

in_array 関数 | PHP Manual
http://www.php.net/manual/ja/function.in-array.php

Ruby 2014/02/28

arr.include?(elem)

# 最初に見つかった要素のインデックスを返す。見つからなければ nil を返す
arr.index(elem)

# ブロックを与えれば、trueを返す最初の要素のインデックスを返す。見つからなければ nil を返す
arr.index {|v| ...}

例

arr = [3, 5, 7, 9]

p arr.index(7)
# => 2
p arr.index(8)
# => nil

p arr.index {|v| v % 7 == 0}
# => 2
p arr.index {|v| v % 8 == 0}
# => nil
