      • リストから最初のn個の要素のみを取得するには (take)
      • リストの最初のn個の要素を削除するには (drop)

      • リストのリストをリストに平坦化するには (flatten)
      • リストの要素を1つずつ順番に処理するには (foreach)
      • リストの要素を先頭から順番に畳み込むように関数を適用するには (foldLeft, reduce)
      • 同じ要素数の複数のリストを要素ごとにタプルにまとめて1つのリストを作るには (zip)
      • 複数のリストの要素同士をかけ合わせてリストのリストを作るには (product)
      • リストに特定の要素が含まれているかどうかを判定するには (contains)
      • リストの中身を比較するには (compare)
      • 同じ値を並べたリストを作成するには (fill)
      • 連続した数値のリストを作成するには (range)
      • リストを一定の数ごとにグルーピングするには (grouped)
      • リストを要素に応じてグルーピングするには (groupBy)
      • リストの先頭または最後の要素を削除したり、先頭または最後に要素を追加するには (shift, unshift, pop, push)
      • インデックスで要素を設定するには (set)
      • 特定の要素をキーで指定して削除するには (remove)
      • 特定の値の要素をすべて削除するには (remove-by-value)
      • リストをコピーするには (clone)
      • リストを逆順に並び替えるには (reverse)
      • リストの要素を並べ替えするには (sort)
      • リストの種類を相互に変換するには (convert)
      • リストからマップを生成するには (toMap)

リストからセットを生成する方法 (toSet)

Java 2013/11/05

List から Set に変換する例

import java.util.HashSet;
import java.util.List;
import java.util.Set;

List<String> lst = ...;
Set<String> st = new HashSet<String>(lst);

Scala 2013/11/05

val set = lst.toSet;

Signature:

// scala.collection.Seq[A]
def toSet: Set[A]

Python 2013/10/06

set という関数でセットに変換したものを返す。

set(lst)

Ruby 2013/10/06

以下は配列から重複を削除する例

# 重複を削除した配列を返し、自身は変更しない
arr.uniq

# 重複を削除した配列に変更する
arr.uniq!

Perl 2013/07/30

Perlにはセットというものはないが、以下は配列から重複を削除する例。

my @arr = (1, 2, 3, 4, 5, 4);
my %uniq = map {$_ => 1} @arr;
print(keys %uniq);
