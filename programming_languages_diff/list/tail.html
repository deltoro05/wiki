配列(リスト)の先頭以外の残りの要素を配列で取得するには (tail) 2016/11/28

配列(リスト)の先頭以外の残りを配列で取得する方法。

Go言語 arr[1:], sl[1:]
Scala lst.tail
PHP array_slice($arr, 1)
Ruby arr.drop(1)

Go言語 (golang) 2016/11/28

[1:] というスライス表記を使えば先頭を除く2つ目以降の要素からなるスライスを取得できる。配列・スライスどちらに対しても使えるが、配列に対して [1:] と書いても結果はスライスになる。

例

arr := [...]int{10, 20, 30, 40, 50} // (配列)
fmt.Printf("%#v\n", arr[1:])
// => []int{20, 30, 40, 50}            (スライス)

sl := []int{10, 20, 30, 40, 50}     // (スライス)
fmt.Printf("%#v\n", sl[1:])
// => []int{20, 30, 40, 50}            (スライス)

Scala 2015/01/07

Signature:

  • def Seq[A]#tail: Seq[A]

例

val lst = List(10, 20, 30, 40, 50);

println(lst.tail);
// => List(20, 30, 40, 50)

PHP 2016/11/28

array_slice($arr, 1) のようにする。

$arr = array(10, 20, 30, 40, 50);

var_export(array_slice($arr, 1));
// 出力結果
// array (
//   0 => 20,
//   1 => 30,
//   2 => 40,
//   3 => 50,
// )

array_slice は結果を新しい配列で返し、引数の配列を変更しない。

array_slice 関数 | PHP Manual
http://php.net/manual/ja/function.array-slice.php

Ruby / JRuby 2015/01/07

Array#drop を使えばできる。これは非破壊的なメソッド。

arr = [10, 20, 30, 40, 50]

p arr.drop(1)
# => [20, 30, 40, 50]

sh (シェルスクリプト) 2015/04/09

テキストファイルの2行目以降を取得する方法

sed 1d foo.txt # foo.txt の 1行目を除いた部分を出力
# または
tail -n+2 foo.txt # foo.txt の 2行目以降を出力の意味
