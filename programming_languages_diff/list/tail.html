配列(リスト)の先頭以外の残りの要素を配列(リスト)で取得するには (tail)

Scala 2015/01/07

Signature:

// scala.collection.Seq[A]
def tail: Seq[A]

例

val lst = List(10, 20, 30, 40, 50);

println(lst.tail);
// => List(20, 30, 40, 50)

Ruby / JRuby 2015/01/07

Array#drop を使えばできる。これは非破壊的なメソッド。

arr = [10, 20, 30, 40, 50]

p arr.drop(1)
# => [20, 30, 40, 50]
