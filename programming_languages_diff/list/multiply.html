同じ配列(リスト)を繰り返して連結するには(乗算みたいな) (x / *)

Python 2014/07/31

例

print([1, 2, 3] * 3)
# => [1, 2, 3, 1, 2, 3, 1, 2, 3]

print((1, 2, 3) * 3)
# => (1, 2, 3, 1, 2, 3, 1, 2, 3)

*演算子はシーケンス型に対して共通に使えるので、文字列に対しても同じように使える。

Ruby / JRuby 2014/12/31

*演算子を使う。

p [1, 2, 3] * 3
# => [1, 2, 3, 1, 2, 3, 1, 2, 3]

要素がコピーされるわけではないので、可変オブジェクトを繰り返す際には注意が必要。

*演算子は文字列に対しても使える。
-> 文字列の繰り返し

Perl 2015/01/22

x を乗算の演算子のようにして使う。 x は文字列に対しても使える。

my @arr1 = (1, 2, 3);
my @arr2 = (@arr1) x 3;
print Dumper(\@arr2);
# 出力結果
# $VAR1 = [
#           1,
#           2,
#           3,
#           1,
#           2,
#           3,
#           1,
#           2,
#           3
#     ];
