数のリストの合計を計算するには (sum)

Java 2013/11/08

簡単に計算できるメソッドはないので、for文などでループして計算するしかない。

Scala 2013/05/07

Signature:

// scala.collection.Seq[A]
def sum: A

要素が Int や Double などの場合に限る。

Groovy 2013/10/01

Signature:

// java.util.Collection
Object sum()

このメソッドはJavaでは定義されていない。

PHP 2013/11/08

例

$arr = array(1, 2, 3);
echo array_sum($arr);

array_sum | PHP Manual
http://www.php.net/manual/ja/function.array-sum.php

Python 2013/09/30

例

lst = [1, 2, 3]
print sum(lst)

Ruby 2013/05/07

def sum(arr)
  arr.inject(0, :+)
end

Perl 2013/09/11

例

my $sum = 0;
$sum += $_ for @arr;

sh 2014/07/07

1行に1つの数字が書かれたファイルからその合計を計算する例

# awkでのワンライナー
cat hoge.txt | awk '{sum+=$1}END{print sum}'

# perlでのワンライナー
cat hoge.txt | perl -nle '$s+=$_;END{print $s}'

平均の例

# awkでのワンライナー
cat hoge.txt | awk '{sum+=$1}END{print sum/NR}'

# perlでのワンライナー
cat hoge.txt | perl -nle '$s+=$_;END{print $s/$.}'
