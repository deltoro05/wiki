リストから特定の条件にマッチする要素のみを取り出すには (filter) 2013/08/21

リストから特定の条件にマッチする要素のみを取り出したリストを作成する方法、または特定の条件にマッチする要素のみのリストとマッチしない要素のみのリストに分ける方法。

Scala 2013/08/21

// Seq[A]
def filter(p: A => Boolean): Seq[A]

フィルター済みのリストを返し、自身は変更しない。

// Seq[A]
def partition(p: A => Boolean): (Seq[A], Seq[A])

返り値の1つ目は条件にマッチする要素のみのリストで、2つ目はマッチしない要素のみのリスト。

Ruby 2013/08/21

Array#select
Array#partition

select は条件にマッチする要素のみを抽出した配列を返す。

partition は条件にマッチする要素の配列と、マッチしない要素の配列を返す。返り値は配列の配列になる。

どちらも条件はブロックで表現する。

例

positive = arr.select {|elem| elem > 0}

positive, negative_or_zero = arr.partition {|elem| elem > 0}

Perl 2013/07/30

例

my @arr = (1, 2, 3, 4, 5);
my @arr2 = grep { $_ % 2 == 1; } @arr;
print "@arr2";
# => 1 3 5
