toMap 2013/05/06

リストからマップに変換する方法。

Scala 2013/05/06

タプルのリストをマップに変換する方法。

//  Seq[A]
def toMap[T, U]: Map[T, U]

例

val list = List(("a", 1), ("b", 2));
list.toMap; // => Map("a" -> 1, "b" -> 2)

Ruby 2013/05/06

偶数個の要素を持つ配列をハッシュに変換する方法。

arr = ['a', 1, 'b', 2]
Hash[*arr] # => {'a' => 1, 'b' => 2}

Perl 2013/07/30

偶数個の要素を持つ配列をハッシュに変換する方法。単にハッシュに代入すればいいだけ。

my @arr = ('a', 'abc', 'd', 'def');
my %map = @arr;
print $map{d};
# => def
