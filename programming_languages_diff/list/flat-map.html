リストの要素1つずつにリストを返す関数を適用して新しいリストを作るには (flatMap) 2013/10/22

map関数とflatten関数をつなげたイメージ。

Scala 2013/07/30

// scala.collection.Seq[A]
def flatMap[B](f: A => Seq[B]): Seq[B]

結果を新しいリストで返し、自身は変更しない。

Python 2014/04/12

リストの内包表記でforを入れ子にして内側のforの中に関数を書けばできる。

def f(i):
    return range(i)

lst = [1, 2, 3]

lst2 = [e2 for e1 in lst for e2 in f(e1)]

print(lst2)
# => [0, 0, 1, 0, 1, 2]

Ruby / JRuby 2014/01/04

Enumerable#flat_map を使う。 ArrayはEnumerableをインクルードしているので、配列でもEnumerableのメソッドを使える。

または Enumerable#map と Array#flatten を組み合わせても同じことはできる。

arr = [1, 2, 3]

p arr.flat_map { |i| (0..i).to_a }
# => [0, 1, 0, 1, 2, 0, 1, 2, 3]

p arr.map { |i| (0..i).to_a }.flatten(1)
# => [0, 1, 0, 1, 2, 0, 1, 2, 3]
