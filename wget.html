GNU Wget 1.9 Manual 日本語

BASIC認証のあるサイトをダウンロードするには 2013/04/12

wget --http-user=user --http-passwd=password URL

ウェブサイトを丸ごとダウンロードするには 2013/04/10

ウェブサイトを第3階層までリンクをたどりダウンロードする。

wget -r -l3 URL

ウェブサイトを第5階層までリンクをたどりダウンロードする。かつローカルでサイトを閲覧できるように、パスを相対パスに書き換える。

wget -r -l5 --convert-links URL

標準出力に吐き出すには 2013/05/03

wget -q -O - URL

-O - で結果を標準出力に出す。 -q は結果以外の進捗の表示などを無効にする。
