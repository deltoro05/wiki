ndarrayの属性 2015/04/01

ndarrayにはshapeという配列の次元をタプルで保存している属性と、 dtypeという配列の要素の型を保存している属性がある。

from pprint import pprint

npdata = np.array([[1.0, 2.0, 3.0], [11.0, 12.0, 13.0]])

# shapeは配列の次元を表すタプル
print(npdata.shape)
# 出力結果
# (2, 3)

# dtypeは配列の要素の型を表す
print(npdata.dtype)
# 出力結果
# float64

pprint(npdata.dtype)
# 出力結果
# dtype('float64')
