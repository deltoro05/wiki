iptables 2013/07/10

Ubuntuではiptablesの設定をufwで行う。

-> ubuntu.md#ufw

CentOSをインストールしたあとにApacheのポートを外部に向けて開けるには 2013/06/26

# root権限で
iptables -L –line-number
iptables -I INPUT 5 -p tcp --dport 443 -j ACCEPT
iptables -I INPUT 5 -p tcp --dport 80 -j ACCEPT

5 というのは、 iptables -L –line-number で現状の設定を確認した結果、 Chain INPUT の最後にある REJECT が5行目だった場合で、5行目に新しいルールを挿入する、という意味。

iptablesを設定した後は、設定の保存が必要。そうしないと、OS再起動時にリセットされてしまう。

# root権限で
/sbin/service iptables save
