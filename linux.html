デーモンの管理 (CentOSの場合)

起動時にデーモンを自動起動させるには

例えば、mysqlの場合

# root
chkconfig mysqld on

chkconfig の設定状況は chkconfig --list で確認できる。

手動でデーモンを起動させるには

例えば、mysqlの場合

# root
service mysqld start

手動で作った起動スクリプトを chkconfig に管理させるには

/etc/init.d/mydaemon というのがあるとすると

# root
chkconfig --add mydaemon
