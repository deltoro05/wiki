[CSS] ボックスの最低の高さを指定して、IE6でも対応させるには 2013/04/10

CSSの min-height はIE6に対応していないので、以下の例のように2行を付け足す。 height を2つ指定するのは、IE6のバグを利用したものらしい。 IE7以降は min-height だけでも大丈夫。

例

.hoge {
  min-height: 120px;
  height: auto !important; /* for IE6 */
  height: 120px; /* for IE6 */
}
