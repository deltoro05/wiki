readlink コマンド 2014/11/23

シンボリックリンクのリンク先を確認できるコマンド。

-> シンボリックリンクのリンク先を取得する

例

readlink -f シンボリックリンクのパス

-f を付けると階層の途中にあるすべてのシンボリックリンクを解決して、絶対パスにしてくれるみたい。付けないと、途中にシンボリックリンクがあった場合に1つだけしか解決しない。

-f を付けないと、指定したパスがシンボリックリンクでない場合は、なにも表示せずに、終了コード1を返す。

実行例

mkdir symtest
cd symtest
mkdir -p aaa/bbb
ln -s aaa a
ln -s bbb aaa/b

readlink a
# aaa

readlink a/b
# bbb

#readlink a/bbb
# 終了コード1で異常終了

#readlink aaa/bbb
# 終了コード1で異常終了

readlink -f a
# /home/hoge/symtest/aaa

readlink -f a/b
# /home/hoge/symtest/aaa/bbb

readlink -f a/bbb
# /home/hoge/symtest/aaa/bbb

readlink -f aaa/bbb
# /home/hoge/symtest/aaa/bbb
