特定の名前のプロセスをすべてkillするには 2013/04/18

たとえば "ruby" が名前に含まれるプロセスの場合、

まず一覧を表示してkill対象を確認する

ps aux | grep ruby | grep -v grep

次にプロセスIDの一覧だけにする

ps aux | grep ruby | grep -v grep | tr -s ' ' | cut -d ' ' -f2

それをkillする

kill `ps aux | grep ruby | grep -v grep | tr -s ' ' | cut -d ' ' -f2`
