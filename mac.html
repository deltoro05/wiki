パッケージ管理ツール 2013/05/03

  • Macports
    • パッケージ数は多い
    • FreeBSD の ports がベース
    • インストール先は /opt/local
  • Homebrew
    • パッケージ数は少ない
    • インストール先は /usr/local
    • Ruby で実装されている
  • Fink
    • 海外で多く使われている
    • パッケージ数は圧倒的
    • Debian の apt がベース
    • インストール先は /sw
  • Gentoo Prefix
    • パッケージ数は多い
    • Gentoo の portage がベース
    • インストール先は ~/Gentoo
    • 管理者権限不要

どれも Xcode が必須。

Phone Disk というソフトで MacFUSE file system is not available (-1) というエラー 2013/06/01

MacFUSE file system is not available (-1)

Phone Disk というソフトをインストールしている Mac に iPhone を接続したときに、上記のようなエラーが出た。試しに iFunBox というソフトでも同様のエラーが出て、いずれのソフトでも iPhone のファイルシステムをマウントすることができなかった。

システム環境設定の中に MacFUSE というのがあって、それを見てみたら、 MacFUSE がインストールされていないことになっていたので、その画面で Install ボタンを押して、しばらく待ったらインストールされた状態になったので、再度 Phone Disk を試してみたら、無事マウントできた。

つい先日まではなにもしなくてもマウントで着ていたはずなのに、なぜだろう？

スクリーンキャプチャ

Microsoftの Remote Desktop Connection が固まったときに一発でkillするには 2013/04/04

kill `ps aux | grep Remote | grep -v grep | tr -s ' ' | cut -d ' ' -f2`

拡張子sfxのファイルを解凍するには 2013/04/08

自己解凍書庫ファイルで、単に実行権限を与えて実行すればよい。

chmod 755 ファイル名.sfx
./ファイル名.sfx
