コマンドリファレンス

date

YYYYMMDD-hhmmss の書式で日時を出力する方法

date "+%Y%m%d-%H%M%S"

touch

touch でファイルのタイムスタンプを変更する方法

touch -t 日時 ファイル名

日時は YYMMDDHHmm.ss の書式。

ssh

サーバにある大量のファイルを tar しながら、ローカルにダウンロードする方法。

ssh <ID>@<IP> tar cvzf - <リモートのバックアップ対象ディレクトリ> > <ローカルの保存先tarファイル名>

リモートでtarを実行してファイルに書き出すのではなく標準出力に出して、それをsshでそのままローカルに転送し、ローカルでファイルに書き出している。
