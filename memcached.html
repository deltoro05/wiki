memcached 2013/06/26

データをメモリ上に保持するKVS。メモリ上にのみ保存するため、memcachedのプロセス再起動やOS再起動時にはデータが失われる。キャッシュとして使うのがよい。

PHPから使うためのインストール 2013/06/26

Ubuntuの場合

sudo apt-get install memcached php5-memcached

これを実行するだけで、memcachedが起動し、11211番ポートの待受が始まり、OS起動時の自動起動も設定される。

PHPでのサンプル 2013/06/26

<?php

$mc = new Memcached();
$mc->addServer('localhost', 11211);
$mc->set('testkey', 'testdata', 86400);
var_export($mc->get('testkey'));

CentOSでうまくいかない場合 2013/06/27

SELinuxが有効だと、memcachedに接続できないことがあるみたい。

set や get をしても false が返ってきて、 $mc->getResultCode() が26を返してしまう。 26の意味は不明だが。

26は、
http://www.php.net/manual/ja/memcached.constants.php#95793
によると RES_ERRNO ということで意味がよくわからない。
