Macの Phone Disk というソフトで MacFUSE file system is not available (-1) というエラー 2013/06/01

MacFUSE file system is not available (-1)

Phone Disk というソフトをインストールしている Mac に iPhone を接続したときに、上記のようなエラーが出た。試しに iFunBox というソフトでも同様のエラーが出て、いずれのソフトでも iPhone のファイルシステムをマウントすることができなかった。

システム環境設定の中に MacFUSE というのがあって、それを見てみたら、 MacFUSE がインストールされていないことになっていたので、その画面で Install ボタンを押して、しばらく待ったらインストールされた状態になったので、再度 Phone Disk を試してみたら、無事マウントできた。

つい先日まではなにもしなくてもマウントできていたはずなのに、なぜだろう？
