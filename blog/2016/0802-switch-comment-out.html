ソースコード中のコードをコメントアウトで簡単に無効化したり有効化したりするには 2016/08/02

デバッグの過程とかで、なにかしらのコードをいったんコメントアウトしたり、コメントアウトを解除したりを繰り返すときに便利なやり方。

PHPの場合。試してないけどC言語とかでも同じかも。

以下のようにすると、1行目の /* を // /* にすることで、コメントアウトの一時的な解除を簡単にできる。

/*
  無効化されたコード
  無効化されたコード
// */

コメントアウト解除

// /*
  有効化されたコード
  有効化されたコード
// */

次はその応用。

2つのコードブロックの1つ目を有効にしたり、2つ目を有効にしたりを簡単にする方法。

/*
  無効化されたコード
/*/
  有効化されたコード
// */

入れ替え

// /*
  有効化されたコード
/*/
  無効化されたコード
// */
