少なくとも1つのチェックボックスにチェックが入っているかどうかをチェックするカスタムバリデータ 2013/04/09

チェックボックスが複数あって複数選択可の質問に対して1つもチェックがなかった場合にエラーにする方法。

モデルのファイルに以下を記入する。

validates_with AtLeastOneValidator, fields_pattern: /^q5_.*/, fields: @@fields,
    message_field: :q5_other_text, message: "少なくとも1つを選択してください"

/config/application.rb に以下を記入する。デフォルトでは config.autoload_paths がコメントアウトされている。

config.autoload_paths += %W(#{config.root}/lib/validators)

/lib/validators/at_least_one_selected_validator.rb

class AtLeastOneSelectedValidator < ActiveModel::Validator
  def validate(record)
    regexp = options[:fields_pattern]
    fields = options[:fields].select {|m| regexp =~ m.to_s}
    fields.each do |m|
      v = record.send(m)
      return unless v == false || v == ''
    end
    record.errors[options[:message_field]] << options[:message]
  end
end

任意のパスの階層を特定のアクションに割り当てるルーティング 2013/04/05

match 'hoge/:path' => 'hoge#view'

とすると、hoge/foo にはマッチするが、hoge/foo/bar にはマッチしない。

match 'hoge/*path' => 'hoge#view'

とすると、hoge/foo にも hoge/foo/bar にもマッチし、param[:path] で 'foo' や 'foo/bar' を受け取れる。
