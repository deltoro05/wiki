tmux

キーバインド 2013/07/24

以下で は `Ctrl+b` など、tmuxのプリフィックキーで、設定によって変わる。

`x` : ペインを閉じる `&` : ウィンドウを閉じる `w` : ウィンドウ一覧を表示して選択する `o` (小文字のオー) : ペインの間を移動する ## 用語の説明 2013/07/24 {#dce5b9b} セッション : タブブラウザに例えると1つのウィンドウ。1つ以上のウィンドウが含まれる。 ウィンドウ : タブブラウザに例えると1つのタブ。1つ以上のペインが含まれる。 ペイン : ウィンドウの中の1つの端末で、ウィンドウの中を水平や垂直に区切って複数並べることができる。 ## コマンドラインでのコマンド 2013/07/24 {#00bb43e} # セッション一覧 tmux ls # 名前をつけてセッションを開始 tmux new -s hoge # 特定の名前のセッションに接続 tmux attach -t 名前
